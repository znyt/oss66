　　楚风眠要是全力，刚刚那一掌，已经是足以将司空轩轰杀了。
<br/>　　不过像是司空轩这等强者本身也不会像是表现出来的这般简单。
<br/>　　出身自武道世家，楚风眠肯定是有着自己的底牌。
<br/>　　楚风眠真正想要杀了他，也要耗费一些手段，不过要是楚风眠一心想杀，这司空轩，也得死。
<br/>　　“既然司空兄这么说了，那倒是没什么，反正此人修炼龙族武技，前往颜古龙帝的洞府中，也许还有些作用。”
<br/>　　麟木看了一眼白萱道。
<br/>　　“不过这小子要是陨落到了颜古龙帝洞府中，可别怪我们不救他，颜古龙帝的洞府危险无比，救了他可能会拖累了我们，倒时候他要是拖累了我，我第一个要杀了他。”
<br/>　　麟木冷哼一声道，言语之中，自负到了极点。
<br/>　　麟木说完，便是也坐了回去，继续安静喝茶。
<br/>　　“你们两位叫来的帮手，似乎还没有来？”
<br/>　　白萱扫视了一圈，缓缓的开口道。
<br/>　　这颜古龙帝的洞府，便是由白萱，麟木，司空轩三人发现的，因为其中的种种危险，难以对付，所以他们三人便是决定，每人各自再去找一个帮手前来。
<br/>　　“我叫的人，已经快到了，白萱姑娘不要急。”
<br/>　　司空轩笑了笑，看向外面，突然指着外面说道。
<br/>　　“你看。”
<br/>　　“哈哈哈，白萱姑娘不用急，本少只是在虚空海中，遇见了一只虚空兽杀了他浪费了些时间罢了。”
<br/>　　一尊声音，突然从天而将，落入到了这茶楼之中来。
<br/>　　这是一尊衣着华贵的男子，一举一动之间，都给人带来极大的压力。
<br/>　　楚风眠感觉的出，这名男子的实力，只怕是要比起白萱，司空轩等人还要强大几分。
<br/>　　“云公子？你居然来到北域了？”
<br/>　　看到这男子，旁边的白萱也是脸上露出几分诧异。
<br/>　　同时白萱暗地里对着楚风眠传音道。
<br/>　　“此人是云公子，来自南域皇室的一尊皇子，虽然有时回来北域走动，但是没想到这次被司空轩邀请过来了。”
<br/>　　说道这云公子的时刻，白萱的语气中都有这些深深的忌惮。
<br/>　　“这云公子手段狠毒，你要小心点，不然很有可能被他利用，陷入万劫不复之地。”
<br/>　　“哦？”
<br/>　　楚风眠打量了一眼那云公子，这云公子的相貌到像是仪表堂堂，脸上也始终带有这微微笑容，令人看到不由的心生好感。
<br/>　　不过在这云公子的双眼之中，楚风眠却是察觉到了其中的一些阴狠之意，就算是他隐藏的再好，也被楚风眠轻易的捕捉到。
<br/>　　显然这云公子，可并非善类。
<br/>　　“无妨，谁敢来对付我，谁就得死。”
<br/>　　楚风眠冷哼一声向着白萱回到。
<br/>　　“这次云兄，也是特意被我请来的，我们司空家，跟南域皇室有些交情，云兄听说了关于颜古龙帝洞府的事后，便是很感兴趣，所以便是被邀请而来了。”
<br/>　　司空轩笑着开口道。
<br/>　　南域不同于北域，北域中虽然都有着一尊尊国度，但是实际上真正的庞然大物，还是北邙学院，地魔门这样的宗门势力。
<br/>　　可以说最能代表着北域的，便是北邙学院。
<br/>　　而南域，却整个属于一方国家，其中的无数宗门势力，都是受制于这一方国家皇室管制的。
<br/>　　这云公子便是出身自南域皇室之中，不过南域皇室的公子，足有上千，这云公子也是其中一人罢了。
<br/>　　“颜古龙帝的洞府，本少当然有兴趣去探察一番。”
<br/>　　听着白萱的询问，云公子随意笑了笑，看了楚风眠一眼，随后便是坐在了椅子上，继续等待，喝茶。
<br/>　　“还剩下最后一人了，麟木，你邀请的人是谁？怎么还没到？”
<br/>　　如今已经到来五人，只差最后一人了，白萱不由的出声问道。
<br/>　　“颜古龙帝的洞府，现在虽然还没有被人发现，但是在继续拖延时间下去，也还是很危险的。”
<br/>　　“白萱不用着急，等那人来了，你自然知道。”
<br/>　　麟木笑了笑道。
<br/>　　“不错，白萱，你的急性子，也该改改了。”
<br/>　　又一道身影，突然从天而降，在此人的身影之中，楚风眠居然是觉察到了一种混乱之力。
<br/>　　跟曾经楚风眠从拓跋鼎身上见到的那混乱种子的力量，极为接近，不过却是要强大的多。
<br/>　　“罗辰，居然是你？”
<br/>　　白萱看到这身影，一眼便是认了出来。
<br/>　　“地榜第一，罗辰。”
<br/>　　此人不用白萱解释，听到这名字的一刻，楚风眠便是猛然张开了双眼，打量起来了此人。
<br/>　　这罗辰，正是北邙学院内门弟子中的第一人，地榜第一。
<br/>　　罗辰的实力，也是神力境九重的地步，但是他的实力，显然是要比起白萱，还要高上一头。
<br/>　　尤其是他身体中的混乱之力，更是难以对付的多。
<br/>　　“这罗辰，是圣党的人，你要小心，他很有可能会针对你。”
<br/>　　白萱悄然之间对着楚风眠传音说道。
<br/>　　“圣党党首夜天君，一直都在追求寒月漓，只是不成，以你跟寒月漓的关系，你便是夜天君心中的头号仇敌，这罗辰很有可能会为了讨好夜天君而对付你。”
<br/>　　“哦？”
<br/>　　楚风眠目光闪烁，看到这罗辰身上混乱之力的一刻，楚风眠便是猜到了一些。
<br/>　　他暗自催动灵识，探察过去，这罗辰的身躯深处，却是正好有着一枚混乱种子，显然也是夜天君给予他的。
<br/>　　不过罗辰，显然是要比起拓跋鼎在圣党中的地位要高得多，这混乱之力，拓跋鼎只能够是借助其中的一部分力量。
<br/>　　而现在罗辰看起来，却是已经将这混乱之力，融会贯通，凭借他身躯中的混乱种子，这罗辰也可以算是半个混乱神体。
<br/>　　“恩？”
<br/>　　罗辰落下，跟在座众人打了个招呼，突然间他看向楚风眠的时刻，眼神中目光开始闪烁起来，似乎是在想着什么。
<br/>　　在罗辰的眼神中，一股杀意，缓缓涌现出来。
<br/>　　“既然诸位都到齐了，那就准备出发吧。”
<br/>　　司空轩看了一眼众人，便是出声说道，刚刚起身，准备离开。
<br/>　　“等等！”
