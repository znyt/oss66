　　这些天剑宗弟子，可都是在天羽神域之中摸爬滚打下来得到过不少奇遇的武者。

<br/>　　尤其是这宝车的威力，比起寻常的天级灵器，都要厉害的多。

<br/>　　这数十位天剑宗弟子，同时出手，施展出无数剑术，斩向楚风眠。
<br/>　　可在这七彩霞光之下，却是全部都被挡住，根本无法伤及到楚风眠分毫。
<br/>　　这天魔宝车，不愧为九魔城至宝，如今被楚风眠所炼化，威力更甚于数倍在小魔主手中。
<br/>　　这七彩霞光笼罩，几乎便是将楚风眠立于不败之地。

<br/>　　楚风眠冰冷开口一声。
<br/>　　在他身边的七彩霞光，猛然散开，楚风眠手持祖龙至尊剑，突然拔剑之间，一跃而出。
<br/>　　“好机会！”

<br/>　　那些天剑宗弟子，看到这一幕，都是同时出手，无数剑光，交织在一起，化为了三十六道无上剑气，组成天罡大阵，向着楚风眠斩杀而去。

<br/>　　无数天剑宗弟子，都是大吃一惊，他们联手之下，布置出这天罡剑阵来，为的便是要斩杀楚风眠。
