　　　　丑是丑，但江行泽不在乎了，他这次回来，犹如王者归来，带着绝世蛇种吸血蛇，睥睨天下。
<br/>　　这座山的毒蛇都要成为吸血蛇的食物，不断的促进它的血脉提升，直至成为真正的蛇王。
<br/>　　不，不单单的蛇，连这座山的人，也要沦为吸血蛇的食物。
<br/>　　这两位捕蛇师傅的死，仅仅是一个开始。
<br/>　　“吸血蛇尽情的吃吧，这座山的蛇跟人都是你的食物，跟着我，你才能成为真正的蛇王”
<br/>　　蛇王号令天下群蛇，头顶显化王字印记。
<br/>　　此时的江行泽眼中闪过怨毒跟快感，梦呓似的说：“银白蛇，你不是被白镇人民奉为蛇王吗，那么我就要看看你这条蛇王敢不敢出现，救他们于水火”
<br/>　　逃出山野集团的江行泽跟吸血蛇就宛如脱缰的野马，无所顾忌。
<br/>　　这个世界上最怕的就是疯子，因为疯子是没有逻辑的，没有顾虑的，所有的条条框框在他眼里都如同虚构。
<br/>　　只为达到自己的目的。
<br/>　　江行泽变成了这般模样，如今一心想找那条该死的蛇报仇，吃它的肉，喝它的血。
<br/>　　他就要把动静闹大，闹的满城风雨。
<br/>　　佘山上，每天都有不少捕蛇人进山，渐渐的，有些人上山了就再也没有下来过。
<br/>　　期初，还没人注意跟发现，但随着几天过去，他们的家人开始慌了，出去一打听，居然有好些捕蛇人都没有从山上下来。
<br/>　　一人如此，二人如此，短短几天，光捕蛇人就失踪了不下二十多人。
<br/>　　白镇的人民不淡定了，捕蛇人在佘山出现意外事故其实是常有的，但是每天都有人失踪，就不寻常了。
<br/>　　而且属于活不见人死不见尸，慢慢的，一种无形的恐惧笼罩在白镇人民的心头。
<br/>　　后来有人说佘山是出了一头会吃人的怪兽，那些消失的捕蛇人就是被怪兽给吃了。
<br/>　　此谣言一出，顿时人心惶惶，但谣言终归是谣言，得不到证实，次日，就有不少捕蛇人自愿组成一只十五人的小队伍，专门上山寻找那些失踪的捕蛇人。
<br/>　　顺便调查一下是不是真的有吃人的怪物。
<br/>　　当这只队伍不断的深入佘山，慢慢的他们发现一路上连一条毒蛇影子都没看到。
<br/>　　不禁有人诧异：“莫非连山上的毒蛇都被怪物给吃了？”
<br/>　　佘山真的出现了吃人的怪兽吗，暗地里不少人都咽了两三口唾沫，好像是嗓子里发干似的。
<br/>　　队伍四下排查，深入密林野径，这是属于白镇人民的山，就算有吃人的怪兽，也要给找出来铲除。
<br/>　　不然隐患存在，没有人能心安，也不可能大家永远都不上佘山捕蛇了。
<br/>　　能成为捕蛇人，都是胆大心细，十五人的队伍在佘山杂道纵横的地方探索，各种各样的路径如数家珍。
<br/>　　找，不管是失踪的捕蛇人，还是怪物，总要找一种结果出来才行。
<br/>　　不断的查找，不断的深入佘山。
<br/>　　“大家快来，有发现”
<br/>　　有一人在前方忽然大喊了一句。
<br/>　　队伍的人都冲过去，只见前方地面有一条蜿蜒的蛇形印记，有水桶般大小，蛇形印记看上去异常光滑，像是被抹了油一般。
<br/>　　这绝对是一条非常巨大的巨蟒游曳留下的蛇印。
<br/>　　都是有经验的捕蛇人，相互对看一眼，心中诧异之余，仿佛找到了前进的方向。
<br/>　　难不成吃人的怪兽也一条巨蟒？
<br/>　　“顺着蛇形印记，大家摸过去看看，不过都小心点”
<br/>　　队伍顺着地上的蛇形印记出发，穿过一片密林之后，走着走着大家发现了不对劲。
<br/>　　忽然队伍整齐的停了下来，一个个面面相觑，既然都不敢往前方去了。
<br/>　　佘山有两处蛇类汇聚最多的地方，第一处叫蛇头山的地方，遍地毒蛇，品种繁多。
<br/>　　而另一处便是在佘山的深处，那里有一条几十多里长的长沟，沟里沟外水草肥美，适合蛇生长，可谓毒蛇纵横，数量多的吓人。
<br/>　　几百年前就存在了，被称为蛇沟。
<br/>　　蛇沟里面究竟有多少条蛇，恐怕没人计算过，也无法去猜测，据上了年纪的人讲，相传蛇沟曾经是蛇王出世的地方。
<br/>　　那时候还没有证道的蛇王是一条无比巨大的蟒蛇，它每日都会进入，因为巍峨庞大的身子游曳拖地，在地面渐渐的形成了一条长沟。
<br/>　　后来蛇王证道离开，这条长沟繁衍了很多毒蛇出来，就变成为蛇钩，更是佘山毒蛇的大本营。
<br/>　　没一人敢进出蛇沟，进去的人也没有人在出来过。
<br/>　　里面到底有多少品种的毒蛇，里面到底有多大，都一无所知。
<br/>　　“要不要进去看看”
<br/>　　队伍上的捕蛇人看着地面的蛇形印记，分明就是指向蛇沟里面。
<br/>　　吃人的巨蟒是蛇沟里出来的？
<br/>　　这点很多可能，很久以前就有人见过水桶粗的大蛇从里面游曳出来。
<br/>　　“进一点看看吧，有什么不对，就立刻退出来”
<br/>　　蛇沟里面是神秘的，基本上是捕蛇人的禁地，但如果去里面捕蛇，绝对是抓个盆满钵满，但也足够撑死你。
<br/>　　十五人的队伍小心翼翼的朝着蛇沟里面出发，但都如履薄冰。
<br/>　　还没前进一公里，他们就看到前方蛇沟旁有一座山洞，洞穴中黑漆漆，阴森森的。
<br/>　　看上去，好像是什么大蛇的巢穴。
<br/>　　这山洞坐落在蛇沟的旁边，本身就足够说明很多问题。
<br/>　　捕蛇人可以准确的说，在蛇沟这个地方，里面只有蛇，不会有别的动物，如果有，也是蛇类的食物。
<br/>　　这个洞穴透着诡谲，让人不禁浑身冷颤。
<br/>　　“怎么办，是继续往前面走，还是进这个洞穴看看”
<br/>　　“你们有没有察觉到一股很强烈的蛇类气息”
<br/>　　好几人道：“我只感觉好像被什么东西给盯上了，浑身难受”
<br/>　　忽然，阴森森的洞穴中好像有一道身影若隐若现，慢慢的他们既然看到一位黑袍男子走了出来，看着众人笑了起来：“欢迎来到我江行泽的新家做客”
<br/>　　“洞穴中走出来一个人？”
<br/>　　众人顿时撑着惊骇的目光看着黑袍男子，要说里面爬出一条大蛇，他们能没什么好惊讶，但出来一个人，就有点剧情反转了。
<br/>　　还未等众人想明白为什么阴森森的山洞会走出一个人，就在此时，他们后方一条长长的影子拖过，挡住了他们的后方退路，只见一条二十三米的庞大巨蟒，浑身布满红火鳞片，凶煞狰狞的口吐人言。
<br/>　　“吼...人类，你们可是自动送上门给我吃的”
<br/>　　顿时间，所有人都陷入了一种死亡恐惧之中。
