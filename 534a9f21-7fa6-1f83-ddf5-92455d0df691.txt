　　城主府，前厅之中，
<br/>　　黑岩城的一众高层，齐聚一堂，个个满脸愁色。
<br/>　　“那位叶先生那边，还没有任何动静吗？”
<br/>　　灵山坐在首位之上，沉着脸，开口问道。
<br/>　　下方两名黑甲统领，对视了一眼，纷纷摇了摇头，
<br/>　　“回城主，从昨晚到现在，那位叶先生一直待在二小姐的院落之中，未曾有任何举动！”
<br/>　　众人听到这话，脸色顿时越发难看，
<br/>　　“这不对劲啊！按理说，他真要是某个大宗门的弟子，也应该通过神符传音，请宗门的长辈前来助阵才对啊！”
<br/>　　“是呀！这都一夜过去了，也没见个人来？”
<br/>　　“他该不会真的想，一个人去对抗整个血月宗吧？”
<br/>　　……
<br/>　　就在众人忧心忡忡之时，外面突然传来一阵阵躁动之声。
<br/>　　继而片刻之后，
<br/>　　“杀我孙儿的小子，你给本座滚出来！！”
<br/>　　“滚出来！！”
<br/>　　一道犹如天雷般的声音，自九天之上滚滚传来，久久不息。
<br/>　　听到这道声音，灵山众人纷纷霍然站了起来，个个面色惨白，
<br/>　　“这声音……好像是血无影的爷爷，血月宗的太上长老，血天海！”
<br/>　　“之前不是一直说此人在闭关吗？没想到他竟然亲自来了！”
<br/>　　“这血天海可是出了名的残暴，若是那位叶先生不能挡住此人，我们黑岩城也要跟着遭殃！”
<br/>　　……
<br/>　　众人在说话之间，外面已经响起了一道道惨叫声，显然杀戮已经开始了！
<br/>　　灵山等人正惊恐不已，不知道该如何是好，
<br/>　　轰隆！
<br/>　　伴随着一道巨响，似乎有一件重物，落在了大厅前的广场之上。
<br/>　　刺啦啦！
<br/>　　整个由上等岩石打造的广场，瞬间全部化成齑粉，
<br/>　　连同整个城主府，都一阵剧烈晃动，仿佛发生了大型地震一般！
<br/>　　哗啦啦！
<br/>　　很快，周围房屋全部轰然倒塌。
<br/>　　灵山等人来不及多想，纷纷闪身而出，逃到大厅之外，这才免去了被房屋砸中的厄运。
<br/>　　待众人站稳身形，定睛望前一看，
<br/>　　只见此刻，前方广场正中心的位置，正站着一名满头赤发、杀气腾腾的高大老者。
<br/>　　这老者一身红袍，周身散发着强大的血煞之气，宛如一头上古凶兽，似乎随时都能吞噬他们的性命。
<br/>　　“是——血天海！”
<br/>　　扑通！
<br/>　　扑通！
<br/>　　扑通！
<br/>　　看到这老者的面目之后，感受到对方身上那恐怖的气息，灵山等人忍不住纷纷跪倒在地，一个个吓得瑟瑟发抖，甚至连头都不敢抬了。
<br/>　　如果说之前的血无影是恶魔，那眼前的这个老者，那就是恶魔中的恶魔，不折不扣的大魔头！
<br/>　　传言在数十年前，血天海为了修炼一门邪功，曾经屠杀了整整一个星球的生命！
<br/>　　相比之下，区区一个黑岩城，根本就不够他杀的。
<br/>　　之前灵山等人还抱有侥幸，以为血天海最近这些年，一直在闭关修炼，应该不会亲自前来，可是万万没有想到，越是怕什么，就越是来什么！
<br/>　　“黑岩城城主是哪个，给本座滚出来！”
<br/>　　血天海似乎对众人的反应还算满意，并没有再继续大开杀戒，连声音也比之前缓和了许多。
<br/>　　灵山听到这话，顿时吓得浑身一哆嗦，不过最终还是咬着牙，从人群中跪着爬出，恭恭敬敬行了一礼，
<br/>　　“晚辈灵山，添为黑岩城城主，拜见血老前辈！”
<br/>　　“哼！”
<br/>　　血天海冷哼一声，淡淡瞥了一眼趴在地上的灵山，犹如俯视蝼蚁，冷冷开口道：
<br/>　　“小子！本座来问你，我那孙儿，死在你的城中，此事你作何解释？”
<br/>　　灵山连忙道：
<br/>　　“血老前辈明鉴！我黑岩城不过区区一座小城，如何敢与血公子为敌？杀害血公子之人，另有其人，绝非我黑岩城所为啊！！”
<br/>　　为了保住性命，灵山果断先将自己撇清。
<br/>　　血天海冷冷一笑，
<br/>　　“谅你们这帮废物，也没有那个胆量！”
<br/>　　灵山听到这话，顿时心头一松，还以为总算逃过一劫。
<br/>　　却不料，血天海说到这里，话锋忽然一转，
<br/>　　“可是本座听说，此事是因为你的女儿而起，你也逃脱不了干系，所以，你也去死吧！”
<br/>　　说完这话，血天海随手一抓，
<br/>　　轰！
<br/>　　一个巨大的血色手掌凭空出现，一把将跪俯在地上的灵山抓了起来。
<br/>　　“啊！！”
<br/>　　灵山顿时惨叫了一声，却根本没有任何反抗的余地，心头一阵绝望。
<br/>　　就在这时，忽然一道清脆动听的声音陡然响起，
<br/>　　“不要伤我父亲！此事全都是因我而起，你要是就杀了我吧！”
<br/>　　声音落下，一道妙曼的倩影，出现在广场之上。
<br/>　　此人自然正是灵韵！
<br/>　　“嗯？”
<br/>　　血天海看到突然跳出来的美貌女子，不由得眉头微微一挑，
<br/>　　“原来你就是那个，害我孙儿丧命的女人！倒也生的标致，难怪能被我孙儿看上！也罢，既然如此，本座就成全了你，这就让你去地下陪我的孙儿吧！”
<br/>　　说完这话，血天海将灵山随手丢到一旁，然后一掌朝灵韵拍落下去！
<br/>　　轰隆！
<br/>　　那硕大的血色巨掌，犹如一座小山般，朝灵韵狠狠砸落下来，以后者的实力，根本抵挡不了，甚至在那股强大的气息压迫之下，身体连动弹一下都做不到！
<br/>　　“韵儿！！”
<br/>　　另外一边，犹如死狗般，被丢在地上的灵山，顿时大吼了一声，可惜面对如此强大的存在，他也根本毫无任何办法。
<br/>　　“唉！我这一生，就这样结束了吗？”
<br/>　　灵韵不由得叹息了一声，绝望的闭上了双眼。
<br/>　　然而，
<br/>　　等了片刻之后，
<br/>　　灵韵没有感觉到任何痛苦，反而忽然觉得浑身上下一阵轻松，之前那股强大的压迫感，居然消失的无影无踪了！
<br/>　　继而，
<br/>　　一道淡漠的男子声音，在耳边响起，
<br/>　　“有我在，你还没有那么容易死！”
<br/>　　灵韵顿时娇躯一颤，慌忙睁开双眸一看，顿时喜形于色，激动的娇躯乱颤。
<br/>　　只见，她的身前，不知道何时，出现了一道瘦削的身影，此时一手负在身后，单手托掌，居然将血天海的强大一击，生生拦了下来！
<br/>　　“叶、叶先生！”
<br/>　　此人自然正是叶尘。
