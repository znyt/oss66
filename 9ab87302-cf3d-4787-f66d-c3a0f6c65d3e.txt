　　一路回到宿舍，李虎、唐威和陈东三人，跟在叶尘的身后，谁也没有吭声，但是眼眸深处，全都透着浓浓的敬畏之色。
<br/>　　“我知道你们心中有很多疑惑，现在尽管问吧！”
<br/>　　叶尘转过身来，冲着三人微微一笑，开口道。
<br/>　　三人不由得对视了一眼，最终还是李虎鼓足了勇气，小心翼翼道：
<br/>　　“那个，尘哥，你……您真的就是那位传说中的华夏仙人，云州叶先生吗？”
<br/>　　叶尘淡淡一笑，
<br/>　　“怎么？我看着不像？”
<br/>　　“不像！不不，我的意思是，我打死也没有想到，震慑全球的云州叶仙人，竟然会是我的舍友，这简直……简直太不可思议了！”
<br/>　　李虎越说越激动，都有些语无伦次。
<br/>　　陈东也挠了挠头，道：
<br/>　　“我还以为华夏仙人，应该是个白胡子老头呢……”
<br/>　　唐威则两眼放光，激动道：
<br/>　　“尘……不，叶仙人，我听说，仙人都有通天彻地之能，还会各种仙家法术，这是真的吗？”
<br/>　　叶尘微微一笑，
<br/>　　“什么仙人不仙人的！我说过，你们永远都是我室友、兄弟！你们还是像以往称呼就好！”
<br/>　　说完这话的同时，叶尘缓缓抬起手掌，猛然一张，
<br/>　　呼！
<br/>　　一团炙热的火焰，自叶尘的掌心中发出，将原本灰暗的宿舍，瞬间照的通明，甚至连屋子里的温度也急剧飙升，犹如一下子进入了一个大熔炉之中。
<br/>　　“啊！”
<br/>　　李虎三人可曾见过如此场面，顿时纷纷惊呼了一声，不由得连连倒退。
<br/>　　好在叶尘只是展示了一下，便又将火焰迅速收回体内，否则整个宿舍楼恐怕都要被焚毁一空了。
<br/>　　“这……真的是神仙手段啊！”
<br/>　　对于三人来说，纵然是之前，叶尘一掌灭掉了何坤山，一指斩杀了杨之凡，也不及眼前这一幕来的震撼。
<br/>　　毕竟，凭空召唤出火焰，这已经完全超出了普通人的思维。
<br/>　　“尘哥，传说你一人压倒了整个米国，曾经还逼迫岛国首相公开道歉，横扫东南亚六国！这些也都是真的？”
<br/>　　“我还听说，你可以用身体硬抗核弹，当初米国动用核弹对付你，你都毫发无损……”
<br/>　　“还有还有，传言还说，你如今已经是华夏军中元帅，不知道这些是真是假？”
<br/>　　……
<br/>　　三人回过味来之后，如同追星族一般，各种八卦问题纷纷问了出来。
<br/>　　叶尘没有隐瞒，全部一一作答。
<br/>　　三人似乎也发现，叶尘对他们，并没有那种高位者的姿态，依旧把他们当做室友、兄弟一般对待，在无比激动的同时，更加狂喜不已。
<br/>　　和大名鼎鼎的叶仙人成为了兄弟，这恐怕是他们这辈子做的最牛逼的一件事情了。
<br/>　　……
<br/>　　一晃眼，又过去了半月有余，
<br/>　　虽然这件事在天海大学引起了不小轰动，叶尘的身份也再次成为了众人议论的话题，不过叶尘根本不予理会，再加上他悄悄动用神龙卫的力量，整件事情很快被压了下来，时间一久也就渐渐的被人淡忘了。
<br/>　　随着时间进入冬天，叶尘进入大学的第一学期，也马上就要结束。
<br/>　　临放假之前，学院按照以往的惯例，组织了一次冬游，准备前往华山游玩。
<br/>　　叶尘原本并不打算参加，却没想到班长陶婉清，竟然会主动找上门来，
<br/>　　“叶尘！这是班级团体活动，我不管你是什么身份，只要你还是我们生物学院的学生，你就必须参加！”
<br/>　　听到陶婉清这话，叶尘原本并不想理会，不过就在他准备转身离开的时候，却猛然察觉到，陶婉清的身上，似乎隐隐透着一股诡异的气息。
<br/>　　‘这气息……和当初的神子卡斯特，以及西方教廷的老教皇，似乎有些相似，莫非这女人，和神界的人有过接触？’
<br/>　　察觉到这个异样之后，叶尘顿时也来了兴趣，
<br/>　　很显然，陶婉清非要自己参加冬游，此事肯定没有那么简单，说不定可能是神界的一场阴谋！
<br/>　　‘神界的人，终于要来了吗？既然你们主动送上门来，我也不能让你们失望啊！’
<br/>　　一念及此，叶尘微微一笑，
<br/>　　“好，我去！不过，我要带上我女朋友一起去，这应该没有问题吧？”
<br/>　　曦月的元神恢复，已经到了最后时刻，离最终圆满成功，也只有一步之遥，叶尘自然不可能丢下她不管。
<br/>　　陶婉清原本还担心，叶尘根本不会听自己的，见他竟然答应了下来，顿时心头一喜，连忙道：
<br/>　　“当然可以！每位同学都可以带一名伴侣，不过吃住自费。”
<br/>　　“明天早上八点，在学校大门口集合，不要迟到！”
<br/>　　陶婉清似乎生怕叶尘反悔一般，飞快的说完之后，便逃也似的离开了。
<br/>　　而叶尘见状，也顿时更加确信了自己的判断，这个陶婉清，绝对有问题！
<br/>　　于是心念一动，一道神念自他的眉心射出，将陶婉清的行踪暗暗锁定。
<br/>　　果不其然！
<br/>　　陶婉清离开之后，来的一个无人的角落，拿出手机，开始拨打一个陌生号码。
<br/>　　片刻之后，手机接通，手机那头，传来一道阴测测的女子声音，
<br/>　　“小美人，事情办得怎么样了？”
<br/>　　陶婉清听到这个声音，脸色明显有些凝重，先确认了一遍周围的环境，方才深吸了一口气，压低声音道：
<br/>　　“叶尘已经答应参加这次学院组织的冬游，不出意外的话，明天就回出现在华山之上！”
<br/>　　“我已经按照你们说的做了，你们之前答应过我的事情，应该没有忘记吧？”
<br/>　　电话那头，先是传来一道娇笑，语气中透着一抹兴奋，道：
<br/>　　“放心！小妹妹，只要你将那姓叶的小子引到华山，姐姐我自然会替你，帮你那个小情人报仇的！咯咯咯~~”
<br/>　　陶婉清自然不知道，自己和神秘人的通话，已经全部被叶尘听在耳中。
<br/>　　“看来我还真是小瞧这陶婉清了，没想到她竟然可以和神界搭上关系！或者说，这神界为了对付我，还真是不择手段啊，竟然要借助一个凡人之手！”
<br/>　　“看来那华山之上，此刻多半已经布下天罗地网了吧？”
