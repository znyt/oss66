        “还会拆炸弹！”童筱颖微笑着补充道。
<br/>
<br/>        傅夜沉看着她脸上那灿烂的笑容，会心一笑。
<br/>
<br/>        其实，他还杀过人，杀的都是穷凶极恶之人。当然，他也救过人，救的都是手无缚鸡之力的人质。
<br/>
<br/>        他的身份，是不可说的秘密。
<br/>
<br/>        但是眼前这小女人，已经将他猜得八九不离十了。
<br/>
<br/>        只不过，他还有另一个身份，除了上级那一个人，其他人一概不知，就连他爷爷也不知道。
<br/>
<br/>        就在此时，童筱颖的手机响了。
<br/>
<br/>        一段“我爱的人，她已有了爱人”的铃声，让傅夜沉感同身受。
<br/>
<br/>        为了避免大哥又说她手机铃声是噪音，童筱颖连忙滑动到接听键，将手机举到了耳边。
<br/>
<br/>        “昼景。”她的声音，温柔似水。
<br/>
<br/>        “你在哪儿？”
<br/>
<br/>        “我在大哥的办公室里。”
<br/>
<br/>        “你怎么去我大哥那儿了？”
<br/>
<br/>        “我给你送完U盘后，在电梯里遇到了孙副部长。是孙副部长，让我来给大哥讲解办公室的设计理念，以及大哥他想要修改的地方，我要做好记录。”
<br/>
<br/>        “吃饭了吗？”
<br/>
<br/>        “嗯，正在吃。”
<br/>
<br/>        “吃完饭，早点回家。”
<br/>
<br/>        “好。”
<br/>
<br/>        “算了，还是我去接你吧！正好，我待会要路过傅氏集团大厦楼下。”
<br/>
<br/>        “嗯，好！”
<br/>
<br/>        这次虽然仍旧是傅昼景先挂的电话，但是童筱颖心里高兴不已。
<br/>
<br/>        傅夜沉见童筱颖挂了电话，淡淡地问了一句：“阿景应酬完了？”
<br/>
<br/>        “嗯！我得赶紧把饭吃完，他待会会来接我一起回家！”童筱颖放下手机后，端起碗筷，快速往嘴里扒饭，并夹了一大把菜往嘴里塞。
<br/>
<br/>        她已经完全不顾自己的吃相是否优雅，一心扑在了傅昼景身上。
<br/>
<br/>        傅夜沉看着她这般狼吞虎咽的样子，忍俊不禁道：“慢点吃，别噎着了。”
<br/>
<br/>        其实，阿景在工作上的动静，爷爷一直暗中派人盯着。
<br/>
<br/>        包括今天阿景突然和季氏集团董事长，也就是季思妍她父亲签订合同的事情，他也知道。
<br/>
<br/>        只是，他不知道他们具体签的是什么合同。
<br/>
<br/>        这都是爷爷派给他的秘密特助阿凯告知爷爷后，再告知他。
<br/>
<br/>        爷爷说，阿景可能起了歹心，让他最好防一防。
<br/>
<br/>        但是，傅夜沉至今都有些无动于衷。
<br/>
<br/>        因为，防任何人，他都不想去防阿景。
<br/>
<br/>        当傅昼景赶到傅夜沉的办公室时，童筱颖刚把午饭吃完。
<br/>
<br/>        傅昼景走进来后，先唤了声：“大哥”，然后才走到童筱颖的面前，见她嘴角还有油渍，便从衣兜里掏出纸巾，温柔地替她擦拭。
<br/>
<br/>        童筱颖心里就像抹了蜜糖一般，甜得嘴角情不自禁地微微上扬，就像纯洁的小天使那般发自内心的微笑着。
<br/>
<br/>        他俩秀恩爱，只能让傅夜沉站在一旁，默默地吃狗粮。
<br/>
<br/>        “大哥，我就带筱颖先回去了。”傅昼景替童筱颖擦拭完嘴角，然后牵住了童筱颖的手，对傅夜沉说道。
<br/>
<br/>        傅夜沉故作淡漠地迎了迎手，冷冷地说了一句：“不送。”
<br/>
<br/>        而童筱颖，从傅昼景进来后，就再也未看过他傅夜沉一眼。
<br/>
<br/>        她的眼里，心里，满满的，全是傅昼景。
<br/>
<br/>        傅夜沉觉得自己心痛得像要死过去了一样。
<br/>
<br/>        她明明是他的女人，她从始至终都是他的，可是……
<br/>
<br/>        就因为他自己不去核实，才会失去她。
<br/>
<br/>        如今，他只能眼睁睁地看着她对别人微笑，成为别人的妻子。
<br/>
<br/>        从傅夜沉的办公室里出来后，傅昼景脸色一沉，语气不满中又透着一股子隐隐的酸味，醋意十足地说道：“筱颖，以后不许再为我大哥下厨！”
<br/>
<br/>        “诶？”童筱颖怔愣了一下。
<br/>
<br/>        傅昼景微微眯起眼眸，抬起手来在童筱颖的鼻梁上轻轻地刮了刮：“大中午的，你在我大哥的办公室里吃饭。我大哥又不会做饭，除了你下厨，还能有谁下厨？你为我大哥下厨，我会吃醋！”
<br/>
<br/>        “嘻嘻——我知道了啦！以后不会了！”童筱颖挽住傅昼景的胳膊，轻轻地将脑袋靠上了他的肩头。
<br/>
<br/>        “不过，我也觉得奇怪。大哥为什么要在自己的办公室里设置厨房？”傅昼景微微耸了耸肩。
<br/>
<br/>        童筱颖微笑着说：“说不定大哥跟你一样了，要是工作忙过头了。办公室里的二楼，有厨房，有内卫，有卧室，还有阳台，家具电器齐全，很适合加班过夜。”
<br/>
<br/>        “也是！”傅昼景莞尔一笑。
<br/>
<br/>        “你中午怎么想着要回来呢？”童筱颖一边跟傅昼景走进电梯里，一边好奇地问。
<br/>
<br/>        傅昼景轻轻地揉了揉童筱颖的额头，回答道：“我下午要出差，回来收拾行李。”
<br/>
<br/>        “那我替你收拾好了！”童筱颖乖巧地眨了眨眼睛。
<br/>
<br/>        傅昼景点了点头。
<br/>
<br/>        童筱颖又想起了什么，接着说道：“孙副部长今天跟我道歉了！让我明天继续去设计部上班，还给我提高到月薪一万的工资。”
<br/>
<br/>        “我说过，我会养你。不需要你出去工作。”傅昼景微微皱了下眉。
<br/>
<br/>        童筱颖站直了身子，拉起傅昼景的手，摇了摇：“我知道的。但是，我一个人在家里好无聊啊！你就让我去工作，好不好？等我拿到了大学通知书，我就不用去工作了。”
<br/>
<br/>        “那我和大哥说说，让你去傅氏集团总部的设计部吧！”傅昼景若有所思地说道。
<br/>
<br/>        “为什么？”童筱颖反而不解了，“我留在地利置业的设计部挺好的，为什么要让我来大哥这儿？”
<br/>
<br/>        “我和你的关系，我暂时不打算对外公开，所以，今天，我才会说你是我表妹。如果，我两在一栋大厦里工作，以后难免会碰到像今天这样的尴尬场面。所以，我觉得，你来我大哥这儿会比较好。”傅昼景耐着性子解释道。
<br/>
<br/>        其实，他是不想筱颖遇见季思妍。
<br/>
<br/>        现在，他已经和季氏集团签订了合作合同，那么，季思妍肯定会经常来地利置业找他。
<br/>
<br/>        季思妍现在对他傅昼景来说，就是一个合作伙伴兼床伴的关系，也算是他的地下情人吧！
<br/>
<br/>        所以，他不希望，筱颖知道季思妍的存在。
<br/>
<br/>        童筱颖抿了抿唇，有些闷闷不乐地点头答应了。
<br/>
<br/>        “我会跟大哥说，让他命令下面的人多关照你，不会再有人敢欺负你了。”傅昼景又接着说。
<br/>
<br/>        童筱颖只是对傅昼景强颜欢笑。
<br/>
<br/>        傅昼景从未想过，他的大哥是否对筱颖有什么私心，也从不认为，他的大哥会对筱颖有什么私心。
<br/>
<br/>        所以，他很放心，筱颖在他大哥手下工作。
<br/>
<br/>        回到家后，童筱颖便开始给傅昼景整理行李箱。
<br/>
<br/>        这次，他说，他要出差半个月才会回。
<br/>
<br/>        还说，让她如果有什么急事，自己无法解决，可以去找大哥。
<br/>
<br/>        就连大哥傅夜沉的私人号码，他都留给她了。
<br/>
<br/>        童筱颖一脸无奈地看向傅昼景。
<br/>
<br/>        “筱颖，我大哥长得帅，我不许你多看他一眼！懂了吗？”出门前，傅昼景看着愁眉不展的童筱颖，打趣地说。
<br/>
<br/>        童筱颖立即提起精神来，一脸认真地扣起三根手指头来比在额角，郑重地发誓：“嗯，懂了！我发誓，绝不多看大哥一眼！”
<br/>
<br/>        “筱颖。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “你觉得大哥帅，还是我帅？”傅昼景忽然无厘头地问道。
<br/>
<br/>        童筱颖“噗嗤”一声，忍俊不禁起来：“当然是我老公最帅啊！”
<br/>
<br/>        “乖，等我回来。我给你带礼物！”傅昼景会心一笑，一手握住行李箱的拉杆，一手托住童筱颖的后脑勺，在她的眉心上印上了一个浅浅的吻。
<br/>
<br/>        “嗯，我等你回来。”童筱颖腼腆地垂眸一笑。
<br/>
<br/>        傅昼景轻轻地揉了揉童筱颖的头顶，然后转身离去。
