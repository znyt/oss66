　　“你……怎么样？”叶少阳在她对面找了个椅子坐下，没话找话说。
<br/>　　陈悦转头看了他一眼，没有作声。
<br/>　　叶少阳有点尴尬，只好开门见山，问道：“那个，你想起自己是谁了没有？”
<br/>　　陈悦摇摇头。
<br/>　　“但是这一切我都很熟悉。”
<br/>　　“什么？”

<br/>　　陈悦道：“有那么一点印象，我好像生活在一座山里，但我想了很久，具体真的想不起来，但之前看你作法，我也觉得很熟悉……”
<br/>　　叶少阳愣了一下道：“你会法术？”
<br/>　　陈悦站起来，摆出了一个手势，叶少阳一下愣住：这是道门的一个很常见的手印，果然，她是法师！
<br/>　　“我们来试一试！”陈宇说着走过来，单手结印，朝叶少阳打过来。
<br/>　　“我怕伤到你。”
<br/>　　“没让你出全力！”
<br/>　　陈宇的手掌已到，叶少阳急忙结印，用两成法力迎了上去。
<br/>　　啪！
<br/>　　双掌相接，叶少阳直接被震飞出去，撞在门上，十分狼狈地摔倒。
<br/>　　靠！
<br/>　　这妹子，法力这么强！自己刚才还说把她打伤来着，尴尬了……
<br/>　　瓜瓜在一旁吓的捂住了嘴巴。陈悦自己也吓了一跳。
<br/>　　叶少阳满脸通红地站起来，故作深沉地说道：“可以可以，我只用了两成法力，居然被你打成这样，你还挺厉害啊。”
<br/>　　在内心中估算了一下，自己两成法力，大概相当于一个天师施展全力的水平，而陈悦一下子把自己击倒，那就说明……她的实力远远不止天师，甚至是地仙？
<br/>　　这可能吗！
<br/>　　别说是地仙，就算真是天师，在法术界也一定会有一席之地，不可能连名字都不知道。
<br/>　　“那个，姑娘，你要是愿意，让我感知下你体内真气的情况。”
<br/>　　陈悦没有意见，于是叶少阳坐到她边上，一只手扣住她手腕的命门，释放真气进入他体内……
<br/>　　跟上次的感觉完全不一样。
<br/>　　上一次，她能感觉到陈悦的气海异常坚固、牢不可破，里面不知道储存着什么气息，而今，不知道什么原因，她的气海打通了，体内经脉之中，流淌着充沛的……叶少阳陡然一惊，这不是真气，而是戾气！
<br/>　　但仔细感知了一番，叶少阳震惊发现，她体内的戾气之中，没有那种狂暴的因子，而是相当稳定，与少量真气混合在一起、完美融合，就像……自己体内被炼化的戾气一样！
<br/>　　叶少阳一下迷茫了。
<br/>　　自己是靠着诸多奇遇，再加上大周天吐纳心法，不断炼化之下，才将戾气都炼化吸收，至今还有少许戾气没被炼化，叶少阳轻易都不敢运用。
<br/>　　这姑娘难道跟自己有一样的经历？
<br/>　　看她一脸平静，丝毫没有被戾气侵扰的样子，叶少阳心中更加疑惑，问她会不会大周天吐纳心法。
<br/>　　陈悦表示听都没听过。
<br/>　　叶少阳无语了。
<br/>　　更让他吃惊的是，从她体内如此雄浑的戾气来看，至少是地仙实力，甚至更强，但开天眼看她，身上没有灵光——法师一旦突破天师以上的境界，阴司册封之后，头顶和双肩会生出只有天眼等特殊手段才能看到的灵光，这灵光没什么用，只是身份象征。
<br/>　　而且这这种册封，是通过生死簿直接勾画，会通过魂力相连，直接体现在肉身上，想不要都不行，除非……是邪修。
<br/>　　但是，陈悦体内戾气非常纯正平和，不带一丝邪气，显然不是邪修……
<br/>　　这也不是，那也不是，叶少阳彻底无语。

<br/>　　叶少阳见此安慰道：“你也别烦恼，你暂时先跟着我们，等回去之后，慢慢帮你找到身份吧。”
