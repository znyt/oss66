　　第203章
<br/>　　弹指光阴，转眼便是一年光景。
<br/>　　盛夏，京城外十里。
<br/>　　烈日炎炎，落在人身上如火烤一般。可这样的天气，还有一队队人走在官道上，他们三五成群，或骑马，或坐车，或结伴步行，有锦袍，有麻衣，也有穿着草鞋。
<br/>　　老张在官道旁开了一家茶摊，因为朝廷要开武科举，天下习武之人都将在最近赶到京城，路上人来人往的，生意却还不错。
<br/>　　“三位客官，您的茶，还有一碟熟肉，三位慢用。”
<br/>　　围着桌子坐着三个人，看上去像一家三口，老父亲带着一对兄妹。老者年逾五旬，下巴留着山羊胡，兄长年纪稍大些，膀大腰圆，表情憨厚，那姑娘年纪却不大。
<br/>　　“三位也是进京赶考的吧？”老张明显是有意搭话。
<br/>　　“看出来啦。”山羊胡呵呵笑着：“带我大儿子去。”
<br/>　　“这还能看不出来，每天在我茶摊上坐的，十个有九个半是为了这件事。瞧小哥这身板，起码能封个百夫长。俺老张要是年轻几岁，也要去试试，拼着这把老骨头不要，也得杀几个戎族。”
<br/>　　“茶掌柜，您就别凑热闹了。那戎族岂是好杀的，当年那一战，一路打，一路败，最后被戎族打到天子脚下，嘿，我堂堂帝国，竟无一人能抵得住戎族。”有个客人道：“对了，老张，去年戎族围了京城，你住在这里，见过戎族么？”
<br/>　　“怎能没见过，我那小儿子……嗨，不说这个，不说这个。”老张干笑两声
<br/>　　“若说帝国没人能挡得住戎族，我看不对，想那帝国战神，一战歼敌十万，实实在在替咱们出了口心中恶气。”有个长须客人道。
<br/>　　“嘿，你说这事谁不知道，不瞒你说，我们这次来，一为武科举，二是为了见见那位帝国战神，若能得到他一二教诲，怕此生已受用无穷。”
<br/>　　“今天帝国谁不想见他啊，只不过，那样的神仙人物，怕是不好见到的。”
<br/>　　“朝廷要在长安封赏大人，或许有机会见上一面的。”
<br/>　　“如果帝国都是这样的大将，我们也不用被戎族这么欺负了。”
<br/>　　提到『帝国战神』四个字，众人变得热络起来，便是素昧平生之人也像许久未见的故友一样。不知这四个字有如何大的魔力，竟可以将陌生人凝聚在一起。
<br/>　　“哼！”
<br/>　　一个小姑娘重重的冷哼了一声，声音并不大，但却像在一锅肉粥里加了一把沙子，有些硌牙。
<br/>　　“这位姑娘，对帝国战神，还是不要不敬得好，对英雄还是要保持敬意的。”
<br/>　　“这位姑娘年纪还小，不知世事，利兄还是莫与她一般见识。”
<br/>　　“呸。”小姑娘啐了口茶叶：“狗屁英雄，一条狗我凭什么要敬他。”
<br/>　　“你！”长须人气得青筋爆现。
<br/>　　“哈哈，我这小女儿言语粗鄙，惹诸位不开心了。灵儿，什么狗屁，这是你个姑娘能说的嘛。”山羊胡呵斥一声，在桌上搁下一叠铜板，道：“诸位告辞，告辞了。”
<br/>　　三人起身欲离开，老张这才觉得有点不对。这老者只是说小姑娘言语粗鄙，可没有说她说得不对。而这小姑娘模样长得挺俊，但言谈举止却显得没什么教养，像，像个……土匪。
<br/>　　要不要上去问一问？
<br/>　　“呔，休走！”
<br/>　　长须男拎一根哨棒拦在三人面前，道：“你们对我不敬没关系，可我却不能容忍你们对战神大人不敬。你这小姑娘必须收回自己说的话，否则休怪我利南行无礼。”
<br/>　　“这位兄台，你要如何无礼呐？”
<br/>　　一直没开口说话的憨厚汉子把手掌搁在他的肩膀上，他的手如何伸过来的，长须男竟没有看清楚。只感觉肩膀山像压了一座大山，骨节咯吱咯吱作响，啪地一声，哨棒握不紧，竟掉在地上。
<br/>　　三人缓缓离去，在茶楼众人的目光中，三人身影消失在京城方向。
<br/>　　“爹爹，咱们找了这么久，走了好多地方，都没找到大当家，大当家会不会已经……”
<br/>　　“一年前，大当家虽被逼得跳崖，可后来官兵并没有找到大当家尸体。我有种感觉，大当家应该还活着。”
<br/>　　“不知什么时候才能找到他？”
<br/>　　“这次武科举，天下习武之人都要到京城，这么热闹的事情，大当家不会缺席的，更何况，姓杨的要到了……”
<br/>　　三人对视一眼，互相点了点头。
<br/>　　这三人，自然就是徐神机，秦蛮，徐灵儿。徐神机和秦蛮脱身后，去南方寻到徐灵儿，之后便在帝国大地上寻找程大雷。一年之间，走了荆州，豫州，幽州，甚至是关外戎族的地界，但却都没有寻到程大雷的消息。有人说在某地出现过一个使斧头的憨人，抓紧时间赶过去，却也没碰到对方。
<br/>　　一路碾转到京城，比武科举更重要的事情是，明帝要在天下人面前封赏杨龙停。
<br/>　　蛤蟆寨灰飞烟灭，多少兄弟流浪天下，这一切都是拜杨龙停所赐。而蛤蟆寨所做的事情，却被按在杨龙停身上，享受无上荣耀，被奉为帝国战神。
<br/>　　蛤蟆寨的山贼不在乎荣耀，也不在乎封赏，可却不能容忍小人猖狂。不然，那些在战斗中死去的兄弟，会死不瞑目的。
<br/>　　每一个蛤蟆寨还活着的人，都会向杨龙停讨个公道。在幽州他躲在城内，不好杀他，而在京城，却未必没有机会。
<br/>　　程大雷若还活着，他会来京城。
<br/>　　少羽，林冲若还活着，也会来京城。
<br/>　　刘关张赵也会来京城。
<br/>　　在今日，怕是有很多兄弟会在京城相遇。
<br/>　　而今日，徐神机一行三人来到京城。
<br/>　　京城，已在眼前。
<br/>　　京城，位于京州，另有一名，唤——长安。
<br/>　　长安，位于京州，乃是帝国之京都，天下之腹地，人说江南水乡，水墨芬芳，东海之滨，观海听涛，川蜀之地，天府之国……可所有人都明白，这些地方都无法和长安相提并论。
<br/>　　因为这里是长安，是天下第一城。
<br/>　　长安长安，天下长安，这是历代帝皇的之梦，许多人终生未到过长安，梦中也定踏入过这座城池。
<br/>　　
<br/>(本章完)
