　　先祖殿。
<br/>　　在这古老宫殿的牌匾之上，写着这三个字。
<br/>　　“这是，五行天先祖的祭祀之地？”
<br/>　　楚风眠自言自语道。
<br/>　　先祖殿，应该就是五行天先祖的祭祀之地。
<br/>　　五行天乃是传承数万年的宗门了，曾经建立五行天的许多强者都纷纷陨落，所以有着这祭祀先祖的大殿。
<br/>　　“我们五行天的宝库，就在这先祖殿之中。”
<br/>　　林天殿主走到这先祖殿的门口，对着楚风眠开口道。
<br/>　　“带我进去。”
<br/>　　楚风眠观察了一眼这先祖殿开口道。
<br/>　　“是。”
<br/>　　林天殿主恭敬的回答道，没有任何的异样，直直的走向先祖殿，打出了一道法决。
<br/>　　这法决打入其中，顿时围绕在先祖殿周围的阵法禁制，都是纷纷消失了。
<br/>　　这先祖殿之上的禁制阵法，虽然已楚风眠的实力，也足以轻易的破解。
<br/>　　不过为了小心行事，楚风眠还是等待这林天殿主去破。
<br/>　　砰！
<br/>　　随着禁制阵法的破碎，这先祖殿的大门也是轰然打开，一股纯粹的檀香味道，传递到了楚风眠的鼻子之中。
<br/>　　这样的先祖殿之中，肯定是日日夜夜燃烧这檀香，来祭奠先祖。
<br/>　　“看起来没诈？”
<br/>　　楚风眠目光闪烁，观察着先祖殿内，看起来在这其中，不像是什么危险之地。
<br/>　　而且既然是以禁制阵法守护的，那多半五行天的宝库，的确是藏在先祖殿之中。
<br/>　　看着林天殿主已经是踏入到了先祖殿之中，楚风眠便是也紧随其后，踏入到了先祖殿之中。
<br/>　　可就在楚风眠踏入先祖殿之中的刹那间，在那先祖殿中心，所祭奠的牌匾之上，一道身影，突然浮现而出，凝聚惊天灵力，向着楚风眠猛然轰击而来。
<br/>　　灵魂！
<br/>　　这先祖殿祭奠的牌匾之中，居然是藏有了一缕先祖的灵魂，而且更是一尊仙魂。
<br/>　　仙人之魂。
<br/>　　踏入天人境的强者，所留下的一缕灵魂，留在了这先祖殿之中，日夜受到五行天的祭拜，居然是存留了数万年之久，成为了五行天的守护神。
<br/>　　这一缕仙人之魂凝聚出来的一刻，便是感受到了外敌入侵，向着楚风眠发起了攻击。
<br/>　　“果然有诈！”
<br/>　　楚风眠早知这林天殿主居心不良，定然有诈，所以在进入这先祖殿之中的一刻便是早就小心了起来。
<br/>　　看着这仙人之魂的出现，楚风眠身上的血脉之力猛然爆发而起，一掌跟这仙人之魂碰撞了过去。
<br/>　　两股灵力在空中交锋，眼前这一道身影，虽然是一缕残魂所化，可是却是要比起当初那五行刀圣师尊，那神秘仙人林五行的一缕化身，还要强横。
<br/>　　留下这一缕残魂的人物，至少是一位仙士，甚至是境界更高的仙人。
<br/>　　不过只是一缕残魂罢了，楚风眠也不惧怕。
<br/>　　轰隆！
<br/>　　楚风眠掌心之中的灵力猛然加剧，始祖第二式直接轰击而出，那一缕仙人之魂，顿时是被震飞了出去。
<br/>　　楚风眠的实力，无法是跟真正的仙人，天人境强者抗衡，可对方一些残魂，还绰绰有余。
<br/>　　“仙人之魂倒是好东西，融入到灵器之中，足以是让灵器的力量强横不少，不过这只是一缕残魂，倒是不足差了一点。”
<br/>　　魂魄是融入灵器的最好东西，像是一些强大天级灵器的器灵，其实都是以生灵的魂魄，融入其中的。
<br/>　　眼前这一缕仙人之魂，如果可以融入到一件天级灵器之中，那么哪一件天级灵器的威能，绝对足以达到媲美剑鼎的地步。
<br/>　　不过融入他人魂魄，那一件灵器也就会失去晋升成为极道仙兵的可能。
<br/>　　毕竟极道仙兵，可是会自然的诞生出其中的器灵来，借助外力的灵器是无法晋升成极道仙兵的。
<br/>　　楚风眠身上的这几件灵器，都是有着晋升极道仙兵的可能，尤其是祖龙至尊剑，只差一个契机，楚风眠当然不会因噎废食。
<br/>　　这一缕仙人之魂，楚风眠倒是没有灵器来将其融入。
<br/>　　“就给心魔之王，增添实力吧！”
<br/>　　楚风眠心思一动，下了决定，虽然他现在的实力，这心魔之王对楚风眠已经没有多少的帮助了。
<br/>　　不过日后这心魔之王的力量越来越强，也可以脱离楚风眠，成为独立的存在，倒时候加入剑道门之中，也算是多一位强者。
<br/>　　毕竟现在心魔之王的力量已经是足以影响九劫古帝了，再进一步，半步天人境的强者，这心魔之王都可以对付了。
<br/>　　让未来心魔之王坐镇剑道门，也让剑道门多以为去强者守护。
<br/>　　“给我灭！”
<br/>　　想到这里，楚风眠便是不在跟这仙人之魂浪费手脚，大手一挥，一股强横的灵力当即是笼罩在了这一缕仙人之魂上。
<br/>　　随着楚风眠猛然一抓，这一缕仙人之魂，直接是被楚风眠捏在了手心之中。
<br/>　　“心魔之王！”
<br/>　　轰！
<br/>　　楚风眠的一生呼喊着之下，心魔之王当即现身而出，楚风眠将那仙人之魂一扔，心魔之王便是张开大口，将这一缕仙人之魂，全然吞噬。
<br/>　　仙人之魂，纵然是残魂，都拥有这不亚于半步天人境强者的力量。
<br/>　　尤其是这残魂，更是心魔之王最好的补品。
<br/>　　心魔之王的力量节节攀升，刹那间便是达到了半步天人境的境界。
<br/>　　“多谢主人，赐我新生。”
<br/>　　在楚风眠的脑海之中，顿时响彻起了心魔之王的声音。
<br/>　　踏入了半步天人境，心魔之王都是诞生出了自己的灵智，寻常的心魔，因为是人心所化，所以是没有灵智的。
<br/>　　纵然是心魔之王，也是如此，只有本能，没有灵智，无法交谈。
<br/>　　而现在心魔之王踏入了半步天人境，却是属于开启了灵智，未来甚至是不需要楚风眠帮助，心魔之王都可以自己修炼，成长，已经是跟人类无异了。
<br/>　　“很好，心魔之王，现在你就去镇守剑道门，守护我剑道门，以后还有好处，也不会少了你的。”
<br/>　　楚风眠看着心魔之王笑着说道。
<br/>　　心魔之王诞生灵智更好，也就相当于剑道门多了一位真正的半步天人境强者。
<br/>　　现在剑道门内，说起来实力还有些寒酸，真正可以抗衡半步天人境的人，只有楚风眠一个。
<br/>　　洛汐，寒月漓，凭借剑源界的力量，在剑源界内还可以对抗半步天人境，但是在外就不行了。
<br/>　　现在多了一个心魔之王，也算是一大助力了。
