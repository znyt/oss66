　　远方…
<br/>　　横天殿主的身影，疾速飞行着，已然飞离百万里之外。
<br/>　　同一时间，早就远远飞离的，还有童烨等人。
<br/>　　这时。
<br/>　　黑云学教一行人，疾速飞行着，往黑云学教方向而回。
<br/>　　忽然，一个学教弟子皱眉道，“童烨，你之前那句话，说真的？”
<br/>　　“真的脱离学教？”
<br/>　　“你说呢？”童烨耸耸肩，脸色有些淡漠，也有些无奈。
<br/>　　“自从那女人来我们学教后，学教现今都成什么样子了？”
<br/>　　“忘了上次秦翊被她打伤的事了？连一些长老都受了不少气。”
<br/>　　“现在的黑云学教，简直乌烟瘴气，留着，还有什么意思？”
<br/>　　周遭学教弟子，摇了摇头，叹了口气。
<br/>　　“算了。”秦翊摇了摇头，道，“莫悠师兄前些天已出关，那莫忧也已经回天藏学宫了，别多想了。”
<br/>　　童烨点点头，“便是如此，我们现今才还回学教。”
<br/>　　“学教之内，各位长老毕竟待我们极好，又有授业之恩。”
<br/>　　“萧逸那边怎么办？”一个弟子忽然面露忧色地问道。
<br/>　　“八殿强者齐至，即便他再怎么妖孽，怕是也双拳难敌四手。”
<br/>　　“说起来，那通缉令是怎么回事？”秦翊也皱眉说道。
<br/>　　“我怎么没听说药尊殿被掠夺了一批天材地宝？”
<br/>　　“还有，我之前看过学教任务上的卷宗记载，写的是那批天材地宝由我们学教弟子护送。”
<br/>　　“我怎么不知道有这回事？”
<br/>　　“一个借口罢了。”童烨不屑地冷笑一声。
<br/>　　“嗯？”周遭学教弟子，疑惑地看着童烨。
<br/>　　童烨撇撇嘴，道，“大概大半月前吧，我亲眼看到她缠着副院长，说着什么通缉令和报复之类的话。”
<br/>　　“副院长本是百般不愿，后来不知怎么地，副院长又答应了。”
<br/>　　“哼。”童烨冷哼一声，“你别让我猜准了，莫忧这小娘们，心肠歹毒，怕是在外头吃了亏，回来拿我们学教当枪使呢。”
<br/>　　“我听青麟说，这小娘们仗着洛前辈的令牌，胡作非为，早就不知道逼着学教内的长老为她报私仇多少次了。”
<br/>　　“那个陈长老，本来不过是他们在外结识的朋友，却愣是直接进了学教，当了长老，本事不大，倒是借着长老之权，没少拿学教好处。”
<br/>　　“所以我才说学教内一片乌烟瘴气。”
<br/>　　“不对。”秦翊忽然脸色一惊，“刚才那群‘通缉者’，是萧逸的朋友吧。”
<br/>　　“我还是第一次见他如此盛怒，一言不合便分尸杀人。”
<br/>　　“要不我回去一趟看看。”
<br/>　　“我们一起回去。”周遭学教弟子齐声道。
<br/>　　“别。”童烨皱眉道，“现在回去，免不得又有误会。”
<br/>　　“秦翊一人回去吧，万里无踪，幻风秦翊，本就是风刹殿中人。”
<br/>　　“我猜，萧逸这家伙只是一时之气，并未真的和我们计较。”
<br/>　　“所以，我们下次再寻他叙旧吧；现在，秦翊独自回去就好。”
<br/>　　秦翊闻言，点了点头。
<br/>　　……
<br/>　　同一时间，萧逸这边。
<br/>　　萧逸，仍旧直视着副院长。
<br/>　　只不过，脸上的冰冷，已然化作了满脸的复杂。
<br/>　　萧逸淡淡地张开了嘴巴，语气，同样只剩复杂。
<br/>　　“若他们不是我萧逸的朋友，那么…这些个通缉令，若无人去细查，去较真，便永远无人知晓此事。”
<br/>　　“无人知晓副院长你滥用殿主权职，无人知道一堆主殿主、副殿主，与你一并滥用八殿权力。”
<br/>　　“若他们不是我萧逸的朋友，那么…现在他们已经枉死了，而且是死在恪守职责的八殿武者手中。”
<br/>　　对，无缘无故，不会有人特意去纠结一张通缉令。
<br/>　　毕竟中域之内，被通缉的人，多了去了。
<br/>　　那些个邪修，穷凶极恶之人，也多了去了。
<br/>　　“萧逸，我…”副院长脸色复杂，想说些什么。
<br/>　　萧逸摇了摇头。
<br/>　　他其实不必知晓具体的事情，但他知道副院长的为人，虽圆滑，但却有着自己的原则。
<br/>　　一个能当上药尊殿十大主殿之位的人，绝不可能会是个没有原则的人。
<br/>　　而这样的人，也绝不愿意违背自己的原则，甚至是那条自己在药尊殿内恪守了一生的原则。
<br/>　　“副院长，确实能说会道，甚至差点便把我带偏了过去，瞒了过去。”
<br/>　　“我只是为你好，我深知以你的性格…”副院长脸色复杂道。
<br/>　　“我知道。”萧逸点点头，打断道，“事实上，我甚至不愿意怪副院长你。”
<br/>　　副院长之前的最后两句话，充满着真挚，确实是真的担心他，关心他。
<br/>　　“即便当年副院长当众宣布逐我出黑云学教，我也未曾怪副院长。”萧逸缓缓说着。
<br/>　　“因为我知晓，副院长确实是个极好的前辈。”
<br/>　　“但，也是因此，偏偏不断骗我的人，是副院长你…我便才更加失望。”
<br/>　　萧逸的脸色，复杂到极点，也失望到极点。
<br/>　　副院长闻言，身躯蓦地一震。
<br/>　　特别是听到那句‘我才更加失望’时，竟是不自觉地呆愣住。
<br/>　　“唉。”半晌，副院长，无力地叹了口气，整个人，仿佛苍老了十几岁。
<br/>　　副院长未再言语。
<br/>　　萧逸，也不再言语，他的脸色，渐渐恢复了之前的冰冷，扫视在周遭一个个主殿主身上。
<br/>　　“在我了解清楚前，谁也不许走，胆敢动一步，那便休怪我剑下无情。”
<br/>　　冰冷的话语落下，萧逸看向剑姬前辈，“剑姬前辈，我们先回船上，我替你和大伙先疗伤。”
<br/>　　嗖…嗖…
<br/>　　萧逸、叶铭、白冰雪和剑姬前辈，身影一闪，回到飞行类亚圣器上。
<br/>　　“你…”高空中，一众主殿主，脸色一变，随后又一怒，纷纷看向承风殿主。
<br/>　　“承风殿主，你风刹殿的天骄，便是如此蛮横无理？还要威胁强留我们各殿主殿主不成？”
<br/>　　承风殿主耸耸肩，只当作听不到，直接不语，闭目假寐。
<br/>　　“承风殿主，你…”周遭一众主殿主见状，脸色难看到极点。
<br/>　　在场的最强者，乃是承风殿主。
<br/>　　若承风殿主不出手，他们谁也不是萧逸的对手。
<br/>　　不久前，他们还气势逼人地围攻着‘通缉者’。
<br/>　　再不济，也是在萧逸加入后，金绝玄强行要继续围攻，才被萧逸一剑反杀。
<br/>　　可现在，竟已是落到他们连退离的机会都没有的地步。
