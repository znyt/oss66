        对于傅夜沉的回答，童筱颖不以为然：“可我在梦里叫你大哥。”
<br/>
<br/>        这句话让傅夜沉感到语塞。
<br/>
<br/>        突然，一道银光晃过，傅夜沉剑眉微蹙，连忙从皮夹里掏出一张钞票放下，起身拉起童筱颖的手，说道：“我们得离开这里。”
<br/>
<br/>        当他们刚走出店门口，就有五个西装革履的男人拦住了他们的去路。
<br/>
<br/>        傅夜沉二话不说，便牵着童筱颖的手，和这五个男人厮打起来。
<br/>
<br/>        一片混乱后，这五个男人被傅夜沉打趴在地，童筱颖来不及去问那几个人的来历，就被傅夜沉拉着跑了起来。
<br/>
<br/>        不知不觉中，他们跑进了一片“红灯区”，打扮性感的女人，正站在路边招揽客人，一阵颓废的气息，与这城市的繁华显得格格不入。
<br/>
<br/>        傅夜沉带着童筱颖走进了一条暗巷，尽头的左边有一道生锈的铁门。
<br/>
<br/>        童筱颖只见他打开铁门后，轻车熟路地牵着她下楼梯。
<br/>
<br/>        楼道里的声控灯，随着他们的脚步声时而亮起时而熄灭。
<br/>
<br/>        走到一处平台，眼前却出现了三条路。
<br/>
<br/>        傅夜沉带着她，走左边这条路。
<br/>
<br/>        童筱颖不知道这条路拐了几个弯，路过了几张门，直到尽头出现了一台电梯，傅夜沉按了指纹锁，电梯门才打开。
<br/>
<br/>        她原本以为，他们是要上去，却没想到是下行。
<br/>
<br/>        电梯去了地下三层，门开了后，眼前又是另一番超现代风格的景象。
<br/>
<br/>        除了地面，墙壁全是镜子，就像迷宫一样。
<br/>
<br/>        傅夜沉将童筱颖牵紧，伸手推向镜子墙，镜子上突然出现五指解锁的绿色提示，镜子墙变成了一张旋转门。
<br/>
<br/>        进了这张门，室内的灯光一亮，金碧辉煌的屋子，让童筱颖觉得自己好像是走进了一幢豪华别墅。
<br/>
<br/>        “筱颖，你先在这里住下。等我把他们都解决了，你就安全了。”傅夜沉抬起手来，轻轻地按在了童筱颖的头顶。
<br/>
<br/>        裴金澈说，这个男人会保护她，看来是真的。
<br/>
<br/>        只是……
<br/>
<br/>        她以后，是不是不能出去了？
<br/>
<br/>        童筱颖知道，实验室的人在抓她，但是，今晚那五个黑衣人却不是实验室里的人。
<br/>
<br/>        实验室里的人若是要抓她回去，一定会对她使用麻醉枪，但是那五个人使得却是电击棒。
<br/>
<br/>        “他们想要找的是这个。”童筱颖忽然想起了什么，连忙从衣兜里掏出那张当票，递给傅夜沉，“我把它藏在了典当行里，你可以去把它赎回来。”
<br/>
<br/>        “你认为我值得你信任？”傅夜沉看着童筱颖手中的当票，苦笑着反问。
<br/>
<br/>        童筱颖却口直心快地回答道：“裴金澈说的，你会保护我。”
<br/>
<br/>        “你相信裴金澈的话？”傅夜沉心口一紧，有些隐隐作痛。
<br/>
<br/>        “我是‘药引’，他是‘辅药’，我和他同根，他不会害我。”童筱颖坦白地回答。
<br/>
<br/>        直到这一刻，傅夜沉终于明白了连华生所说的那句话是什么意思了。
<br/>
<br/>        筱颖，不再是以前的那个筱颖。
<br/>
<br/>        因为，他爱的那个筱颖，是真的已经死了。
<br/>
<br/>        现在的这个筱颖，对他无欲无求。
<br/>
<br/>        “你接下来，有什么打算吗？”傅夜沉岔开了话题。
<br/>
<br/>        童筱颖看着他，微微摇了摇头，长及腰际的发梢，也因为她摇头的动作微微飘动。
<br/>
<br/>        “裴金澈说得没错，我确实会保护你。等所有的事情都解决了，你就自由了。到时候，没有人再拿你当解药，也没有人再囚禁你。你想去哪儿，就去哪儿。”傅夜沉慢条斯理地说。
<br/>
<br/>        童筱颖微微一笑：“谢谢你，大哥。”
<br/>
<br/>        明明他和她是最亲密的人，现在却变成了最熟悉的陌生人。
<br/>
<br/>        听到这声久违的“大哥”，傅夜沉暗自苦笑。
<br/>
<br/>        她这是潜意识里把他当哥哥呢？
<br/>
<br/>        “这里什么都有，你的衣服，我明天去帮你买一些过来，今晚你穿我的衣服好了。”傅夜沉说着，带着她去了卧室。
<br/>
<br/>        童筱颖洗完澡出来后，傅夜沉拿来了吹风机准备给她吹头发时，她却将他手中的吹风机夺走，云淡风轻地说道：“我自己来就好。”
<br/>
<br/>        “好。”傅夜沉应了声，没有强迫她的意愿，并说了声，“晚安。”
<br/>
<br/>        童筱颖一边拿着吹风机给自己吹头发，一边看着傅夜沉离去的背影，心里莫名其妙地感到发慌。
<br/>
<br/>        她不明白自己为何会有这种感觉，就像起初不明白自己为何会悸动一样。
<br/>
<br/>        傅夜沉替童筱颖关上了房门，也就只是隔着这一张门，他心里落空的感觉，就像是一个深渊，让他呼吸凝重。
<br/>
<br/>        经历过这些事情之后，他也没料到自己再次见到她时会如此冷静。
