　　回到玉虚宫后，元始向着自己的四位弟子吩咐一声，便开始闭关了。
<br/>　　他首先整理自己此次得到的先天灵宝，将极品、上品、中品、下品与不入品级的灵宝各自分开。极品先天灵宝有两件，分别是中央戊己杏黄旗与尊神刀，上品先天灵宝有数十件，如，二十四颗定海神珠、玉虚琉璃灯、指天剑、辟邪剑、阴阳镜、落魄钟等。
<br/>　　中品先天灵宝那就更多了，什么火箭枪、遁龙桩、九龙神火罩、吴钩双剑、莫邪宝剑，混元幡等。至于下品灵宝与不入品的，那就更多了，元始都懒得数了。
<br/>　　特殊至宝一件，即是崆峒印，又名“封天印”，印座的四面以及中央五方大帝圣容，印玺下刻有大道符篆“崆峒”二字，执此印，可以废立人皇。
<br/>　　元始将自己用得到的收起来，用不到的，以后用来赐予弟子，或者用作以后自己炼器的炼器材料融掉。
<br/>　　最后，元始看着眼前的二十四颗地海神珠，默默道：
<br/>　　“只有二十四颗，有些可惜了，相传定海神珠共有三十六颗，汇聚三十六颗，则为极品先天灵宝，另外十二颗此刻又在何处？”
<br/>　　之后，元始一手持着二十四颗定海神珠，运转因果法则，顿时一条因果线以元始手中的定海神珠为起点，另一端没入虚空，消失不见。
<br/>　　过了不久，只见虚空波动，于那虚空之中飞来十二颗定海神珠，落入元始手中。元始轻声笑道：
<br/>　　“呵呵！原来是在龙族宝库之中啊。这下齐了，极品先天灵宝三十六颗定海神珠。”
<br/>　　恐怖！
<br/>　　因果法则，果然诡异，无声无息，便能取走他人宝库之中的宝物，有了这因果法则，哪还需要废其他更多力量去寻找呢？
<br/>　　“鸿钧已经合道，没了这柄悬在头顶的剑，洪荒以后会更加繁荣，各方势力崛起，东海那里也不能放任不管。也罢，便使善尸前往吧。”
<br/>　　言罢，一道身穿白衣、头顶白莲的道人从元始身体之中走出，两者记忆相同，他自然是明白元始的意思。
<br/>　　“见过道友，自今日起，吾为永恒岛主。”白衣道人向着元始行了一礼，道。
<br/>　　“道友去吧，顺便带上那四个弟子。”元始淡淡道。
<br/>　　“也罢，是该让他们回永恒岛了，毕竟那里才是他们真正的家。”永恒岛主如此说道，之后便出了玉虚宫，向着他的两位兄弟的宫殿去了。
<br/>　　过了不久，永恒便带着老子与通天一起出来，他们亦是派遣善尸跟着去东海，毕竟东海岛屿闲着也不是一个事儿。如此，永恒带着自己的四位弟子，与老子，通天斩去的善尸回到东海三清岛。
<br/>　　永恒岛主走后，元始便不再多言，开始参悟法则，准备尽快斩去自己最后一我——“超我”，然后证道。
<br/>　　之前道祖所讲再加上合道时的感悟，元始就有把握可以斩去第三个“我”了，如今只不过是整理一番，之前的功法还存在很多问题，现在等完善《斩我明道诀》后，便可斩去最后一个我，得见真我。
<br/>　　时间流逝，草木枯荣，元始沉浸在大道之中，感觉不到时间的感念，眨眼间便已过百年。
<br/>　　玉虚宫中，元始盘坐在道台之上，身影有些虚幻缥缈，气势内敛于身，让人完全看不出丁点修为。
<br/>　　“是时候突破了。”
<br/>　　元始喃喃道，在一年前，他便已经完善了《斩我明道诀》，不过他并没有急着突破，一直在调理自己的状态，如今一年过去了，也足够他调理的了，也是时候尝试突破了。
<br/>　　元始静静地立身于虚空之中，如同一尊雕像，没有任何动作，没有散发出任何气息，但却给人一种神圣不可侵犯的感觉。
<br/>　　“超我是未来的我，未来是虚无缥缈的，是不可预测的，是充满变数的。未来之我，与我何干？如今的我都还没有超脱，又何须去管未来？”
<br/>　　“斩断未来，不见未来！”
<br/>　　“不问前尘，不信后来，只尊今朝！”
<br/>　　“过去种种，全都烟消云散，未来一切，休挡吾道！纵然你是未来之我，也不行。”
<br/>　　“超我，给本座斩——”
<br/>　　“斩！”
<br/>　　“斩！”
<br/>　　三声大喝，坚定决心，不分生死，只为斩我。一柄乳白色的时光之刃缓缓出现元始头顶，那刀刃虚幻无影，充满时光的气息，道韵流转，玄妙无比。
<br/>　　蓦然间，只见那刀刃以一种不可思议的速度斩向元始自身，元始面不改色，眼中充满坚决，道心坚固不可动摇。此刻，若是他稍微有些动摇，那刀斩断的就是真正的自己了。
<br/>　　“哗啦——”
<br/>　　一声轻响，时光之刃透体而过，斩在了虚无之中，斩去了元始的未来。此时，一道身穿白衣，头发雪白的、满脸沧桑的男子从元始头顶飞了出来。这道身影刚开始还有些虚幻，但慢慢地，待他落地之后，便成了真实的存在。
<br/>　　这男子虽然满头白发，但面容依旧是一副三十来岁男子的面容，白发倒是给他增添了几分仙风道骨的气质。
<br/>　　“你为未来之我，你的遭遇倒是有几分与那传说中的原始有几分相似，那么，从今以后，你便叫原始吧。”元始看着白发原始，淡淡道。
<br/>　　“元始是元始，原始是原始，元始是我，你为原始，此原始，终究非彼元始。”
<br/>　　这一刻，元始只觉得自己全身枷锁尽开，有一种大解脱，得大自在的韵味在其中。这一刻，他仿佛认识到真正的自己，他只觉得以往的自己都不是他，那只是一种潜意识的自己，如今的自己才是真正找回真我的自己。
<br/>　　“哈哈！如履薄冰两世人，一往无前始作仙；
<br/>　　解开身上旧枷锁，今日方知我是我！”
<br/>　　顿时，一道恐怖的气势从元神身上升起，元始意志俯视诸天万界，静静地看着这个世界：
<br/>　　“我，元始，终会证道不朽！”
<br/>　　玄光绽放，神采飞扬，这一刻，他的脸上露出一股恬静地笑容，颇有自在逍遥之意，但又隐隐透出一股主宰天地的霸气威严。
<br/>　　“如今，本座已经可以证道成圣了。不过，吾终究是夺舍盘古元神所来的，虽说之前是因为盘古欠下本座因果，所以才有我夺舍一说。但，是别人的就是别人的，本座已经隐约有感，这样证道，不是最好的方式，那么——”
<br/>　　元始睁开眼睛，看向白发原始，点了点头，之后将元神之中的鸿蒙紫气取出，并迅速没入白发原始身上。
<br/>　　“自今日起，原始成圣，元始证混元。”
