　　轰！
<br/>　　下方废墟之中，一道金色身影爆射而出，直冲到九天云霄之上。

<br/>　　这是一名头发花白、一身金袍的威严男子，之前那股强大的威压，便是来自这金袍男子的身上。
<br/>　　“叩见陛下！”
<br/>　　周围那些无论是金甲将军，还是银甲侍卫，全都纷纷跪倒在地，恭恭敬敬的行了一个大礼。
<br/>　　原来，此人正是那汉天王朝的国主，宇文邦！
<br/>　　宇文邦双目若辰，在叶尘身上扫视了一眼，露出一抹浓浓的震惊之色，
<br/>　　“你是叶欢？不可能！你怎么可能在这么短的时间内，具有如此强大的修为？这绝对不可能！”
<br/>　　宇文邦感受到‘叶欢’身上的气息，不由得失声惊呼起来。
<br/>　　在宇文邦的印象中，眼前的这个‘叶欢’，虽然是一个难得的天才，可是毕竟年轻尚轻，记得他只有帅级境界的修为，如今竟然一跃达到了王级巅峰境界，就算比他也相差不了太多了！
<br/>　　不光宇文邦难以理解，恐怕整个毁灭大陆，也没有人可以搞懂怎么回事。
<br/>　　“你就是宇文邦？”
<br/>　　叶尘瞥了一眼眼前的这个金衣男子，叶欢的相关记忆，开始浮现在他的脑海之中，不由得冷冷开口道：
<br/>　　“我今日，为复仇而来！”
<br/>　　宇文邦听到这话，先是微微一愣，继而哈哈大笑起来，
<br/>　　“就凭你？虽然朕不清楚，你得到何种机缘，修为竟然进步如此之快，可终究只是王级境界罢了！”
<br/>　　“而朕，乃是皇级，足足比你高出两个大境界！在朕的面前，你不过就是小小一只蝼蚁而已，也敢口出狂言？”
<br/>　　宇文邦说完这话，身上陡然升起一股强大的气势，霸气侧漏！
<br/>　　“哼！”
<br/>　　叶尘冷哼一声，虽然以他如今的修为，想要击败皇级境界的宇文邦，还真不太容易，但是他已经参悟到了毁灭法则，就等于立于不败之地，只要不断的吸收对方的攻击，他的修为自然可以飞速暴涨！
<br/>　　“对付区区一个小辈，何须劳烦陛下出手？让末将来擒他！”
<br/>　　眼看大战一触即发，忽然从下方窜出一名金甲大将，直接张开大手，便朝叶尘狠狠擒抓了过来！
<br/>　　这人的修为，虽然不及那宇文邦，却也远远超过叶尘，估摸着应该是君级境界以上的实力。
<br/>　　“毁灭规则，同化！”
<br/>　　叶尘并没有着急动手，而是站在空中一动未动，任由对方的攻击落在了他的身上。
<br/>　　“竟然敢不还手？找死！”
<br/>　　那名君级强者显然没有想到，对方居然不躲不避，完全一副求死的样子。
<br/>　　不过，很快他就傻眼了，
<br/>　　他攻击落在叶尘的身上之上，不但没有伤到他分毫，攻击中包含的毁灭之力，被叶尘同化之后，气息反而大涨，竟然一举突破了王级境界，达到了君级！
<br/>　　“什么！”
<br/>　　这下所有人全都目瞪口呆，就连那宇文邦也两眼瞪得老大，显然无法理解眼前所发生的一切。
<br/>　　“我还就不信了！”
<br/>　　轰轰轰！
<br/>　　那金甲大将微微愣神之后，当即开始对叶尘疯狂的进攻起来。
<br/>　　然而叶尘依旧不躲不避，将他的攻击尽数吸收，同时修为也在稳步提升，不一会便提升到了君级中期，速度提升之快，简直到了匪夷所思的地步！
<br/>　　“住手！”
<br/>　　一旁的宇文邦率先反应过来了，当即冲着疯狂进攻叶尘的金甲大将命令了一句，后者当即停止了进攻。
<br/>　　“终于反应过来了？”
<br/>　　叶尘嘴角微微一扬，
<br/>　　“既然你们不打了，那就该轮到我了！”
<br/>　　声音落下，叶尘的身上，陡然升起一道淡灰色的刀芒，朝那名金甲大将，狠狠劈落了下去！
<br/>　　叶尘发出的这道刀芒，看似普普通通，似乎比没有什么威力。
<br/>　　“哼！你也就这点能耐吗？”
<br/>　　那金甲大将冷哼一声，直接一掌拍出，就要硬抗下叶尘这道刀芒。
<br/>　　然而，
<br/>　　轰！
<br/>　　灰色刀芒在空中划过之后，瞬间便撕破了金甲大将发出的巨掌，而且去势不减，狠狠劈落下来，自他头顶穿过！
<br/>　　滋滋滋~
<br/>　　一股难以形容的毁灭之力，瞬间便侵蚀了那名金甲大将的身体，竟然连一丝渣都没有剩下！
<br/>　　看到这一幕，就连之前一直自信满满的宇文邦，此刻眼中也露出一抹震惊之色。

<br/>　　宇文邦满脸凝重之色，死死盯着叶尘，忍不住大声冷喝道。

<br/>　　说完这话，叶尘再次凝聚出灰色刀芒，朝宇文邦斩落了过去！
<br/>　　宇文邦身为一国之主，而且还是皇级境界的顶级强者，此刻面对那看似不起眼的灰色刀芒，心底竟然生出一股强烈的危机感！
<br/>　　“国师救我！”
<br/>　　宇文邦一边飞速倒退，一边冲着下方大吼了起来。

<br/>　　眼看叶尘这一刀，就要斩落在宇文邦的身上，忽然下方一道灰色身影闪过，瞬间挡在了宇文邦的面前，而且竟然还真的挡住了，叶尘这充满毁灭法则的一刀！

<br/>　　轰隆！

<br/>　　“想不到这毁灭大陆上，竟然也有人领悟了毁灭规则，有意思！那就让我看看，是你领悟的规则之力厉害，还是我领悟的规则之力更强？而且说不定还可以让我，对毁灭规则有更深层次的理解……”
