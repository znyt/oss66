　　“叶圣府的那个小子？”
<br/>　　天煞府主看着萧逸身旁的傲然年轻人，脸上，头一次露出了忌惮之色。
<br/>　　这种忌惮，哪怕他之前见到公孙玄或者天星府主时，都不曾露出过。

<br/>　　“叶流。”萧逸同样笑笑。
<br/>　　不错，来人，正是叶流。

<br/>　　“可以。”萧逸点了点头。
<br/>　　“便是你们伤了怜星？”叶流冷傲的眸子，扫视着周遭之人。
<br/>　　天星大长老、金虎、万金府武者，无不一阵脸庞刺痛。
<br/>　　“叶流？”金虎眯了眯眼，“我想起来了，十八府天骄中，那个排在最后的废物。”
<br/>　　“素闻你向来风流，浪荡不羁，怎么，今日忽然醒悟了，要来逞英雄？”
<br/>　　“聒噪。”叶流眼眸一冷，手中一道流光凝聚而出，瞬间出手。
<br/>　　“拦下他。”万金府武者，冷喝一声。
<br/>　　十八个身处七星巨神阵的武者，再次狂猛攻来。
<br/>　　“你先挡一阵。”萧逸说了一声。
<br/>　　“行。”叶流应答一声。
<br/>　　一己之力，轻松对付十八人，以及那十数万金府武者。
<br/>　　嗖，萧逸身影一闪，闪身到顾怜星身旁。
<br/>　　“怜星姑娘，现在可愿意续命了？”萧逸轻笑问道。
<br/>　　“有劳萧逸公子。”顾怜星虽仍旧虚弱，但却展颜一笑，笑得无比轻松。
<br/>　　萧逸笑笑，手中一道星光打出。
<br/>　　星光，通过顾怜星的手臂，源源不断进入体内。
<br/>　　原本虚弱的脸色，渐渐涌起一股血色。
<br/>　　当然，顾怜星已自散星光，萧逸之前所做努力，早已前功尽弃。
<br/>　　故现今萧逸打入星光，也仅仅是替她续命一段时间。
<br/>　　另一边，叶流鏖战数十人，却是呈碾压之势，手中无一合之敌。
<br/>　　天煞府主见状，看了眼不远处正被萧逸救治的顾怜星，眼中霎时一冷。
<br/>　　眼眸中原本的忌惮之色，陡然化作了杀意。
<br/>　　“你要找死，便别怪本府主了。”天煞府主冷喝一声，瞬间出手。
<br/>　　天煞府主的速度，极快。
<br/>　　掌中，一道道阴煞之气凝聚。
<br/>　　叶流同样反映极快，手中数道流光凝聚。
<br/>　　嘭…
<br/>　　两掌对轰，天煞府主纹丝不动，叶流反被震退十数步。
<br/>　　顾怜星眼眸一惊。
<br/>　　“别急。”萧逸按捺下顾怜星，手中星光之力，远远不断打出。
<br/>　　“想在本府主眼皮子底下救人？”天煞府主看了眼萧逸，不屑一笑。
<br/>　　掌中阴煞之力凝聚，陡然而来。
<br/>　　“滚。”叶流脚步一震，激射而过，堪堪拦住了天煞府主。
<br/>　　啪啪…又是两声掌力对轰。
<br/>　　天煞府主纹丝不动，叶流被震退十数步，脸色微白。
<br/>　　“圣皇境九重？不自量力罢了。”天煞府主冷笑一声，再度出手。
<br/>　　叶流身影一闪，再度挡在了萧逸三人身前，双手齐出，六道流光，环绕而出。
<br/>　　一个星光图案，轰然从手中凝聚。
<br/>　　嘭…天煞府主的手掌轰来，在轰到星光图案之时，竟是阴煞之力悉数溃散。
<br/>　　叶流嘴角一丝鲜血溢出，却拦下了天煞府主的脚步。
<br/>　　“叶圣府的叶圣六绝，果然厉害。”天煞府主双眼一眯。
<br/>　　“不过，你还能挡本府主多少掌？”
<br/>　　锵…
<br/>　　恰在此时，一道冷白剑芒激射而来。
<br/>　　天煞府主一掌轰来，叶流双手齐出，流光卸去了大部分掌力。
<br/>　　萧逸一剑劈出，剑影震颤，瞬息之内，三十剑齐至。
<br/>　　嘭…一声爆响。
<br/>　　萧逸、叶流二人，冷笑一声。
<br/>　　天煞府主手臂一收，顷刻被震退十数步。
<br/>　　“好了？”叶流认真问道。
<br/>　　“嗯。”萧逸点了点头。
<br/>　　亏得他这三天时间星幻剑道再度突破，所施展之星光威力更强，效果也更大。
<br/>　　保住顾怜星的性命数日，应该问题不大。
<br/>　　叶流回头看了眼顾怜星，见顾怜星脸色不再苍白，松了口气。
<br/>　　“接下来，也该轮到我们了。”叶流擦了擦嘴角刚才的些许鲜血，战意滔天。

<br/>　　“好可怕的战意，这两个家伙是怪物吗。”金虎脸上，闪过一丝不可置信。
<br/>　　“退。”天星大长老低喝一声。
<br/>　　众人，连连后退。

<br/>　　“黄泉阴煞。”
