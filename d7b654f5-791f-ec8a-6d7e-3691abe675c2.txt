        探望阿景。
<br/>
<br/>        噢，不，其实，他只是想去看看筱颖。
<br/>
<br/>        可是，他却不敢去。
<br/>
<br/>        他害怕再见到筱颖时，会控制不住自己的感情，狠狠地将她搂入怀中。
<br/>
<br/>        筱颖……
<br/>
<br/>        被他埋藏在心底三年的筱颖……
<br/>
<br/>        落以柔站在一旁，无意间看到傅夜沉写的收货人名字为“童筱颖”。
<br/>
<br/>        傅夜沉写完这三个字后，笔尖在纸上停顿了一下，又握着笔把这三个字给划掉了，改写成了“傅昼景”。
<br/>
<br/>        他选择的是跨国次日达，明天，筱颖就会收到他寄给她的宝宝的礼物。
<br/>
<br/>        而寄货人，傅夜沉却用了自己爷爷傅炎廷的名字。
<br/>
<br/>        他也不知道自己这是怎么了，像是做贼心虚了一样，不敢跟任何承认，他是爱着筱颖的。
<br/>
<br/>        从店内出来后，傅夜沉很难得地主动开口问道：“你去哪儿，我送你。”
<br/>
<br/>        落以柔顿时仿佛受宠若惊般看着傅夜沉。
<br/>
<br/>        她怔了一下后，脸上露出了灿烂的笑容，一只纤纤柔荑般的手，很自然地就挽上了傅夜沉的臂弯。
<br/>
<br/>        落以柔一边拉着傅夜沉往前走，一边甜甜地邀请道：“不如，大哥陪我去逛超市吧！然后，我请你到我家里去吃饭。我的手艺很好噢！绝对不会输给酒店里的大厨，大哥你就放心吧！”
<br/>
<br/>        恍神间，傅夜沉在落以柔的脸上，看到了令他梦回萦绕的笑容。
<br/>
<br/>        “筱颖……”傅夜沉忍不住呢喃轻唤。
<br/>
<br/>        马路上车水马龙，声音嘈杂，落以柔并未听到傅夜沉说了什么，而是拽着他的臂弯，自顾自地往前走。
<br/>
<br/>        她这几声温柔而清甜的“大哥”，几乎将傅夜沉的心给叫软。
<br/>
<br/>        傅夜沉鬼使神差地走在落以柔的身旁，陪着她一起逛超市，却总是一副心不在焉的样子。
<br/>
<br/>        落以柔起初在挑选食材，但是挑着挑着，无意间发现，路过的几个女人，总是有意无意地往她这边看过来。
<br/>
<br/>        她在她们的眼神里看到了惊艳，随即柳眉微挑，心里顿时升起了一丝十足的得意感。
<br/>
<br/>        那几个女人，一定是在看她身旁的“大哥”。
<br/>
<br/>        很帅吧？很羡慕她吧！
<br/>
<br/>        落以柔心情大好地继续挑选食材。
<br/>
<br/>        “大哥”可是真的满足了她那丁点小小的虚荣心。
<br/>
<br/>        在收银台前买单的时候，落以柔刚掏出钱包，傅夜沉便已经主动把自己的银行卡递给了收银员。
<br/>
<br/>        女收银员在看到傅夜沉后，笑容比刚刚更加灿烂了，声音也格外清亮：“您好，一共肆佰叁拾元，这是您的卡，请收好。”
<br/>
<br/>        傅夜沉接过自己的卡收好后，将两大袋的东西全提在了自己手中。
<br/>
<br/>        落以柔此刻就像一个小女友一样，娇羞地跟在傅夜沉的身后。
<br/>
<br/>        傅夜沉开车将落以柔送回了家，但是到了落以柔家楼下，等他将刚刚买的那两大袋的东西全部提下车后，合上了后备箱车盖，径直回到了自己的驾驶座。
<br/>
<br/>        落以柔见状，惊愕地追了上去，纳闷地询问：“大哥，不是说好了我做晚饭给你吃吗？”
<br/>
<br/>        “……”傅夜沉什么话也没说，就连车窗也未落下，完全无视车窗外的落以柔，直接发动了车子，扬长而去。
<br/>
<br/>        落以柔瘪着嘴，刚刚喜上眉梢的好心情，此时一落千丈。
<br/>
<br/>        她都已经想好了要做什么菜给他吃了，他怎么就突然变卦了呢？
<br/>
<br/>        落以柔有些闷闷不乐地提着两大袋的食材回到了家中，她一边将食材整理放入冰箱，一边还在想着“大哥”的事情。
<br/>
<br/>        无意间，她才意识到，她至今都不知道他叫什么名字。
<br/>
<br/>        反正也没关系，她和他现在在同一家公司上班，总有机会再见面的。
<br/>
<br/>        晚饭虽然没有做给他吃，落以柔却在第二天为他准备了自制的办公室小零食。
<br/>
<br/>        而郑忠威虽然不在公司，却可以通过电话，远程“遥控”落以柔去替他办事。
<br/>
<br/>        当郑忠威无意间得知，落以柔把他家大少爷傅夜沉当成是总裁秘书的时候，忍俊不禁地说道：“小落，你就把文件交给那个男秘书就行了。”
<br/>
<br/>        “哦，好的！”落以柔点了点头，和师父郑忠威寒暄了几句，就把电话给挂了。
<br/>
<br/>        郑忠威打算让她继续误会，说不定这误会还能让他俩擦出意外的火花来。
<br/>
<br/>        到时候，他这位“忠臣”还可以当“媒人”。
<br/>
<br/>        所以，这次，落以柔带着文件和自制的小零食来到傅夜沉的办公室里时，非常落落大方地将自己的文件递给了傅夜沉。
<br/>
<br/>        “师父说了，交给你跟交给傅总一样。”落以柔微笑着说。
<br/>
<br/>        傅夜沉接过文件，直接“唰唰唰”地在上面签上了自己的大名。
