　　第592章
<br/>　　燕不归一楞，随即便觉得滑稽可笑。这年轻人当真不知道自己是何许人也。草原王族的大将，大长老的高徒，说是帝国第一高手也不过分。这年轻人竟然想让自己当他的手下。
<br/>　　随之燕不归一激灵，突然想起这年轻人是谁。在最近一段时间，有一波势力兴起，极短时间内便吞并了数个部落，堪称来势汹汹。
<br/>　　他们的首领好像就是个独眼汉子，名字唤作野原火。也不知是姓野，还是姓野原，关于这个人具体从哪里来的，倒也无人知晓。
<br/>　　燕不归完全没想到，这个人会突然出现在自己面前。他的手下意识握紧剑柄，虽然这个人冒出来的时间不长，但已经有许多部落注意到他。假以时日，或许他就会成长为戎族的心腹大患。
<br/>　　如果自己想要杀他，现在岂不是最好的机会。一剑在手，燕不归有把握在十步之内杀了他，他绝无逃脱的可能。
<br/>　　可杀还是不杀，燕不归还有些犹豫。毕竟对方刚才帮了自己，恩将仇报什么的，戎族也不太在意。真正令燕不归犹豫的是，杀掉对方实在太容易了，容易得令人感觉不太真实。
<br/>　　“你想杀我？”
<br/>　　燕不归抬起头来，发现对方眼睛正盯着自己，目光宛若实质，恍惚有看透人心的力量。
<br/>　　“如果你现在不愿意做我的手下，日后或许会想通的，当你想通的时候，记得来找我。”
<br/>　　对方口中一声呼哨，一匹快马从远处奔来。他翻身上马，临行前还看了燕不归一眼。
<br/>　　就是这一犹豫，燕不归已错过杀死对方的机会。等对方走后，燕不归才发现后背已被冷汗打湿，便是一注视的光景，对方便给他带来巨大压力。
<br/>　　不过杀不了便杀不了吧，燕不归也不是很在乎。他不是个心怀天下的人，对戎族的未来也不担忧。争权夺势的事情，他从未考虑过，那并不是他所在乎的。他所在乎的人只有一个，现在那个人已经死了，燕不归想的便是为消失的那个人做些什么。
<br/>　　提着剑继续北上，他正抓紧时间赶回戎族王城。
<br/>　　草原上，一匹骏马奔驰，野原火……或者说曾经的福德勒单人独骑。他只是突然心血来潮，想在青牛山转转。故地重游心情也谈不上有多好。因为和程大雷一样，青牛山所埋葬的，都是不太好的回忆。
<br/>　　野原火并不是很想记得曾经的事，但确实又是这些往事一步步把他推到今天。
<br/>　　骏马在草原上荡起烟尘，前方一排营帐，野原火从马上下来，立刻有手下人迎过来。
<br/>　　“王上，人已经到了，正在等您回来。”
<br/>　　野原火将马鞭交给手下，道：“带他们来见我。”
<br/>　　齐如海是来自青州的商人，兵荒马乱的生意也不好做。在青州，齐如海也算不得什么有名气的商人，官兵吃拿卡要，时不时土匪山贼还来抢你一次，日子委实过得不如何。齐如海也算是见过世面的，大风大浪也经历过，但这次生意，仍旧令他提心吊胆。
<br/>　　毕竟，这次生意对象是戎族。不得不说，戎族是很讲诚信的存在，这一点要比帝国好许多。齐如海所做的事情，就是将帝国的特产卖给戎族，再从戎族那里换来奴隶马匹什么的，凭此获取利益。
<br/>　　这种生意也并不是没有利润，但却很危险。不提路上有可能遇见的强盗土匪，最危险的反而是被官府知道。朝廷是禁止与戎族经商的，喊出的口号是不许寸铁出关，要是齐如海被官府发现，那就是满门抄斩，诛灭九族的事。
<br/>　　不过生意人么，利润只要达到百分之一百，天王老子他们都不会认，更何况是朝廷的律法。
<br/>　　有人来请齐如海，他忐忑的跟在对方身后，也不知等待自己的是什么。等进入部落首领的大帐，见到对方首领时，齐如海顿了顿。
<br/>　　对方看上去年纪并不大，腰背挺直，立在那里如一棵青松。齐如海倒是没有想到，现在草原上杀威赫赫的野原火，不过是个模样普通的独眼年轻人。
<br/>　　具体事项双方已经交流过，简单来说，便是对方向他购买一批刀枪运到草原。连茶叶丝绸都不许卖给戎族，何况是铁器，而且对方要求的数目委实不小。不过，只要钱到位了，也没有什么生意是不能做的。
<br/>　　齐如海担心的就是：对方可能没钱。
<br/>　　如此，他才一直拖拖拉拉，不肯给个痛快话。
<br/>　　野原火拍了拍手，有手下搬来一口箱子。在箱子打开那一刹那，齐如海的眼睛突然直了。
<br/>　　里面堆满了金银珠宝，金灿灿的黄金，白生生的银子，还有成串的珍珠，碧绿的翡翠……
<br/>　　如今帝国战乱，整个货币制度几乎处于半崩溃状态，大家多是习惯以物易物。但今天这口箱子里的东西，无论在什么时候，都是硬通货。
<br/>　　令齐如海想不到的是，这伙人从哪里寻来这些金银珠宝，一般和戎族人做声音，他们都是给些皮毛，给些草药。
<br/>　　不过，这也并非是他所关心的，只要有钱赚，其他事情都可以谈。
<br/>　　“首领放心，您需要的货物不日就可以运到，如果阁下不放心，我可以留在这里做担保。”
<br/>　　敢提着脑袋做生意，齐如海也知道信用二字的重要性。
<br/>　　野原火点点头，对齐如海的话倒是没有多少反应。他只是慢吞吞的问道：“这里只是一部分，事成之后，还有另外一部分。”
<br/>　　齐如海的眼睛烁烁放光，心中感慨道：这个部落还真是有钱呐。
<br/>　　“除了事先谈好的武器，我还需要另外一些东西。”
<br/>　　“只要首领开口，您想要什么货物，尽管包在我身上？”
<br/>　　“这次不是货物，而是人。”
<br/>　　“人？”齐如海一楞，随即便恍然大悟。
<br/>　　看对方的样子，说不得是想要帝国花花世界的女人，年轻人么，这一点也不是不可理解。
<br/>　　“我想要一批读过书的，认识字的人。”
<br/>　　齐如海脸上的表情愣住，难以置信道：“读书人……能做什么？”
<br/>(本章完)
