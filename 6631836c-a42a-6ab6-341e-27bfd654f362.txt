　　以美国海军陆战队情报室为主导，其他国家情报机构辅助的情报系统全力运转，一份接一份情报传递到“鹰巢”，几乎每隔一个小时都会有新的情报传递过来。
<br/>　　再加上有四颗近地轨道卫星的配合监控，西伦科特的行踪彻底暴露在联合行动司令部的眼皮子底下。
<br/>　　联合行动司令部的参谋团对情报进行梳理筛选，并且制定行动计划，而詹姆斯在得到了白宫授权和支持后，直接跟该国总统进行对话，说明了自己这边的来意。
<br/>　　面对美国这个有着全球最强大空军和海军，并且流氓气息十足的超级军事大国，这个位于南美洲中部的内陆国家的总统根本没有说“不”和“拒绝”的权利。
<br/>　　拉蒙，这个在三年前那次缉毒行动中就跟美军特种部队有过交手的家伙，成了这次联合行动中的牺牲品。毕竟在三年前那次缉毒行动是国际刑警和美军特种部队的联合行动，拉蒙作为制毒贩毒组织的保护伞涉入其中，是要付出相应代价的。对于詹姆斯来说，让拉蒙多活了三年时间，已经是上帝的恩赐了。
<br/>　　“小伙子们，我们的目标已经确定了。”詹姆斯将各国特种部队队长召集到三维电子沙盘前，指着上面一个用红色圆点进行标注的别墅三维影像，说道：“西伦科特这个家伙就藏在这栋山顶别墅中。”
<br/>　　听到詹姆斯的话，几个国家的特种部队队长都是跃跃欲试，恨不得现在立刻就出发把西伦科特给解决掉，毕竟这个家伙手上，沾着这次联合行动所有参与国家特种部队队员的鲜血。
<br/>　　“各位，根据我们情报系统所提供的情报显示，西伦科特离开中东的时候，带走了一批我们的人质，而这批人质现在也在南美洲。”詹姆斯用手中的激光笔点出几个位置，说道：“这几个位置就是人质被关押的地方，当然在西伦科特藏身的山顶别墅中，也关押着人质。这次，我们不仅要西伦科特的命，还要把剩余的人质全都营救出来，这次行动我们拥有最大的行动权限，也就是说，小伙子们你们可以动用一切手段采取行动。”
<br/>　　“麦克米兰中校！”詹姆斯对着站在一旁的麦克米兰招呼了一声。
<br/>　　“是的，将军！”
<br/>　　麦克米兰“啪”的一声站直身体，看着詹姆斯。尽管詹姆斯还什么都没有说，但是麦克米兰已经大致能猜出詹姆斯后面会跟自己说什么，麦克米兰脸上的神情有些激动。
<br/>　　詹姆斯把一张拉蒙的照片扔给麦克米兰，说道：“三年前，你在他手下吃了亏，你的副队长也死在他的手上，现在他是你的了！海军陆战队会配合你们的行动！”
<br/>　　“是！”
<br/>　　麦克米兰上前一步，拿起桌子上的照片，用力的握紧，眼睛中喷吐着愤怒的火焰。
<br/>　　“少校！”詹姆斯看向罗昊，说道：“这次斩首西伦科特的任务就交给你和你的小队了，虽然你们只有四个人，但是你们的战斗力有目共睹，我相信你们能把西伦科特那个混蛋的脑袋砍下来！”
<br/>　　“明白！”
<br/>　　罗昊眼神中透露着无比的坚定。
<br/>　　经过了一天的准备，各国特种部队依次按照各自的任务出发前往目的地，罗昊、李毅峰、李然、慕容飞四个人是最后出发的，按照计划，直升机会把他们运送到西伦科特藏身的山顶别墅上空，他们将在山顶别墅周围的树林中进行伞降，然后自行设法进入别墅，对西伦科特实施斩首行动。
<br/>　　罗昊站在桌子前，将散在桌子上的子弹一颗接一颗的压进手中高强度工程复合材料弹匣中，这次行动所使用的步枪子弹和手枪子弹，并不是常规弹药，而是钨芯穿甲弹！这种穿甲弹比起常规的钢芯穿甲弹，穿透性要更加优越，可以更有效的杀伤掩体背后，以及穿着防弹背心的目标。
<br/>　　将最后一颗子弹压入弹匣后，罗昊把桌子上压满子弹并且用黑色绝缘胶布两两缠裹在一起的弹匣塞进战术背心的弹匣袋中，最后把剩下的那组弹匣装入桌子上那支在短枪管上安装了消音器的G36C自动步枪中，把枪背在身上，对着桌子边其他三个人招呼一声，朝着厂房外的直升机走去。
<br/>　　云熙站在厂房门口，看到罗昊走出厂房，立刻走上前去，伸手替他整理了一下缠在脖子上的阿拉伯方巾，叮嘱道：“注意安全。记住，你还欠我一个孩子，在没换给我之前，你可不能死了。”
<br/>　　罗昊身后抱着云熙后脑勺，在她微凉的额头上浅吻了一下，说道：“在没有把那个孩子还给你之前，我还舍不得死呢。我一定会平安回来。”
<br/>　　说着，追上李然、李毅峰、慕容飞三个人，爬上了那架负责运送他们的“黑鹰”直升机。
<br/>　　“呼呼呼……”
<br/>　　直升机的螺旋桨飞快的旋转起来，吹起的气浪令得站在旁边的云熙向后倒退了两步，伸手挡在眼前，阻挡吹向自己的尘土。
<br/>　　“黑鹰”直升机缓缓起飞拉升高度，上升到一定高度后，在空中调转了方向，朝着西伦科特藏身的山顶别墅的方向飞去。
<br/>　　半个小时后，坐在“黑鹰”直升机中，罗昊已经能看到远处山顶上那栋装修豪华的别墅。
<br/>　　情报显示，这栋位于山顶的豪华别墅并不是西伦科特名下的产业，而是属于当地一个制毒贩毒组织头子布克尔特所有，这栋别墅是布克尔特跟当地政府和军队高官商谈合作，为他们提供特殊服务所用，是一个藏污纳垢之处。
<br/>　　不过，西伦科特在之前跟这个布克尔特并没有任何接触，就连詹姆斯也想不明白，为什么西伦科特一到南美洲就能跟布克尔特迅速搭上关系，并且让布克尔特将这栋别墅拿出来给西伦科特作为藏身之处，就是用脚趾头想都能想到西伦科特和布克尔特之间有关系，但是没人知道他们之间到底是什么关系。
