<br/>    那位人王气得吹鼻子瞪眼，喝问道：“哼，小小的一个入道境，也敢在此妄言大道，我看，你也就嘴上这点本事了，若不是如此，你为何现在还是入道境，连悟道境都突破不了？”

<br/>        “后面那一句更是可笑，他若是超脱，便是群仙也要跪拜于前，开什么玩笑？”

<br/>        “此人，怕不是失心疯了吧？”

<br/>        后面那些群星仙门的长老本以为让卓青瑶出来商谈有戏，结果听到这句话，瞬间心里就凉了半截，和里面那位比谁丹药多，这不是找虐呢？
<br/>    
<br/>        那位可是一个月出手百万枚眼睛都不眨一下的存在，听到他们的报价，自己都觉得脸上火烧的羞愧。
<br/>    
<br/>        卓青瑶冷冷道：“抱歉，我们燕山，就是记名弟子，闻道境的丹药也不限量。嗯，就是把丹药当饭吃的那种。”

<br/>        把丹药当饭吃.......
<br/>    
<br/>        那位人王的眉头跳了跳，记名弟子闻道境丹药不限量！这是个什么概念，就是说人家的记名弟子，都比他们宗门培养传人的待遇还要好！
<br/>    
<br/>        这个宗门，是有着一块能够长出来丹药的地吗？
<br/>    
<br/>        那位人王整理了一番自己道袍，擦了擦额头的冷汗，若是陈明说出来，他自然是不信的，但是卓青瑶说的，他还是信的。毕竟闻道榜第一，有着闻道榜第一的骄傲。

<br/>        卓青瑶将背后的巨阙拔了出来，“灵兵？有这柄剑好吗？”

<br/>        人王愣住了，差点忘了，这位小祖宗可是有着神兵巨阙！

<br/>        什么灵兵，就是国器，也比不上这一柄巨阙啊！

<br/>        看看人家的宗门，亲传弟子都是拿着神兵榜上的仙器，一般这种重宝，都是宗主，或是太上长老才有资格拥有的，这到底是一个什么样的宗门啊，燕山，没听说过啊！

<br/>        别说他们了，就是古仙门，也没有这么奢侈啊！
