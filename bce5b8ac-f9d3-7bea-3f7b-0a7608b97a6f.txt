　　数年后，人间界再次轰动，一道浩瀚的气息直冲云霄，肯与苍天比高，与日月争辉，天地皆黯然失色，一股正气凌天。
<br/>　　与此同时，一道充满浩然正气、人道不屈信念的声音传出，声音之中，满是正义与大义，响彻周天：
<br/>　　“君怀德，小人怀土；君怀刑，小人怀惠。君子坦荡荡，小人长戚戚……君去仁，恶乎成名？”
<br/>　　“知者乐水，仁者乐山。”
<br/>　　“人而无信，不知其可也。”
<br/>　　“言必行，行必果。”
<br/>　　“学而不思则罔，思而不学则殆。”
<br/>　　“敏而好学，不耻下问。”
<br/>　　“己所不欲，勿施于人。”
<br/>　　“……”
<br/>　　轰！
<br/>　　人间之中，一股巨大的气势冲天而起，这股气势与修为无关，仿佛是从人内心灵魂深处散发而出。在这股气势出现的瞬间，气势直接将周边的空间完全覆盖，一阵浩大的声音在这股气势之中不断响起。
<br/>　　“天地有正气，杂然赋流形。下则为河岳，上则为日星。于人曰浩然，沛乎塞苍冥。皇路当轻夷，含和吐明庭，时穷节乃见，一一垂丹青。”
<br/>　　随着这一道道吟唱响起，天地间的气势陡然变得更加强大了，这气势的面目也彻底显露出来，这是一种洪荒之中早已存在但却无人发觉利用的气息，这是浩然正气，存在于天地之间，无处不在。
<br/>　　与此同时，这气势也惊动了洪荒诸多大神，诸神纷纷探查，发觉发出这股气势的正是人族鲁国的孔丘，亦是阐教孔宣的善尸转世之身。
<br/>　　瞬间，诸神更是将孔丘的身世及一身经历都演算了出来。
<br/>　　原来这孔丘还是贵族，是当初商朝皇室的后裔，是纣王的兄弟微子启的后人。孔丘降生时相貌丑陋怪异，被人弃之荒野，天气炎热，苍鹰降下振翅扇风为他驱赶酷暑，一只母虎将他叼进虎穴，用虎乳喂养他，这才让他活了下来。
<br/>　　后来母虎将孔丘还给孔子母亲，孔母悉心照料。
<br/>　　可惜，孔母是个小妾，地位极低，孔丘少时被父亲以及正妻赶了出来。
<br/>　　孔丘少年时候非常贫困，所以会很多粗活计，能够养活自己。
<br/>　　孔丘年轻时候很平凡，但他的梦想却是希望自己有朝一日能够入仕为官，帮助国家，可惜，鲁国一直不曾重用他。
<br/>　　孔丘不放弃，他开创了私学，教导学子，他认为人不分贵贱，人人都有机会接受教育，人的性格会受外界环境影响，只有接受教育，才能将人导向正途。
<br/>　　孔丘一生收了三千弟子，其中包括颜回等七十二贤士，孔丘之名，渐渐传开。
<br/>　　后来，鲁国内乱，孔丘被猜忌，无奈，他只好带着自己的弟子们开始了长达十三年的周游列国，一路上吃苦不尽，先后经过卫、陈、曹、楚、宋、郑、吴等国，最后才返回鲁国。
<br/>　　老年的孔丘回到鲁国当官，当的是少有的大官大司寇，但最后因被别的官员排挤而辞官。
<br/>　　辞官后，孔丘重新修订《诗经》、《尚书》、《礼记》和《乐经》，并为周文王的《易经》作序。
<br/>　　最后，孔丘亲笔编写出《春秋》，此书一成，才气冲销，天花乱坠，群星天照，千里云霞，万里雷鸣，天下震动。
<br/>　　也正是此刻。
<br/>　　在洪荒诸神的关注下，孔丘表情严肃，身上气势直冲霄汉，连接天地间，天上雷霆不断轰鸣，给人的感觉是要毁灭万物，而孔丘身上的气息与之相似，同样刚正无比，但却又不同，这种光明正大的气息之中，是一种令人从心底臣服的感觉。
<br/>　　“哼！”
<br/>　　面对天上的毁灭雷霆，孔丘冷哼一声，手轻轻举起，袖袍向后一挥，整个人绷直了起来，大声吟唱道：“是气所磅礴，凛然万古存。当其贯日月，生死安足论！地维赖以立，天柱赖以尊，三纲实系命，道义为之根！”
<br/>　　“天地浩然，正气长存，不求诛仙，但斩鬼神！”
<br/>　　孔丘一步步走上虚空，与雷霆对立，他左手握着“春秋书”，右手持有“春秋笔”，看到天上无尽会没雷霆，他依然不惧，大声道。
<br/>　　天地雷霆更加暴虐了，只见孔丘提笔，风起云涌，天地变色，春秋笔接连写下九个“斩”字，一字一剑，化成九道凌厉正气的剑光，冲上雷霆之中，将无边雷霆斩成十份碎片。
<br/>　　随后，孔丘一手将手中《春秋》书扔出，《春秋书》瞬间变大，遮天三万里，大书一动，将所有雷霆卷灭、轰碎。
<br/>　　“吾为人族孔丘，有幸悟文祖之文道，走出浩然儒道，今日立下儒道，将来为天地立心，为生民立命，为往圣继绝学，为万世开太平！”
<br/>　　“儒道，立！”
<br/>　　随着孔丘最后一声大喝，天地一阵光亮，群星摇曳，随后，一片功德金光从天而降，落入孔丘身上，他一身儒道修为不断往上增长，最后来到儒道亚圣，方才停止。
<br/>　　孔丘胸藏日月山河，心怀天地浩然，文气造化，一举一动，莫不带着圣人威严。
<br/>　　“孔子，儒圣！”
<br/>　　“孔子，儒圣！”
<br/>　　“孔子……”
<br/>　　万民震动，纷纷跪下，朝着孔丘方向跪拜，口中惊呼孔子，当为儒家圣人。
<br/>　　孔子开创儒道，最有资格加称“子”，当为孔子。
<br/>　　随后，孔子创办了第一家书院曲阜书院，广招学生，教授儒道。
<br/>　　“儒道，又开创了一种修炼大道，而且这种大道就好似武道一般，最为适合人族修炼，人族果然卧虎藏龙。”
<br/>　　“孔丘，儒家，又是一家崛起，接下来又会是哪家呢？”
<br/>　　“阐教孔宣，更是厉害，善尸走出一道，距离突破恐怕也不远了。”
<br/>　　“……”
<br/>　　天地间，诸神议论纷纷，无不是赞扬孔子之语，同时，他们对于其他的百家之人，也更感兴趣了。
<br/>　　昆仑山，元始暗自点头，孔宣的表现他都看在眼里，非常不错，如今更是阐教众弟子中修为最高之人了，就连广成子也比不上。
<br/>　　人族祖地，文宫中，文祖仓颉也是大笑不已，自当初文祖领悟文道后，还没有人能将文道修炼至如此地步，更何谈创新之说，如今孔子在他的文道之中，又走出一道，这才是他想要看到的。
<br/>　　人族越来越强，他便越是高兴。
<br/>　　“孔子此人，不错！”
<br/>　　数年后，主张“兼爱、非攻”的墨家墨子出现，建立墨家，墨家更是开创了一手精妙绝伦的机关之术，无论是民用，还是战争，机关术都无比适合人族，节约了人们的精力，开创一条科技的先河。
<br/>　　与此同时，得到神农地皇传承之人，广结天下农民，传授开荒垦地，种植农业的经验，成立了农家。
<br/>　　随后，“大道阴阳，无极太一”、以金木水火土五行之术钻研天地间一切对立互化的阴阳家出现，“医治天下”的医家出现，注重辩论“名”与“实”的名家出现，以“诡辩”闻名天下。
<br/>　　后来，“不别亲疏，不殊贵贱，一断于法”，以“术”、“法”、“势”震惊天下的法家出现，法家法尊更是一位绝世高手，一手法家刑法判定天下之法，争斗起来，更好似有如天助一般，能够借天地之势，化天地为牢笼，聚雷霆之力为审判，汇诛神之力为刑刀，执天地之法，绝世无双！
<br/>　　与法家同时出现的还有神秘莫测的纵横家，纵横家之所以说是神秘莫测是因为鬼谷子只是出来建立纵横家，并喊出“苍生涂涂，天下缭缭。诸子百家，唯我纵横”的霸气口号，随后便消失不见，故而让人感觉神秘莫测。
<br/>　　随后，“兼儒墨，合名法”的杂家出现，之后更是出现了大大小小的“百家”，但大多都没有多少本事，不似前几家那般厉害。
<br/>　　春秋战国，诸侯争霸，诸子百家，逆乱天下！
<br/>　　一个“万马奔腾，百花齐放”的百家争鸣大世正徐徐向世人展开它神秘的一角。
