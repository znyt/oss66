        &lt;h3&gt;第三卷第三十七章  傅海生的纠结&lt;/h3&gt;
<br/>
<br/>        傅海生满怀心事的回到了洛城。
<br/>
<br/>        和江俞轩在一起聊了一个上午，可以看出来，江俞轩的专业知识是很强的，他也听过陈子昂他们公司的事迹，但是他并不想千篇一律的做事，每个部门有每个部门的需求和发展方向。
<br/>
<br/>        他们是一个地市级单位，虽然说发展方向是一致的，但是还是要考虑到适用性和可塑性。
<br/>
<br/>        陈子昂是他心里无法抹去的爱恋，是他这一生都无法企及的梦想。
<br/>
<br/>        他们阴差阳错的错过，又不期而遇，如今又让他得知真相，如果陈子昂知道他的妻子、他的父亲就是导致她不能上大学的人，她会不会恨他呢？
<br/>
<br/>        和陈子昂在一起的情景历历在目，每当想起给陈子昂涂药的情景，他的心就无端的幸福和失落，也许是人们说的，越是得不到的，越是想去得到，也越是珍惜。
<br/>
<br/>        他也想对夏颜好，而且他也做到了对夏颜好，既然结婚了，就要做一个负责人的男人，当初他的工作简单而单调，上班回家，陪老婆孩子。
<br/>
<br/>        家里的家务活都是他干，夏颜连袜子都不洗，回家了，臭衣服，臭鞋子到处乱放，他们的卧室常常都是夏颜堆得到处都是的脏衣服。
<br/>
<br/>        没有孩子还好说，他天天帮着洗洗刷刷，孩子生下来之后，夏颜也基本不管，他是又当爹又当妈，晚上都要起来好几次给孩子冲奶粉。
<br/>
<br/>        母亲要上班，也不能帮着他带，他是又苦又累，夏颜还是和在学校一样，该咋玩就咋玩，有时候整晚不回家。
<br/>
<br/>        偶尔他唠叨几句就开始和他吵，一吵起来就没完没了，后来竟然扯上了陈子昂。
<br/>
<br/>        夏颜只要提起陈子昂，傅海生就充满了失落，他会不自觉的拿陈子昂和夏颜比较，陈子昂在上学的时候，出了名的学习好，人热情，懂事，夏颜除了出身比陈子昂要好一点之外，哪里能跟得上陈子昂？
<br/>
<br/>        他知道这样比较是不对的，他也努力的包容夏颜的一切，他也知道过去了就过去了，他要珍惜现在的生活，尤其是有了孩子之后，他的精力一半都放在了家庭上，作为一个男人，他是有梦想的，梦想成功、梦想做出一番事业。
<br/>
<br/>        记得哪一次，他们为孩子的问题再一次发生了争执，原因是，夏颜下午不上班，他就让她带着孩子，等他回来，夏颜在隔壁打麻将，孩子一个人倒在地上睡着了，寒冷的冬天，孩子就那样的睡在地上。
<br/>
<br/>        他很是生气，将夏颜喊回来，俩个人就发生了争吵，吵着吵着，夏颜将他给陈子昂写的信拿出来，扔在了他的脸上和身上，那一刻，他觉得浑身冰冷，他才知道，为什么陈子昂收不到他的信，回忆起和陈子昂仅有的一次见面，他知道他和陈子昂都中了夏颜的圈套。
<br/>
<br/>        那一刻，他是恨的，恨夏颜，他坐在地上看着那些信件，他觉得一切都没有意义了。
<br/>
<br/>        后来，母亲回来了，他才惊觉，他生活不是为了自己，他是一个儿子，一个父亲，既然，陈子昂已经从他的生活中消失，那么就让这一切都过去吧。
<br/>
<br/>        当他和夏颜在一起的时候，他会自我安慰：“夏颜是因为爱他，才会截回他写给陈子昂的信。夏颜是因为爱他，是啊，都是因为爱。”
<br/>
<br/>        傅海生不再是那个懵懂的青春热血而又痴情的少年了，他在一瞬间长大了，担起了家庭的责任，他也努力的修复着和夏颜的关系。
<br/>
<br/>        “颜颜，过去的事情已经过去了，我和陈子昂之间什么都没有，你看信都被你截下了，陈子昂也什么都不知道，况且，我们已经很多年都没有见她了。”
<br/>
<br/>        “可是，你心里还有她，你连做梦都在喊着她！”
<br/>
<br/>        “我原本不再提起她，可是你经常提醒我，要我去回忆她。”傅海生无奈的说，如果说有忘情水的话，他选择喝下忘情水，忘掉陈子昂，毕竟，那只是春春的梦想，是他暗恋的对象。
<br/>
<br/>        只是没有，在岁月的长河中，在他的生命里，在他的心里某一个角落里，陈子昂，深深的扎根在某一个地方。
<br/>
<br/>        “你还怪我了？你和我结婚这么多年，你看看我过的是什么生活？”
<br/>
<br/>        “我们这种生活你还不满意吗？你都可以说是：衣来张口，饭来伸手了，孩子不用你管，你想干什么就干什么，你想出去嗨皮就出去嗨皮，你想打麻将就打麻将，你想不回家就不回家，你还想我怎么样？”
<br/>
<br/>        “傅海生，自己没有本事就不要说了，你也就是一个能洗洗衣服，带带孩子，做做饭的男人，工作了这么些年，一点上进心都没有，看看我们大学的几个同学，你看看人家现在混得多好，我就不知道当初怎么就瞎了眼睛看上你。”
<br/>
<br/>        夏颜的话像刀子一样戳在傅海生的心中，他现在已经是单位的副总了，他努力的工作，努力的做好一个丈夫和父亲及儿子的责任，他下了班就回家，接孩子、管孩子，不抽烟不喝酒不应酬。
<br/>
<br/>        同学聚会，夏颜说：“海生，如果我们两个都去的话，就没有人看孩子了。你就不去了吧。”
<br/>
<br/>        有应酬，夏颜说：“傅海生，你天天在外面应酬，孩子不是你的啊?”
<br/>
<br/>        他在单位加班，夏颜跑到单位，将孩子给他一扔：“借口加班是吧？带着你的孩子一起加班。”
<br/>
<br/>        他的工作卡交给她，夏颜永远是一天一身衣服，每天打扮得花枝招展，脏衣服都由他来洗，他永远就是那么三四套衣服，母亲看不过去了，会拉着他去给他买衣服，到现在他还在花着母亲的钱。
<br/>
<br/>        傅海生终于明白，夏颜爱的只是她自己。
<br/>
<br/>        他调到洛城一年后，夏颜通过关系调到了西城，他们过着两地分居的生活，傅海生终于从家庭的束缚中出来了，他努力的工作，如今已经是现在这家单位的老总，在业界享有很高的声誉，论文发表了很多篇，也会经常被请去兄弟单位讲课。
<br/>
<br/>        他原本想着，夏颜愿意维持着这样的生活，为了孩子，为了有一个完整的家，那么就过着这样的生活也很好，俩个人见面的机会少了，反而没有那么多矛盾。
<br/>
<br/>        可是陈子昂出现了，陈子昂的出现让他发现了真相，夏颜不但是刁蛮、任性、自私、她们的德行还有问题。
<br/>
<br/>        傅海生闭上了眼睛，他是要还陈子昂一个真相，还是将这件事情深深隐藏，这还牵扯到了他深爱着的父亲。
