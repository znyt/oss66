
<br/>　　<br/>　　    车回到市区之后，先把午桥送到医院去看望了王生，然后才又回到了家里，只不过回家的时候却多了个人，小草义正言辞的跟王惊蛰说，她一妙龄女子无处可去，无依无靠的，在哪都不安全，就只能暂时寄宿在这里了，至于房间的问题也简单了，丁武没出院呢，王惊蛰就睡他那屋了，他被收拾的干干净净的房间则被茅小草理所当然的给霸占了。<br/>　　<br/>　　    菜刀文当即就感叹了一句：“原来是在这等着呢，早有预谋了，我说白天怎么把屋子收拾那么干净，这是给自己先铺好后路了啊”<br/>　　<br/>　　    车停好了后，茅小草先进去了，王惊蛰拉着菜刀文小声问道：“她跟我们住合适么？”<br/>　　<br/>　　    “男女合住，你觉得谁更吃亏谁更危险一点？”菜刀文斜了着眼睛问道。<br/>　　<br/>　　    王惊蛰想了下说道：“女的吧？”<br/>　　<br/>　　    “那不就得了，她自己都不担心呢，你操的什么心啊，踩着你尾巴了？”<br/>　　<br/>　　    王惊蛰挠了挠脑袋，总觉得哪里有点不太对劲，但却又说不出来，菜刀文拍着他肩膀说道：“你和午桥能今天出来，怎么说也是这姑娘出的力，人家来住几天或者是一段时间，这是应该的，明白么？还有，你千万不要去尝试跟她讲什么道理，你一张嘴是斗不过五百只鸭子的”<br/>　　<br/>　　    “好吧……”<br/>　　<br/>　　    “上不上来啊，都困死了”茅小草靠着房门打着哈欠催促着。<br/>　　<br/>　　    房门开了，茅小草直接就奔着王惊蛰的卧室去了，然后“咣当”一声把门关紧了，就留下一句话：“我要是没睡醒谁也不许叫我，我上卫生间的时候你们都得在外面眯着，不许吵吵，不要催我，还有……”<br/>　　<br/>　　    菜刀文叹了口气，说道：“唉，这是住进来个奶奶啊”<br/>　　<br/>　　    王惊蛰点头说道：“头疼！”<br/>　　<br/>　　    菜刀文搂着王惊蛰，小声在他耳边说道：“你以后感悟人生的机会有很多，嗯，就从现在开始吧”<br/>　　<br/>　　    王惊蛰对他的话一脸懵逼，那种不对劲的感觉又更加浓重了起来。<br/>　　<br/>　　    一夜无话，隔天日上三竿。他们几个还都没有睡醒的时候，午桥顶着黑眼圈就找了过来，“咣，咣”的敲着房门把人都给吵醒了，菜刀文揉着眼珠子打开门看见午桥拎着个袋子还有好几瓶酒，就有点发懵了。<br/>　　<br/>　　    “不是，你这是要成仙了么，大中午的我们还没起床呢，就要喝酒啊？”<br/>　　<br/>　　    午桥进来说道：“事都结束了，过段时间我差不多也要该回去了，咱们也没好好聚聚，临走之前喝一点聊聊天吧”<br/>　　<br/>　　    午桥把熟食和花生皮拿出来，酒放到桌子上，菜刀文无语的说道：“那行吧，我们洗洗涮涮的，然后就过来吃饭，正好也省事了，关于谁做饭的问题，我们还没来得及商量呢”<br/>　　<br/>　　    午桥斜了着眼睛说道：“不是住进来个女眷么？君子远庖厨啊，做饭这种事还用商量么？”<br/>　　<br/>　　    “唰”一只拖鞋从客厅外面飞了过来，午桥看见披头散发站在门口的茅小草就笑道：“我后面还有一句话没说呢，他俩也不是君子啊，做饭的事不就得他们来么？”<br/>　　<br/>　　    菜刀文伸出拇指，冷笑道：“大爷，您真靠谱！”<br/>　　<br/>　　    茅小草白了他们一眼：“男人啊，要都像你们这么懂事，这世界上就没有男女之间的战争了……唉，小傻鸟你快从卫生间里出来，三十秒不开门，我直接就砸门进去了”<br/>　　<br/>　　    十一点多钟，午桥和王惊蛰还有菜刀文围坐在茶几上，啃着鸡爪子，吃着花生米，面前一人一瓶白酒。<br/>　　<br/>　　    “王生没有什么生命危险，就是身子比较虚，我估计在躺个十天半月差不多就能好了，你们丁武那个兄弟怎么样了？”<br/>　　<br/>　　    “问过齐泉了，人已经醒了脱离危险期了，不过他的伤挺重的，估计要卧床百天左右了”菜刀文一粒粒的夹着花生米，抿了口小酒说道：“唉，谁能想到啊，就因为一块破地的原因，搞出这么多事来呢”<br/>　　<br/>　　    午桥皱眉说道：“那可不是破地皮，而是阎王坟场，不过这地落到手里后根本就没有可用之处，不管地皮做什么到最后肯定得会有人在那里，阎王坟场的风水会把人给耗死的，所以这地等同于是一块废地，没有任何能造就的地方啊”<br/>　　<br/>　　    王惊蛰晃了晃手里的被子，抬头看着午桥说道：“活人住不了，那死人呢？”<br/>　　<br/>　　    午桥和菜刀文同时一愣，两人都不笨，脑袋也都灵光，他点了一句之后他们两瞬间就回过神来了，昨天晚上那跟丧尸一样的玩意就是个活死人，活人在阎王坟场里呆不了，那这种东西呢？<br/>　　<br/>　　    短暂的寂静之后，午桥挠着脑瓜皮说道：“原来是这么回事啊，不是给人准备的！”<br/>　　<br/>　　    “至于他们到底要干什么咱还不知道，算了也别管了，世界上不和平的地方太多了，我们又不是超人，哪来的那么多闲心去维护世界和平啊”菜刀文夹起一根鸡爪子说道：“吃，喝酒，开心最重要了”<br/>　　<br/>　　    一人闷了一口酒后，王惊蛰回头看着洗手间那边，说道：“女人都是这么磨叽的嘛？”<br/>　　<br/>　　    “你现在领悟的还算早，等你结婚之后你就明白了，女人说快了的话，你就找个地方歇着吧，没有二十分钟是出不来的，她们要说等会，你就可以打个盹了，如果说给你个准确的时间比如半小时一类的，那你干脆就该干嘛干嘛得了，没两个小时打底她们是不会完事的”<br/>　　<br/>　　    “我……”王惊蛰瞠目结舌的愣住了。<br/>　　<br/>　　    午桥笑道：“真精辟，但也绝对有道理，科学家能解决无数的科学难题，但唯独解决不了时间这个词在女人身上是怎么应用的”<br/>　　<br/>　　    片刻后，茅小草梳洗打扮完事，搬了张椅子就坐到王惊蛰旁边，瞅着茶几上的猪蹄，鸡爪子还有花生米等零碎的下酒菜，小嘴撇撇着说道：“一早上的就吃这么油腻的东西，对我一个女孩子来说，不太合适吧？”<br/>　　<br/>　　    “所以啊，做饭的问题你得考虑了，要是我们来做的话……”王惊蛰扭头正要跟茅小草商量伙食问题的时候，突然之间就愣住了。
