　　众人听了这话，更加震惊，傻傻地望着神石，都不知道该怎么表态。
<br/>　　谁也没想到，事情会发展到这个样子……
<br/>　　“张掌教，你说的这些，我们压根没听说过。如何证明？我们只知道，天选之子是叶少阳。”慈心师太还在煽风点火。
<br/>　　慕寒指了指神神石，道：“这就是证据！”
<br/>　　随即对大家说道：“大帝的旨意，就在这神石之中，只有天选之子的血液滴在上面，石头才会打开，旨意自会浮现！叶掌教，我也不与你分辨，你先滴一滴血在这神石上，若石化旨现，那你便是天选之子，否则，便轮到我……叶掌教，你敢不敢？”
<br/>　　事情水落石出。原来，他如此自信满满，甚至有点嚣张，是仗着这一重身份。
<br/>　　假如天选之子真的是他而不是叶少阳，那一切事情都不一样了。
<br/>　　一时间，所有人的目光都落在他的脸上。
<br/>　　叶少阳吐出一口气，无奈笑道：“你让我试就试啊，你算老几？”
<br/>　　慕寒不恼，淡然笑道：“你不敢么？”
<br/>　　“我，从来就没说自己是天选之子。”叶少阳淡淡吐出这句话。
<br/>　　这话他之前说了一遍，慕寒还当他是戏耍，此时听他还这么说，表情也一点点严肃起来，道：“你没说过，为何大家都认为你是？”
<br/>　　“也许我曾经说过，那时候我也以为自己是。”叶少阳挠了挠头，冲四宝笑了笑，有点无奈。
<br/>　　四宝早已是脸色铁青，他没叶少阳这么好脾气，冲慕寒骂道：“这么咄咄逼人，够了啊你，再说下去，可就不给你脸了啊！”
<br/>　　慕寒道：“你也当他是天选之子么，所以，你跟错了人？”
<br/>　　“去你妈的，老子才不管这些，只看拳头硬不硬，什么滴血试石，你跟少阳打一架，你要是天选之子，你应该比他厉害才对。少阳，跟他打。”
<br/>　　“实力为尊，那是邪魔外道，在我法术界可不兴这一套。当然，我这么说也不是怕他，只是不想伤了和气，龙虎山与茅山数代交好，只要他认我为尊，法术界团结一心，就算是轩辕山，也不能把我们怎样！”
<br/>　　随即也又冲叶少阳笑了笑，“叶掌教，你的实力，我是听说过的，今后你辅佐我，何愁大事不成？”
<br/>　　“我，辅佐你？”叶少阳看着他，一字一顿说道。
<br/>　　“说这么多干什么，先捶他一顿再说！”
<br/>　　四宝说着就要冲过去，被逍遥飞和几个五台山弟子死死拦住，叶少阳走过去，在他肩膀上拍了拍，本来还想安抚他一下，只听见慕寒在身后说道：“叶掌教，我尊你一声师兄，你的事情，法术界尽知，你未婚妻便是转世鬼童，你若以大局为重，此事还可商榷，你若一意孤行……”
<br/>　　叶少阳转身看他，“那便怎样？”
<br/>　　慕寒盯着他，沉默了一瞬，冷声说道：“那便是自甘堕落，与我法术界为敌！”
<br/>　　叶少阳深吸一口气，用力拉住肩膀上气的直喘气的瓜瓜，说道：“那有怎样？”
<br/>　　“纵然你法力通天，难道斗得过整个法术界？”
<br/>　　这个帽子扣的，简直不知不觉。
<br/>　　叶少阳意识到了，但懒得辩解，他早已是怒火中烧，刚要开口，看到了从慕寒身后站出来的龙阳真人，一脸凄苦地看着自己。
<br/>　　霎那间，他想到了道渊真人，想到了张无生，一下子心软起来，硬生生将一口气憋回去，转身拉住四宝的胳膊，道：“我们走吧。”
<br/>　　“走？”四宝瞪大眼睛，以为自己听错了。
<br/>　　没等他开口，慕寒的声音再度响起：“叶少阳，你若走了，便是自绝于法术界！”
<br/>　　叶少阳猛然转身，眼中两道杀气腾地射过去。
<br/>　　龙阳真人等一干龙虎山弟子，本能得后退了几步，慕寒也是心头一紧，很快冷静下来，转身望着自己一干手下，道：“你们怕什么？”
<br/>　　两个他贴身的弟子定了定神，知道该自己说话了，走上前去，望着早已经呆若木鸡的众人，说道：“你们都看到了吧，叶少阳非但不敢尝试滴血辨真，还包庇他的转世鬼童的未婚妻，不愿服从我掌教师兄的指挥，茅山百世香火相传，也不过如此！”
<br/>　　“我警告你，你骂我可以，别辱我山门！”叶少阳转身，冷冷地看过去。
<br/>　　这弟子本能地后退到慕寒身后，转头看看慕寒，好像有人撑腰似的，又狠狠瞪了叶少阳一眼。
<br/>　　边上另一个弟子说道：“你做都做了，还怕人你说，辱你山门又怎样？”
<br/>　　这两个一看就是慕寒贴身弟子，擅长见风使舵的，见慕寒话里话外攻击叶少阳，也壮着胆子来了一句。
<br/>　　“找死！”
<br/>　　叶少阳一头冲了出去，结果还没冲到这弟子面前，突然一道青色人影从头顶飞过，一下愣住，抬头看去。
<br/>　　所有人都抬起了头。
<br/>　　一头飘逸的长发和长衫，手持一根青铜色的打神鞭，朝石室中央的慕寒等人冲了过去。
<br/>　　“风大大！！”
<br/>　　瓜瓜激动地喊起来。
<br/>　　道风！
<br/>　　这个名字令所有人心惊胆战。
<br/>　　慕寒反应过来，几乎本能地举起九天元辰尺，试图抵挡他的攻击，但道风压根没在他面前停留，一溜烟飞过，手起鞭落，打神鞭砸在他身边那弟子的脑袋上，脑袋瓜子“咚”的一声就开了，魂魄被直接砸进了鬼域，奔着轮回井去了。
<br/>　　“饶命啊！”另外一个口出狂言的弟子看见这一幕，吓的快尿出来，想要往慕寒身后躲，慕寒也调转方向，九天元辰尺亮起神光，朝他胸前甩过来。
<br/>　　这一击，也几乎使出了全力。
<br/>　　结果……道风一只手就捏住了九天元辰尺，慕寒想要作法喷吐灵力，却发现尺子在道风手中似乎加固了封印，一点力气都吐不出，想拔回来也纹丝不动，心中大骇，就在这时，道风另一只手扬起打神鞭，重重地打在边上那个吓呆了的弟子脑袋上，也是脑袋被开了瓢，顷刻毙命，魂魄被抽进六道轮回。
<br/>　　“辱我山门者，杀无赦！”
