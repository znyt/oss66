夏夕绾只要想一想，就觉得不寒而栗。
<br/>
<br/>        那时的陆先生才多大，十几岁，还是一个孩子。
<br/>
<br/>        他不愿意说后来发生了什么，因为后来那段时间是他人生里最糟糕最不堪的岁月，他也不想她知道。
<br/>
<br/>        夏夕绾用力的点头，“好，我答应你，不管他们对我说什么，我都不会听的，我只听你跟我说的。”
<br/>
<br/>        说着夏夕绾就吻上了他的薄唇，小手还顺着他精硕的腰间往下滑，落在了他系着的浴巾上…
<br/>
<br/>        陆寒霆已经感觉到了她的热情，她从来没有这么主动过，刚才血淋淋的伤口迅速淡忘在了她的热情如火里，他感觉自己从冰冷的地窖里一下子被抛到了空中，恣意而畅快的欢愉。
<br/>
<br/>        “绾绾，我爱你…”他满眼纵欲的覆在她耳畔跟她说道。
<br/>
<br/>        ……
<br/>
<br/>        两个人回到了床上，夏夕绾像小猫一样窝在陆寒霆的怀里，她想起一个问题，“陆先生，今天你怎么来研究院了，还恰巧撞到了我跟陆子羡在一起，这未免也太巧合了吧？”
<br/>
<br/>        陆寒霆的大手抚摸着她柔顺的乌发，然后将自己的手机递给她，“我本来就在研究院外面，准备接你回家的，后来夏妍妍正好发了我信息，我就进去了。”
<br/>
<br/>        夏夕绾接过手机，很快就看到了夏妍妍发来的照片，照片里陆子羡轻搂着她的肩头，两个人像抱在一起的样子。
<br/>
<br/>        又是夏妍妍！
<br/>
<br/>        是夏妍妍喊陆先生来捉奸了！
<br/>
<br/>        夏夕绾一点都不觉得奇怪了，她仰起澄亮的水眸望着陆寒霆，然后翻了一个身大胆的骑坐在了他精硕的腰身上，“好啊陆先生，你现在还留着夏妍妍的联系方式，你不把她拉黑是等着她来勾引你么？”
<br/>
<br/>        陆寒霆看着她佯装生气的样子，精致粉嫩的双腮都嘟了起来，秀眉轻拧，真是奶凶奶凶的。
<br/>
<br/>        他勾起薄唇，“那我听陆太太的，现在就拉黑。”
<br/>
<br/>        夏夕绾将他手里的手机又抢了过来，“看你认错态度良好那就算了吧，先留着她，就当找点乐子，陆先生，她一旦勾引你你要立刻告诉我，我要看她的笑话，我就不信我治不了这个狐狸精了！”
<br/>
<br/>        陆寒霆挑了一下狭长的眼梢，溢出了几分成熟男人的风情，他精硕的腰身往上挺了一下，“好好说话，先从我身上下去！”
<br/>
<br/>        夏夕绾现在还坐在他身上，他这么一动让她俏脸一红，她迅速连滚带爬的下去了，整个人埋在被子里，只留一双翦瞳在外面乌溜溜的转，“不管怎么样陆先生这一次你错了，夏妍妍让你来捉奸你就来捉奸，你是信我还是信她，我可是你的陆太太，你这样做让我很没有面子。”
<br/>
<br/>        女孩嗓音软软糯糯的控诉，陆寒霆心尖一软，他俯下身，亲吻她露在外面的眼睛，“恩，我错了。”
<br/>
<br/>        “陆先生，那我们事先说好了，我跟陆子羡真的什么都没有发生，这个夏妍妍现在一定以为我出轨陆子羡了，这么好的机会她不会放过的，肯定会大做文章，无论发生什么事情，你都要相信我，做我最强有力的后盾，好好的打脸给夏妍妍看！”
