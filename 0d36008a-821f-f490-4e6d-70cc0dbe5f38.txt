　　第362章
<br/>　　有时候找不到想要的东西，不是因为看得不够远，反而是因为看得不够近。
<br/>　　程大雷和苏樱朝夕相处，食同餐，眠同榻，只当她是个声娇体媚的屋里人，却是疏忽了一件事：苏樱可是少数几个拥有隐藏属性的人。
<br/>　　而她的隐藏属性是：明察秋毫。
<br/>　　踏破铁鞋无觅处，蓦然回首那人却在灯火阑珊处。
<br/>　　程大雷白了苏樱一眼，双臂摊开，道：“仔细看看，你能不能看出什么？”
<br/>　　苏樱不知道程大雷在搞什么，当然，程大雷经常做一些稀奇古怪的事情，苏樱已经见怪不怪。见程大雷如此说，她微微颦眉，集中注意力在程大雷身上，然后便有太多东西被阅读出来。
<br/>　　“尸体的味道，你走了很远的路，可你鞋底沾着泥，身上衣服却是干净的，因为你回来后就换了衣服。像你这样的人也开始注意仪表……”苏樱撇撇嘴：“樊梨花也来了？”
<br/>　　“这个……哈哈。”
<br/>　　程大雷打了个哈哈，心中是有喜有忧，喜的是苏樱的隐藏属性果然有用，忧的是自己以后若想做点什么坏事，可别幻想能瞒住她。
<br/>　　程大雷摊开面前的资料，道：“发生什么事，你应该已经知道了。这些是凉州城送来了，我刚才看了半天，也没有什么收获。你帮忙看看，能不能找到什么线索？”
<br/>　　“我又没有做过这些事，怕帮不了你。”
<br/>　　“嗳，要相信自己麽。”
<br/>　　程大雷将苏樱让到书案后的座位上，并为她挑亮了油灯。苏樱却不过，只好坐下翻开那些资料。
<br/>　　大部分都是仵作验尸的资料，还包括一些现场勘验的线索，摊开这些，似乎就能嗅到一阵阵的血腥味。
<br/>　　苏樱逐字读着，灯光打在她侧脸，勾勒出完美的侧颜。程大雷帮其将发丝撩在耳后，苏樱抬起头，二人目光对视，互相都笑了笑。
<br/>　　苏樱已阅读出太多内容，眉头微皱，眼中隐隐有痛苦之色。
<br/>　　哪怕是朝夕相处，事实上，程大雷也并不懂苏樱。
<br/>　　苏樱在很小就有一种能力，大概类似于天赋之类的东西：走入一个陌生的环境，她能在极短时间内阅读房间出的信息，甚至推测出房间里曾经发生过的事。
<br/>　　而面对陌生的人，她能将自己代入对方的思想，勘破对方心中的想法。
<br/>　　这并不是一个令人愉悦的能力，明察秋毫，便是看破人心。而人心……大多丑陋。
<br/>　　虚伪、狡诈、懦弱、谎言。
<br/>　　很小的年纪，就感受到这么多负能量，怪不得苏樱如今又娇又小。
<br/>　　到后来，苏樱就很少动用这方面的能力。像今日，当她翻阅这些资料时，不仅能体会凶手的心情，更多是体会被残杀者临死前的恐惧、绝望。
<br/>　　这自然是很痛苦的事情。于是她眉头不自觉皱起，睫毛下意识抖动。
<br/>　　“累了么？”程大雷道：“要不先歇歇。”
<br/>　　苏樱笑着摇摇头：“杀人的可能只有一个人？”
<br/>　　程大雷一怔，道：“怎么可能，整个凉州已经有上百人被杀，如果只有一个人，怎么可能做到。”
<br/>　　“我不知道你说的，但以我的感受来说，杀人者的情绪只有一种。不可能许多人只有一种情绪……”苏樱道：“从这些也看不出来什么，我们去出事的地方看看吧。”
<br/>　　“现在？”程大雷道：“夜已经很晚了？”
<br/>　　“走去看看吧，你不是很着急么。早点解决这些事，你也能好好歇歇。”
<br/>　　程大雷顿了顿，抓起身边的匹夫剑。
<br/>　　“走。”
<br/>　　二人出了将军府，向案发现场赶去，夜静无声，荒原平白给人一种荒凉意味。
<br/>　　樊梨花现在也没有睡着，在院子里纳凉时，无意中窥见程大雷和苏樱鬼鬼祟祟离开将军府。
<br/>　　“这么晚了，这两个人要去干嘛。”
<br/>　　看到这一幕，心里不仅有些好奇，袖了一柄短刀，悄悄跟在二人后面。
<br/>　　一路赶到出事的地点，这里刚死过人，却也没人会靠近。当讲房门推开时，一股凉风扑面，程大雷结结实实打了个寒颤。
<br/>　　程大雷无法体味苏樱的心情，在短暂的瞬间，许多信息冲入苏樱脑海。杀人者的冷酷，被杀者的惨叫……她心神乱颤，脸色一瞬间变得苍白。
<br/>　　程大雷拥住苏樱的肩膀，苏樱向他点头示意，迈步踏入了房间。
<br/>　　隔着老远，樊梨花看见这一幕，微微撇嘴，心底发出一声冷哼。
<br/>　　苏樱静下心来，尝试去体味杀人者的心情。
<br/>　　程大雷立在她身边，手中举着火把，见短暂的瞬间，她像换了个人似的，脸上肌肉绷紧，冷峻如岩石，眼中闪烁着兴奋的微笑。
<br/>　　程大雷觉的有些恐惧了，只因为苏樱脸上的模样实在有些诡异。可他也不敢打扰苏樱，生怕自己有个一差二错吓着对方。
<br/>　　良久，苏樱脸上的表情才恢复正常，整个人变得更加虚弱，程大雷扶住了她，发现她后背已被冷汗湿透。
<br/>　　“人怕是真的只有一个。”苏樱平复着呼吸：“是个男人，身材很高大，杀人的时候手段很干净。可能他只是为了快乐而杀人，就像人猎杀野兽一样，杀人能令他兴奋。”
<br/>　　“能知道人往哪儿去了么？”
<br/>　　“大概是往……”苏樱手指向一个方向。
<br/>　　“什么人！”
<br/>　　正在此时，程大雷忽听得窗外响起动静，他猛地喝了一声，身如鹊起，撞破窗户冲了出去。
<br/>　　如此行动之快，苏樱根本没有反应过来。等她走出屋子，就看到程大雷的剑搁在一人的肩膀，二人目光相对，互相都有些无语。
<br/>　　程大雷悻悻的收了剑，表情有些无语：“怎么是你啊。”
<br/>　　“怎么不能是我，许你们来就不许我来。”樊梨花嘴硬道：“有线索没？”
<br/>　　的确是有了些线索，苏樱的隐藏属性还是靠得住的。本来带着苏樱去寻找更加保险些，但苏樱的身体状况，怕已有些撑不住。
<br/>　　“樊寨主，麻烦你将苏樱送回去，我去找那魔头。”
<br/>　　“还是我和你一起去吧。”苏樱。
<br/>　　“不了，我一个人去就够了。”程大雷收剑入鞘，向西而行。
<br/>　　
<br/>(本章完)
