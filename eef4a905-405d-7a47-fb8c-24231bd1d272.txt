　　洛菲克摸索着自己下巴上蓬松的胡子，说道：“GHOST是这个世界上除了SHADOW以外，最优秀的黑客。他曾经黑入过包括洛克希德公司在内，多家武器公司的数据库拿走了价值不菲的武器设计源代码，这些东西即便是放在现在，不，就是在未来都是有着极高的价值。”
<br/>　　“我不明白，先生。”David那双棕色的眸子看着洛菲克，说道：“就算GHOST手中掌握着很多武器设计源代码，但那至少是十几、二十年前的东西，它们现在还具备曾经的价值吗？”
<br/>　　“二十年时间，对于一款武器而言根本不值一提。风靡全世界的AK-47，从它被设计出来至今已经有近七十年的时间，而在AK-47之后，又有很多比它更先进的自动步枪被设计出来，可即便如此，AK-47依旧是这个世界上仿造和使用最多、最广泛的武器。”
<br/>　　洛菲克伸出汗毛明显的手指，对着David轻轻摇晃着，说道：“更何况，GHOST当年拿走的武器设计源代码中有相当一部分是还没有被投入到生产计划，但是已经有了完善设计的源代码。明白我的意思吗？如果将这些被GHOST拿走的武器设计源代码拿回来，并且重新投入到生产，即便是现在，它们也依旧会是这个世界尖端的武器！”
<br/>　　“如果GHOST现在真是中国军方的人，那么他手里掌握的武器设计源代码不会不拿出来，最近十几年，中方的军备发展虽然十分迅速，但是中间似乎并没有特别之处……”David看向洛菲克，不解地说道。
<br/>　　“中国人很聪明，他们是这个世界最聪明的民族。”洛菲克说道：“即便是他们拿到了GHOST手里的武器设计源代码，他们也不会直接拿来使用，他们不蠢，不会把自己推到风头浪尖上，他们最多只会参考其中的设计概念，然后再整合自己的技术水平完成一套全新的设计。”
<br/>　　David沉默了一下，对洛菲克问道：“我们是要对GHOST动手吗？”
<br/>　　洛菲克从西装口袋中拿出一块有些融化的口香糖，咀嚼着口香糖中的甜味，对着David摇了摇手指，说道：“小镇有小镇的规矩，谁也不能在小镇里动武，就算要动手也要等到拍卖会结束，离开小镇以后再动手。先去情报酒吧，拉塞尔那混蛋也应该快到了，跟他商量一下再做决定。当年被GHOST光顾了数据库的，除了我们洛克希德以外，还有他们的通用原子。”
<br/>　　情报酒吧，是来这里进行源代码交易的人给它起的名字，这是一家当地很具特色的酒吧，来小镇交易的人都会选择到这个酒吧中，花上一些钱来打听自己想要东西的线索，时间一长，这个酒吧就成了小镇上最大的情报集散中心。
<br/>　　酒吧老板是个叫“安妮”的漂亮的白人女子，能说一口流利的拉丁语、英语、葡萄牙语和俄语，据说她是美国人，但是她自己本人却从来没有正面承认过。
<br/>　　最开始，酒吧老板安妮贩卖情报，到后来她从情报贩子成为情报掮客，她不再自己贩卖情报，而是给两个买卖情报的人之间牵线，她则是从双方的交易中抽取提成。
<br/>　　酒吧一共有两层，虽然现在天还完全黑，但是酒吧中却是已经有了不少人，其中有人已经开始进行交易。每年“源代码拍卖会”前，情报酒吧里的生意都是最好的时候，就这么几天的时间，安妮靠收取情报中介费赚到的钱就抵得上她开酒吧一年的收入。
<br/>　　两个俄罗斯人刚达成一笔令双方都满意的交易，两只毛茸茸的手臂握在一起，双方各拿出几张纸币放在一起交给吧台后面的安妮，这是她牵线的中介费。
<br/>　　安妮微笑着把钱收下，目送着两个俄罗斯人离开的时候，正好看到洛菲克和David走进酒吧，那张看不出岁月痕迹的脸上，立刻露出美丽的笑容，对着洛菲克招呼道：“洛菲克先生，您今天来的可是够早的啊，不过您的老朋友拉塞尔似乎还没到。”
<br/>　　安妮环视了一群酒吧里的人，漂亮的脸上露出遗憾之色，对洛菲克问道：“今天想要喝点什么？还是马丁尼吗？”
<br/>　　“是的。”洛菲克从吧台的纸巾盒中拿了一张纸巾，包好嘴里的口香糖渣，扔进旁边的垃圾桶中，把手身上安妮细嫩光滑的小手，说道：“我非常怀念安妮小姐亲自调制的马丁尼，这是我喝过的，最好喝的马丁尼。”
<br/>　　“洛菲克先生您还是这么会说话。”
<br/>　　安妮不着痕迹从吧台上把手抽走，没有让洛菲克占到任何便宜，转身走吧台前离开，去为洛菲克准备他需要的马丁尼鸡尾酒。
<br/>　　洛菲克也没有因为没有占到安妮的便宜而不高兴，如果安妮这么容易就能被人占到便宜，那么她就不是安妮了，这个女人也不可能在这个鱼龙混杂的地方安然无恙的生活这么长时间，并且心安理得收取情报中介费用。
<br/>　　洛菲克带着David走到酒吧角落里的一个位置坐下，David则是站在洛菲克身后的位置。情报酒吧禁烟，任何人都不能在这里抽烟，所以这个酒吧也就不像其他酒吧那样乌烟瘴气，空气中飘满了尼古丁的味道。
<br/>　　安妮把一杯调制好的马丁尼鸡尾酒端到洛菲克面前，正准备离开的时候，酒吧门再次被人推开，进来的是通用原子公司的代表拉塞尔。
<br/>　　拉塞尔年纪跟洛菲克差不多，但是要比洛菲克消瘦，一身得体的白色纯手工定制西装，加上刮得干干净净的面颊，让得拉塞尔看上去要比洛菲克年轻几岁。
<br/>　　“洛菲克先生，您要等的老朋友到了。”
<br/>　　安妮转身对着走进酒吧寻找座位的拉塞尔招呼过去，问道：“拉塞尔先生，您是跟洛菲克先生一样，要来一杯马丁尼还是别的？”
<br/>　　“比起马丁尼，我更喜欢杜松子酒。”拉塞尔对着安妮摇晃着手指，说道：“请给我来一杯杜松子酒，再加上一小块冰，这该死的天气让我很不舒服。”
