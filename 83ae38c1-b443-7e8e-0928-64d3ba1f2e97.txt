        手下顿时传来软软的触感，让郑忠威心里爬上一种不祥的预感。

<br/>        而下一秒，季思妍一阵尖叫声，恍若震撼了整栋大厦。
<br/>
<br/>        “啊——你这死变态！想对我做什么？”季思妍恼羞成怒地大吼。
<br/>
<br/>        郑忠威一脸尴尬，连忙放开了季思妍，并后退了好几步。
<br/>
<br/>        季思妍趁势双手护胸，双眸含泪，贝齿咬住了下粉唇，一副贞洁烈女的模样。
<br/>
<br/>        傅夜沉不禁皱了下眉头，淡漠地开口：“再不滚，我就叫保安上来了！”
<br/>
<br/>        季思妍抿了抿唇，望向傅夜沉时，见他仍旧只是坐在办公桌前淡定地看他自己手里的文件，知道自己这“一哭二闹三上吊”的法子在这男人面前不好使后，立马愤恨地甩手，一边擦拭着眼泪，一边抽泣着像个受了欺负的小女生一样，“呜呜”地跑了出去。
<br/>
<br/>        郑忠威一脸无奈地看向自家大少爷，总觉得大少爷是不是对那位季大小姐太绝情了点？
<br/>
<br/>        男人，难道不应该是要对女人怜香惜玉的吗？
<br/>
<br/>        他家大少爷到底还是不是男人？
<br/>
<br/>        傅夜沉完全不顾郑忠威用那种异样的眼神盯着自己，反倒是云淡风轻地提醒了郑忠威一下：“我的文件快批完了！”
<br/>
<br/>        “是是是！我马上去准备接下来的行程资料！”郑忠威回过神来后，毕恭毕敬地颔首，立即转身去了自己的办公区。
<br/>
<br/>        自从那个季思妍在他家大少爷面前晃悠了之后，他都感觉自己快要成为了“猥琐大叔”了。
<br/>
<br/>        不过话说回来，郑忠威真的发现，他家大少爷对女色这种事情，就像天生自带免疫系统一样。
<br/>
<br/>        各路美女投怀送抱，他傅夜沉居然拒之于千里之外。
<br/>
<br/>        傅夜沉的文件批改完后，和郑忠威一起，离开了办公室，去了停车场，接下来他们要去洽谈一个商务合同。
<br/>
<br/>        那季家大小姐刚走，这季家少小姐又找上了门来。
<br/>
<br/>        郑忠威本不认识这位季家少小姐，但是这小妮子敢拦他家大少爷的车，就证明她来头不小。
<br/>
<br/>        “救命恩人，好久不见，你又帅了不少！”季雅姗眨了一下左眼，冲着傅夜沉放了个电，然后嬉皮笑脸地一屁股蹦到傅夜沉的座驾车头上坐着。
<br/>
<br/>        郑忠威刚打开驾驶座的车门，一脸窘迫地看向傅夜沉。
<br/>
<br/>        “开车！”傅夜沉冷冷地命令道，不禁皱起了剑眉，径直拉开驾驶座后的车门坐了进去。
<br/>
<br/>        季雅姗见状连忙从车头上跳下来，小跑到另一边的副驾驶座后，以迅雷不及掩耳之势，拉开车门钻了进去。
<br/>
<br/>        “救命恩人，大哥哥，我有东西要送给你心爱的那个大姐姐的！”季雅姗一边说着，一边将背上的书包放了下来。
<br/>
<br/>        傅夜沉也下意识地瞥了她一眼。
<br/>
<br/>        只见这小妮子从书包里拿出了一个手工包。
<br/>
<br/>        “喏！这是我特意给大姐姐做的！上面可是真的珍珠！价值不比那些品牌包包低！”季雅姗微笑着将手里的手提包，朝傅夜沉递了过去。
<br/>
<br/>        傅夜沉并未接，只是双手抱臂，目光注视着前方淡定地问：“你有事求我？”
<br/>
<br/>        “大哥哥真聪明，果然什么事都瞒不过大哥哥。”季雅姗见傅夜沉无动于衷，强行将自己手里的珍珠手提包塞到了傅夜沉的臂弯中，接着说道，“大哥哥，这真的是我纯手工做的！我做了两个噢！一个自己留着用，一个就想着要送给大哥哥的心上人。我说的是真的！哪怕没有事来求大哥哥，我也要把这个包包送给大姐姐！”
<br/>
<br/>        傅夜沉这才仔细地打量着臂弯里的这个珍珠手提包，发现它做工确实很精致，而且上面的珍珠莹润圆滑，确实货真价实。
<br/>
<br/>        漫雪一定会喜欢！

<br/>        傅夜沉心想着，将手提包放到了身旁的座椅上，然后掏出自己的钱包来，从里面拿了一张卡，递给季雅姗：“里面有三万，应该足够买下这个包了。”
