　　“陛下，看来这次倭人的所图不小啊，您看这东西值多少武器装备！”御书房中，王玄策将手里的一万贯存单放到了李承乾的桌上，语带调侃的说道。
<br/>　　一万贯虽然不是小数目，但这种钱有些烫手，所以王玄策根本就没打算留着，他可不想因为这点钱把自己的后半生坑进去。
<br/>　　李承乾低头看了看那存单，拿起来抖了抖，嗤笑一声：“这帮小鬼子看来图谋不小啊！”
<br/>　　“那陛下的意思是？”
<br/>　　“该卖的东西都卖了吧，那些淘汰下来的钢弩，还有床弩什么的都可以卖，粮食也可以卖，衣服也可以卖，价格就按正常价格的五倍算好了。”李承乾弹了弹手里的存单，随手又丢回到桌上：“这东西你也收回去吧，毕竟是那些倭人给你的。”
<br/>　　“陛下，这……”王玄策没想到李承乾竟然会把那一万贯钱再次还给自己，一时竟有些不知应不应该拿回来。
<br/>　　“一万贯钱而已，算不上多，朕还看不上。”李承乾把存单又往前推了推：“这些年你也算是劳苦功高，这个就算是一份奖品吧。”
<br/>　　“是！”王玄策见李承乾的确没有什么其他想法，只是单纯的想要收买人心，便也不再犹豫，将那存单拿起来塞进了怀里。
<br/>　　他很清楚，如果这个时候这钱他不拿回来，估计李承乾必然会心生疑窦，怀疑他别有所图，所以不如直接拿了省心。
<br/>　　李承乾见王玄策把钱拿回去了，脸上露出一丝笑容，想了想说道：“朕估计过一段时间倭人还会再来一波人，你不用客气，给你的你都拿着就好，武器装备双份卖给另一波人，明白么？”
<br/>　　“明白。”王玄策点了点头。
<br/>　　李承乾看了看王玄策，生怕他误会什么，又特地叮嘱道：“倭人的战争潜力现在还是很大的，所以先万不要留情，东西的价格可以往死里贵，他们出得起这份钱。”
<br/>　　“是，朕知道应该怎么做了。”以王玄策的智力，很快就理解了李承乾话里的意思，露出了然的笑容。
<br/>　　他王玄策只是一个接待官，可是秦吉胜竟然能拿出一万贯来收买他，这足以证明倭人现在并不缺钱，这种有钱的铠子如果不砸他们实在有些不对起自己的‘良心’。
<br/>　　……
<br/>　　鸿胪寺的驿馆，秦吉胜赶走了所有人，独自一个人在发呆。
<br/>　　大唐是一个强大的国家，想要在其控制之下投机不是一件容易的事情，秦吉胜很清楚自己的地位到底是怎么来的，也隐约可以猜到李承乾的目的是什么。
<br/>　　正是因为这样，他才担心，生怕自己哪一点做的让那位皇帝陛下不满意，就会人头落地。
<br/>　　而他自己死了没有关系，到时候只怕整个倭国真的会像大唐期望的那样，彻底乱起来。
<br/>　　难，真的很难，如果再给他一次机会，他决不会再向以前那样自大，不会想着独立，虽然当初的倭国有着这样那样的弊端，但总好过现在这样人脑子打出狗脑子。
<br/>　　“大人，大唐特使王玄策来了。”就在秦吉胜为倭国的未来担心不已的时候，外面有人来通报。
<br/>　　“快，快请进来，哦不，我新自去迎接！”秦吉胜听到王玄策的名字，人几乎从椅子上跳起来，用最快的速度换好了衣服，冲出了门外。
<br/>　　王玄策此时已经在划分给倭国的小院外面等了一会儿了，不过他却并没有着急，倭国对于大唐来说实在是太过弱小，弱小的完全没有必要与他一般见识的程度。
<br/>　　所以在见到秦吉胜的时候，老王的脸上依旧保持着笑意，一副你好我好大家好的样子。
<br/>　　“王大人，下官迎接来迟恕罪，恕罪啊！”秦吉胜见到王玄策脸上的笑意，心中微微一松，不服自己的官职实际上比王玄策高的事实，谦卑的笑着道歉。
<br/>　　“秦大人言过了！”王玄策笑着摆摆手：“本官刚刚从宫里出来，你的要求陛下答应了。”
<br/>　　“答应了？”秦吉胜的脸上露出一副惊喜的表情。
<br/>　　“秦大人不会想在这里谈吧？”王玄策四下看了看，然后似笑非笑的说道。
<br/>　　“啪”秦吉胜在自己的额头上拍了一下，比了个引路的手势：“王大人，失礼，失礼了，请，快请进！”
<br/>　　“嗯，请！”王玄策应了一声，大步而行，跟在引路的秦吉胜身后，进了小院。
<br/>　　不多时，到了秦吉胜的房间，这家伙将所有人都打发出去之后，亲自为王玄策泡上热茶：“王大人，请！”
<br/>　　“秦大人不必如此客气，王某不过是职责所在而已。”王玄策虽然嘴上说的客气，不过动作上却不有一点客气的意思，伸手端起秦吉胜亲自泡上的茶水，略闻了闻赞了声：“好茶！”
<br/>　　“王大人谬赞了。”秦吉胜的注意力明显不在茶叶好坏上面，谦虚了一句之后马上把话题拉到了正题上面：“王大人，不知皇帝陛下是如何答复的？”
<br/>　　王玄策先是沉默了一会儿，就在秦吉胜急的眼珠子冒火的时候，才缓缓说道：“陛下说了，秦大人是栋梁之材，是大唐的最忠实的朋友，所在秦大人的要求不管是什么，都可以满足，军械，粮食，衣物，任何东西都可以。”
<br/>　　大唐最忠实的朋友？这话怎么听着那么怪呢，都说狗是人类最忠实的朋友好吧？难道老子就是大唐养的一条狗？
<br/>　　不过算了，狗就狗吧，现在这个样子好像和狗也没有什么区别，秦吉胜无奈的想着，赔着笑脸说道：“那，不知道这军械都有什么？陛下可有什么限制？”
<br/>　　王玄策摇摇头说道：“限制道没有，不过考虑到后勤保障方面，还有兵员素质的问题，一些顶级装备就算是卖给你们，你们也用不了。”
<br/>　　“那，那……”秦吉胜一脸的纠结，略有些失望。
<br/>　　但王玄策也没有让他多等，很快便接着说道：“不过钢弩可以对你们放开控制，还有床弩也可以对你们放开控制，怎么样，现在满意了吧？”
