　　将什造的事情暂时放在脑后，刘哔紧了紧手中的拳套，从出口走了出去。
<br/>　　他能感应到，前方还有两股强大的气息，似乎正在战斗，波动十分剧烈。
<br/>　　走了一段距离，刘哔来到一处环形的大厅，大厅内很是空旷，里面翻腾着比别处更加浓郁的白色雾气，两道模糊的人影在白雾中不停战斗着。。
<br/>　　“CRC瓦斯，这些青铜也玩完了吧。”刚刚想要踏进去，刘哔左手边传来一道女人的声音，让他顿时止住了脚步。
<br/>　　这里居然还有旁人，在浓郁的白雾中，连他的视线也受到了很严重的阻挡。
<br/>　　“怎么了，亚门上等。”先前的女声再次响起，这次有所防备之下，让刘哔找到声音的来源。
<br/>　　无声无息的潜伏了过去，刘哔看见一个有些熟悉的脸庞，是亚门钢太郎，正拿着个鸡毛掸子一般的库克因，在白雾中防备着。
<br/>　　他身边的，是一个穿着正装的年轻女子，似乎受了伤，满身是血，瘫坐在冰冷的地板上。
<br/>　　他也大致猜到了那女子的身份，真户晓，吴绪的女儿，虽然他插手改变了主线剧情，但那家伙，还是死了。
<br/>　　这也是合情合理的事情，谁让真户吴绪得罪了雏实，即使失去了店长他们，原古董人员依旧有四方和金木的庇护，这两人，根本不是真户吴绪可以去招惹的。
<br/>　　“那是什么。”亚门似乎觉得有些不太对劲，戒备的向前走了几步，紧接着，瞳目猛然一缩。
<br/>　　浓郁的白雾中，伸出了一道类似于尾巴的赫子，一闪而过，一阵令人头皮发麻的咀嚼声从中传出。
<br/>　　看到亚门的失神状态，刘哔眯了眯眼，身影如同利剑一般窜了出去，瞬间就将手臂穿透了亚门的胸膛。
<br/>　　一颗还在怦然跳动的心脏，被他抓在手中捏爆，攻势太快，亚门甚至都没有从短暂的失神状态中醒来。
<br/>　　身后传来女人的惊呼声，刘哔将手臂从亚门胸膛中抽出，转身拧断了真户晓的脖子。
<br/>　　狮子搏兔，亦用全力，虽然这两人和他现在的境界相比，根本没有任何对抗的机会，可他还是选择了偷袭。
<br/>　　能出其不意解决掉的对手，他才不会冒着风险去正面抗衡，那是沙比才会做的事情。
<br/>　　“叮~恭喜宿主击杀上等搜查官亚门钢太郎，获得积分1000。”
<br/>　　“叮~恭喜宿主击杀二等搜查官真户晓，获得积分500。”
<br/>　　系统提示的声音响起，刘哔甩掉了拳套上的血迹和内脏碎屑，对两名搜查官发动了吞噬技能。
<br/>　　一千五百积分，还不太够，在他的预计当中，如果彻底将什造身体上的伤势恢复，至少需要四千多积分才行。
<br/>　　修复可比破坏困难多了，相对的，治疗伤势所需要花费的积分自然也不会少。
<br/>　　“叮~恭喜宿主吞噬上等搜查官亚门钢太郎，能量增加。”
<br/>　　“叮~恭喜宿主吞噬二等搜查官真户晓，能量增加。”
<br/>　　果然这种级别的能量体，已经吞噬不出来什么东西了吗。
<br/>　　轻叹了一口气，刘哔转过身来，缓缓踏进了白雾之中。
<br/>　　前方还有一顿豪华大餐在等着他，那两人的能量，比他现在杀的这俩货可强上太多了。
<br/>　　……
<br/>　　“啪嗒，啪嗒……”
<br/>　　空旷的大厅中，脚步声极为明显，其中混杂着令人浑身起鸡皮疙瘩的尖锐声音。
<br/>　　就好像有人在吃玻璃一般，咯嘣咯嘣的，让人头皮发麻。
<br/>　　穿过浓郁的雾气，刘哔看到一个满头白发的瘦小身影，那身后如同翅膀一般的夸张赫子，表明了他赫者的身份。
<br/>　　眼神微微一动，他认出了这家伙的身份，金木，他还是走上了这条路。
<br/>　　但是，只有这样，才能尽快成长啊，仅仅是这种程度，又怎么能让他升起吞噬的欲望。
<br/>　　这家伙，和别人不同，他可是未来的喰种之王，这个位面中最强大，击杀之后战利品收获最多的终极能量体。
<br/>　　怎么可能，就这么轻易杀掉呢，那样未免有点太过于可惜。
<br/>　　毕竟和什造不同，这家伙的崛起，靠的不纯粹是潜力，过早的吞噬，也不会吞噬出什么令他满意的东西。
<br/>　　除非饥饿到了极致，否则没多少人会选择将猪仔杀掉食用，因为肉太少了。
<br/>　　恰好，刘哔现在不缺食物，他有耐心，等着猪仔慢慢长大，到时候他会收获更多。
<br/>　　“金木，好久不见。”看着正趴在地上大快朵颐的金木，刘哔脸上带着微笑，向前走了几步。
<br/>　　一根红色的赫子忽然飞起，向他太阳穴的位置袭来，刘哔笑意未减，伸手抓住了红色赫子的尾端，任凭它在自己手中徒劳挣扎。
<br/>　　仅仅是这样的动作就引来了袭击吗，看来喰种进食的时候，对周围也是十分戒备的呢。
<br/>　　但，这样的力道实在是太弱了，根本不可能对他造成什么威胁。
<br/>　　“金木，停下，我有事情要和你说。”刘哔继续向前迈了几步，在金木研身后站定，带上了一丝命令般的语气。
<br/>　　闻言，金木研瘦弱的身体停顿了一下，缓缓从地上站了起来。
<br/>　　“为什么要杀掉店长？”金木研的转过身来，露出了前端尖锐的黑色面罩，这并不是他原本那张。
<br/>　　看到金木眼中的仇恨，刘哔皱了皱眉，但对于金木能知道这则消息，他也没怎么感到意外。
<br/>　　枭被其他喰种杀死的消息，肯定隐瞒不住，毕竟当时现场的人员实在太多，甚至还有一些军人和未撤离的普通群众。
<br/>　　而他当时用的面罩，是之前还还未损坏的那张小丑面罩，身为古董店的一员，知道自己的信息也太过正常。
<br/>　　不过这并没有什么影响，知道了也就知道了，反正他从一开始也没打算隐瞒这则消息。
<br/>　　“你觉得他当时能活下来？”看到金木又有暴动的倾向，刘哔冷哼一声，脸上的笑容也逐渐开始消失。
<br/>　　置身于白色雾气中，金木研瘦弱的身影有些颤抖，两只沾满鲜血的拳头，紧紧握在了一起，发出令人牙酸的咯吱声。
<br/>　　似乎是，骨头被捏断了，但他似是没有感觉到疼痛一般，没有发出一点声响。
