　　张小凡修炼大梵般若苏重是知道的。而且他还知道，大梵般若是从天书第四卷上衍生而来。只要得到大梵般若，就相当于得到了部分天书。
<br/>　　要不要从张小凡这里获得大凡般若？
<br/>　　苏重想了想就摇头驱散这个念头：“以张小凡倔驴性格，想从他嘴里获得秘籍简直难如登天！”
<br/>　　“回去后你按我说的修炼就好，记住了，半年之内除了聚元功，什么功法你都不能练。不然你就别想练成。”苏重叮嘱。
<br/>　　大梵般若和太极玄清道有冲突，但也不是不可调和。张小凡最后两者兼修同时修成就是证据。
<br/>　　只要找到合适修炼节奏，两种功法同修不是问题。可惜张小凡是不可能拿这种问题去请教其他人的。当初普智把大梵般若传给张小凡，首先就要他保密。
<br/>　　其实真要泄漏了也没什么，再怎么说张小凡也是青云弟子。即便两者兼修，到最后还是青云占便宜。苏重觉得，如果真要让田不易知道真相，他可能还会帮着隐瞒。
<br/>　　把张小凡打发走，苏重继续琢磨自己的庚金剑气。和太极玄清道相比，庚金剑气就是一门秘法。
<br/>　　两者的区别在于，只要修炼成功太极玄清道，法力就会生生不息，即便消耗干净，不用修炼它也能慢慢恢复。而如果主动修炼，则恢复更快。
<br/>　　庚金剑气不同，每一枚剑符可以激发一次庚金剑气。激发完成后剑符就会变成不可用状态。只有等苏重再次补充凶戾气才能使用。
<br/>　　说白一点儿，太极玄清道修炼就想挖井，井水少了他会慢慢补充。庚金剑气修炼就是造电池，用没了电它可不会自己回复。这是庚金剑气的缺陷。好在庚金剑气不是一次性电池能反复充电。
<br/>　　“得想法子让剑气符自动回复。”苏重琢磨。
<br/>　　以后和人家斗法，自己一波剑气扔出去就变成个普通人，没有十天半个月修养完全无法回复，万一扔人堵住就只能任人宰割。
<br/>　　“二蛋师弟！二蛋师弟！”一个清脆声音打断苏重思绪。整个大竹峰，会这么称呼苏重的，只有田灵儿。
<br/>　　苏重走出竹屋，就看到田灵儿俏生生站在院子里，笑声问到：“师姐有什么吩咐？”
<br/>　　“嘿嘿，我娘让你去祖师祠堂。其他师兄都去了，要考校你们修为。”田灵儿一脸瞧热闹表情。
<br/>　　“好，我这就去。”苏重关上竹屋房门就走。这半年来，苏重已经不是第一次参加这种考校。
<br/>　　以往他无法修炼太极玄清道，都是在一旁打酱油瞧热闹，看几个师兄弟被师娘苏茹收拾得苦不堪言。
<br/>　　苏重没走两步，田灵儿却喊住苏重，然后挥手放出一条火红飘带，法力激发无风自起飞到半空。
<br/>　　田灵儿一跃而上，然后满脸得意的看着苏重：“师弟，要不要我带你一程！”
<br/>　　“多谢师姐好意，我跑着去就好。”苏重笑着拒绝。田灵儿最近刚得了苏茹的琥珀朱绫法宝，能够御空飞行。有事没事的就满天乱飞显摆能力。
<br/>　　“哼！你可别迟到了！”田灵儿不高兴，狠狠瞪了眼苏重，法力激发嗖的一下就没了影子。
<br/>　　苏重摇头毫不在意，身体一晃瞬间消失在原地。下一瞬间就出现在百米之外。他虽然不会飞行，但会的轻功可不少。有聚元功催动，速度依旧飞快。
<br/>　　田灵儿飞在天上，见苏重一闪一闪竟能跟上自己速度，顿时心里不乐意，不由加大法力束飞得更快。可不管她怎么飞，苏重都能不紧不慢的跟在身后。直到祖师祠堂前，她都没能甩开苏重。几乎同时出现在众人眼前。
<br/>　　苏重气定神闲，倒是田灵儿运转法力急促，累的小脸儿通红。
<br/>　　苏茹一看就知道怎么回事，肯定是自家女儿又在显摆琥珀朱绫御物飞行。却碰到了自家天才徒弟。只是用普通轻身功法，就轻松追上田灵儿。
<br/>　　对于苏重悟性，苏茹是极其惊叹的。聚元功即便是她来修炼，也有不少助力。能创造出快速绝伦轻身功夫，她一点儿都不奇怪。
<br/>　　“师娘。”苏重上前行礼问候。
<br/>　　看了眼气哼哼田灵儿，苏重没在意。虽然速度不落下风，甚至能轻松超过田灵儿，但苏重没觉得有什么可得意。
<br/>　　轻功毕竟是轻功，平坦大地上没问题，一旦进入穷山恶水险境，无法飞行就是一大弊端。他之后需要解决的问题又多了一个，要找个适合他的飞天遁法。
<br/>　　祠堂门口，几个师兄弟已经站成一排，一个个摆着张苦瓜脸。祠堂门口的青铜定已经换了新的，旧的已经被他劈成两半。
<br/>　　苏重默默的走向队伍最后。他在大竹峰排行最后，一直站在末尾。
<br/>　　“尔旦，你来我身边。”苏茹叫住苏重。
<br/>　　“你这几位师兄啊，那可是真有能耐，自己修炼不怎么样，竟还有功夫管别人！”苏重一脸煞气的瞪着众人。
<br/>　　苏重搞不明白自己这几个师兄又干了什么，明显是把自家师娘给惹火了。
<br/>　　“必书，我听你刚才打赌，你和你小师弟动手，说能几招取胜来着？”苏茹斜睨着杜必书。
<br/>　　苏重脸皮一抽，闹了半天，又是杜必书作死。
<br/>　　这家伙原本是个赌徒，被田不易看重带上山修炼，自从取了杜必书这个名字后，他就从来没赢过。可他总是不自制，变着法的和别人打赌。
<br/>　　“一……一招。”杜必书缩了缩脖子。
<br/>　　苏重哭笑不得，这位师兄老毛病又犯了，估计不知道怎么就聊到自己，打赌说能一招打败自己。正好被师娘听到，这才有了今天这场面。
<br/>　　“一招？那你就用一招来打败你师弟给我看看。”苏茹柳眉倒竖，狠狠瞪着杜必书。
<br/>　　“尔旦，去让你师兄好好指教指教！”苏茹寒声道。
<br/>　　苏重满脸苦笑，这是让自己下狠手，狠狠教训一顿杜必书啊。两人无奈站在中间。
<br/>　　杜必书苦兮兮，被师娘抓住把柄，今天有他好受的了：“小师弟，一会儿得罪了。明天我给你弄点儿好酒给你赔罪。”他凑到苏重身前笑声嘀咕。
<br/>　　“不用，还是我给你做顿饭吧。”苏重笑眯眯。
<br/>　　杜必书被苏重说的一愣，不等他反应，苏重已经走开：“师兄请。”
<br/>　　杜必书回过神，不明所以的看着苏重，迟疑片刻才道：“那师弟小心了。”
