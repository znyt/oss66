　　第92章
<br/>　　杏花岭和飞虎寨穷得叮当响，委实没什么能输给程大雷。三人最后是以酒当赌注，规矩和程大雷想得不一样，谁赢了谁喝酒。
<br/>　　程大雷不胜酒力，小白狼却是令他大吃一惊，一介女流喝酒如喝水。
<br/>　　最后，程大雷酒囊里的酒越来越少。看着高飞豹和小白狼开怀畅饮的模样，程大雷心里琢磨着，这两人不会是合起伙来骗自己酒喝吧。
<br/>　　到天快擦黑的时候，杏花岭和飞虎寨两拨人前后脚都回来了。两方人马各自聚在一处，嘀嘀咕咕商量着什么。
<br/>　　估计是抢劫的计划，城外这些富户往往墙高人壮，想要打劫他们，不好好筹划，很容易把人折在里面。
<br/>　　“程当家，都到现在了，你们的目标是哪儿，可以给我们交个底儿了吧？”高飞豹问问。
<br/>　　“你们忙你们的，别瞎打听，刺探我们的商业机密。”程大雷。
<br/>　　程大雷这次神神秘秘，难免高飞豹心里好奇，可程大雷不说他也没有办法。
<br/>　　到一更天，徐神机一伙也都回到土地庙。
<br/>　　“快过来喝口酒，先暖暖身子。”
<br/>　　今天天气奇寒，程大雷窝在破庙里都撑不住，何况几人今天在外奔波了一整天。
<br/>　　“大当家，我们在城里摸了一天，还画了一幅黑石城的地形图。”
<br/>　　“嗬，快拿来我看看。”
<br/>　　徐神机从怀里摸出一封草图，上面画有四面城墙，城主府，以及城内的主要街道。
<br/>　　“军师，不错嘛，我还不知道你有这一手。”程大雷忍不住赞道。
<br/>　　徐神机脸上讪讪：“不是我画的。”
<br/>　　“我一早就猜到不是你，那是谁画的，莫非是少羽，看不出来，你文武全才么！”程大雷。
<br/>　　“我提笔比提枪重，怎么会是我。”
<br/>　　“那是……”
<br/>　　就在这时候，一只胖嘟嘟的手举起来，张肥道：“其实是俺。”
<br/>　　程大雷愣了好久，盯着张肥一双大眼睛，良久没说出话来。
<br/>　　张肥倒是被看得有些不好意思，挠挠头道：“家父在世时，本来是想让俺进京参加科举的。后来世道越来越乱，也就没去成。”
<br/>　　程大雷双手抱拳，献上敬意。
<br/>　　有这份地图，接下来的事情必然会顺利许多，却是没想到外表粗糙的张三爷还有这脑子。程大雷在地图上标注行动时的路线，脑袋里思索着接下来的计划。
<br/>　　“程当家，程当家……”
<br/>　　高飞豹连唤了两声，程大雷才回过神来，询问的目光看着他。
<br/>　　“程当家，既然咱们三家都聚在一处，索性合起伙来，干一票大的？”高飞豹。
<br/>　　“什么大的？”
<br/>　　“咱们放过朱家吕家不管，进城把王家抢了。和王家一比，朱吕两家可就是小门小户，真干他们一票，咱们这一个冬天都吃喝不愁。”
<br/>　　高飞豹说完，小白狼目光也看着程大雷，看来她也有联手的意思。
<br/>　　程大雷挠挠头：“合伙的事情就算了，咱们还是各干个的。”
<br/>　　“整个黑石城，除了贾家以为就是王家，程当家不会是怂了吧。”高飞豹。
<br/>　　高飞豹竟然以为激将法对程大雷有用，这是真不知道程大雷在厚黑学上的造诣。程大雷笑笑：“是呐，我这个人比较谨慎，况且我这次也不是为了求财。”
<br/>　　“不求财，那程当家是为了什么？”高飞豹和小白狼同时问道。
<br/>　　程大雷顿了顿，口中吐出两个字：“扬名。”
<br/>　　这两个字令小白狼和高飞豹都是一头雾水，程大雷这次行动怎么都透着神秘。都到现在，还没摸清他的目的，要说现在蛤蟆寨富得流油，为啥要冰天雪地出门受这苦罪。
<br/>　　翌日，三家继续撒出人去踩盘子，还是剩下程大雷、小白狼、高飞豹三个在一块掷骰子。小白狼和高飞豹想方设法想要套程大雷的话，弄清他这次行动的目的。
<br/>　　程大雷顾左右而言其他，一点口风也不露。
<br/>　　等晚上的时候，蛤蟆寨的人回来得越来越晚，等第三天几个人是彻夜未归。
<br/>　　“程当家，你的人干嘛去了，这么晚不回来，你不担心么？”
<br/>　　“我又不是他们婆姨，他们晚上不回来，我担心什么。”
<br/>　　“难道……他们今夜就动手了？”
<br/>　　“哈，不至于，不至于。”
<br/>　　程大雷说话云遮雾绕，无论如何套不出他的实话，只是小白狼和高飞豹心里嘀咕着。
<br/>　　到第二天，只有刘悲和徐神机回来。张肥，秦蛮，林少羽三人竟不见了。
<br/>　　“徐军师，昨晚忙什么了，该不会进城喝花酒去了吧？”高飞豹。
<br/>　　徐神机打着哈哈，窝在稻草堆上补觉去了。高飞豹心里暗暗骂娘，这徐神机和程大雷学得，越来越狡猾了。
<br/>　　今天，刘悲和徐神机没有再出去，小白狼和高飞豹派出去的人天没黑就回来了。
<br/>　　“二当家，你看看这个，现在城里贴的到处都是。”手下递过来一叠白纸。
<br/>　　高飞豹越看眼睛瞪得越大，过了半晌，从纸上抬起头，眼睛盯着程大雷。
<br/>　　“程当家……”
<br/>　　“什么事？”
<br/>　　“需要玩这么大嘛？”
<br/>　　白纸拍在地上，小白狼也以同样的目光看着程大雷。
<br/>　　程大雷挠挠头：“二爷，这就是你不对了，我好不容易让弟兄们连夜贴上的，你怎么又给我揭了回来。”
<br/>　　纸落在地上，可以清楚看到上面程大雷鬼歪七扭八的字迹。
<br/>　　『黑石城的富户听好了，劫富济贫，替天行道的蛤蟆大王到了，你们平日有欺男霸女，为非作歹的，限你们在一日内散尽家财，如若不然，蛤蟆大王不日便会登门拜会』
<br/>　　下面列着长长一串名单，贾家、王家、朱家、吕家皆在其中。
<br/>　　这只是白话版，通俗易懂，此外还有其他版本。
<br/>　　『善信的人有福了，世间最暴力的恶当有最暴力的善来了解，当正义审判的日子来临时，当死的死，当活的活，所有罪恶绝无藏匿之地』
<br/>　　小白狼和高飞豹一头雾水，琢磨了这么久，到现在才大概摸清楚程大雷的用意。
<br/>　　“程当家，你是要打劫整座城？”
<br/>　　“没那么夸张，扬名而已，扬名而已。”程大雷摆手道。
<br/>　　
<br/>(本章完)
