　　这只十五人组成的队伍最后也在佘山消失了，没有一人下山而归。
<br/>　　白镇人民更是惶恐不安起来，第二日，又有一只二十人的队伍组成，继续上山探索寻找，结果如是。
<br/>　　一去不复返。
<br/>　　报警，报警，失踪的家人一个个疯狂的报警，一下子失踪了那么多人，已经足以引起官方的注意。
<br/>　　很快，佘山就被封山，不准一人轻易上山。
<br/>　　一辆辆鸣笛的警车呼啸而来，停在了佘山山脚下的大青石旁，一批又一批身穿制服的警员带着白镇人民的希望上山了。
<br/>　　这些上山的人真的是被吃人的怪兽给吃了，还是发生了其他的意外。
<br/>　　真正的答案无人知晓，只能将希望寄托在这些警员身上。
<br/>　　时间在流逝，夜晚降临，白镇人民绝望了。
<br/>　　身穿制服的警员也无一人返回，这座百蛇出没的佘山，赫然变成了一个黑暗的无底洞，只要人进了山，就会被吞噬的无影无踪。
<br/>　　捕蛇人失踪，警员失踪，赫然，整个佘山笼罩在一片死亡的恐惧阴影中，事件影响力度之大，震惊整个温城。
<br/>　　将官方的大人物都给惊动了。
<br/>　　然而，在蛇王庙中。
<br/>　　庙堂香火长相续，应得人来礼拜多。
<br/>　　此时的蛇王庙，香火鼎盛，很多失踪的家人都来祈求蛇王保佑，让他们的家人能够平安的回来。
<br/>　　佘山出了会吃人的怪兽，大家虽然报了警，但只是徒添了失踪的人数而已。
<br/>　　于是乎，一个个自发的来蛇王庙祷告，祈求蛇王的庇佑。
<br/>　　“蛇王大人，求你保佑我的老公从佘山上安全的下来，他已经进山三天没有回家了”
<br/>　　一位中年妇人焦虑的冲着雕像不断在磕头，磕着磕着，一行清泪滑落。
<br/>　　与此同时，一位八九岁的小男孩，跪在蛇王雕像前，认认真真，有模有样的先磕了三个响头，然后抬头明眸，略带青涩的声音：“蛇王大人，我爸爸上回答应我，如果他能在佘山捕到一条好蛇，他就给我买一件心爱的玩具，我现在不要玩具了，我只要我爸爸能够安全的回家”
<br/>　　老人，妇人，小孩，一幕幕情景不断的在蛇王庙发生。
<br/>　　不断的有人来磕头许愿，而且出奇的统一，都是希望能让自己的家人能够平安的回来。
<br/>　　而蛇王雕像后面，藏匿着一条迷你的金银蛇，聆听来磕头许愿者的愿望。
<br/>　　“吸血蛇，你终于回到佘山了吗”
<br/>　　如今，整个白镇流传说佘山有吃人的怪兽，这几日不断的有人来磕头许愿，祈求家人的平安归来。
<br/>　　涂小安心里大致有数了，什么吃人的怪兽，想必是吸血蛇归来了。
<br/>　　这成长到二十米的巨蟒，吃人也不算什么。
<br/>　　这几日，上山失踪的人越来越多，但让涂小安疑惑的是，自己分布在佘山的电蜂，却没一只传来吸血蛇的消息，这就让他有点不解。
<br/>　　但事态的严重性，也已经远远超出了他的想象，看来自己必须立刻上山才行。
<br/>　　不管是为了自己的私怨，还是为了不让更多人的遭难，他都不得不立刻出手了。
<br/>　　但刚打定主意的涂小安忽然一怔，一个放眼望去，庙堂内黑压压的涌现了很多人进来，他们每人手中高举三根香，庄严肃穆，好像在做一件非常重要的事情。
<br/>　　一眼看去，涂小安居然在里面看到了很多熟悉的面孔，都是当地有威望的捕蛇人，他余光下既然还看到了自己的师傅沈震南。
<br/>　　只见沈震南扶着一位弯躯如弓，满头白发的老人领头走来，老人手如槁木，岁月的触角爬满了他的额头。
<br/>　　等涂小安看清楚这位老人的样子，差点震惊的从雕像上面摔下来。
<br/>　　“小人林得丰，携白镇人民祈求蛇王出世降妖除魔，救白镇百姓与水火之中”
<br/>　　老人走到蛇王雕像面前，甩开了沈震南扶着他的手，然而高举三炷香，佝偻的身子正了正，发出了声如洪钟的高喊声。
<br/>　　声音之大，好像是用尽了老人的全身力气。
<br/>　　“祈求蛇王出世降妖除魔”
<br/>　　庙堂之内，香火缭绕，众人面色庄严肃穆，齐声一吼，声音绕梁三尺而不绝。
<br/>　　相传数百年前，白镇地区发生了一次大霍乱，百姓无奈之下，磕头请愿，求佘山蛇王出世，那一次，白镇的上空出现一条巍峨的大巨蟒，它如神灵下凡，口吐焰火，驱除了瘟疫。
<br/>　　现如今，佘山出现了吃人的怪兽，不断的有捕蛇人一去不复返，活不见人，死不见尸。
<br/>　　整个白镇都活在恐慌之中，官方一时半刻都解决不了。
<br/>　　短短几日，失踪人口达到了五十人以上。
<br/>　　说失踪是好听，大家心里多少有点清楚，这些亲人恐怕在也回不来了。
<br/>　　老人林得丰在众人跟着喊完之后，再次高喊：“请蛇王出世，降妖除魔”
<br/>　　一声喊完，他高举三炷香，朝雕像双膝一弯，重踏而跪。
<br/>　　老人这一弯，身后之人跟着纷纷的双膝跪地，可当大家都跪在了地上，却发现老人根本没有真正的跪下来。
<br/>　　只见李得丰诧异的望着蛇王雕像，一双浑浊的眼睛慢慢的睁大，露出了喜出望外之态。
<br/>　　不是他不跪，而是他发现自己根本跪不下来，无形中好像一股力量死死拖着他，诚然不让他跪在地下。
<br/>　　隐藏在雕像后面的涂小安看到老人要冲着自己下跪，顿时诚惶诚恐，要是可以，他恨不得先给着老人跪下，磕三个响头在说。
<br/>　　老人是谁...
<br/>　　白镇最德高望重的老人，曾经在十里八乡有一个响亮的名称，人中蛇王，百毒不侵。
<br/>　　他的弟子遍布白镇，最有出名的就是现在站在他身后的沈震南，一代捕蛇大师，都是他的亲传弟子。
<br/>　　老人林得丰在前，沈震南就什么都不是了。
<br/>　　为什么涂小安那么诚惶诚恐，原因很简单，他的父亲，也是这位老人的亲传徒弟。
<br/>　　严格意义来说，这是他的师公。
<br/>　　涂小安没想到佘山的不安定，将这位德高望重，安享晚年的老人都给惊动了出来。
<br/>　　可谓罪过呀。
<br/>　　但说起来，请蛇王出世，也只有这位老人够资格。
<br/>　　此时，庙堂黑压压的人跪地膜拜，独独老人怔怔的望着蛇王雕像，心中难以言喻，老人家感受到了蛇王的存在。
<br/>　　就在此时...
<br/>　　“叮...任务触发，佘山浩劫，宿主解决佘山之难，完成任务，等级提升三级，奖励1000000信仰点数”
<br/>　　系统的声音悄然的出现在涂小安的脑海中。
<br/>　　顿时，他的蛇眸猛然的一亮，精光四射，涂小安等的就是这一刻，触发任务。
<br/>　　庙堂之上，蛇王雕像前，一道金光灿灿的蛇影若隐若现了起来。
<br/>　　霎时间，所有人都撑着惊骇的目光望着，无一人敢动弹一分。
<br/>　　蛇王显化了...
<br/>　　倏然，金光灿灿的蛇影冲天而起，化为一道金光，长啸庙堂，旋即一闪而逝。
<br/>　　咚咚咚...
<br/>　　庙堂之外，响起了深沉而悠远的钟声。
<br/>　　蛇王出世，钟声长鸣...
