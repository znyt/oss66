<br/>　　所以她一上来便用上了杀敌一千自损八百的凶招！
<br/>　　只是让她意外同时也不意外的是，丧尸首领速度极快，一道残影闪过之后，便连带着一道极细的黑血闪过，下一刻它的身躯便已经偏移到了旁边地带。
<br/>　　弯弯的镰刀尖端还留有黑色血渍，以及微不可查的一丁点丧尸碎肉，与银色的流光纹路结合当下，显得妖异异常。
<br/>　　丧尸首领在最快的时间里，用了极小的代价逃脱！
<br/>　　而它的身影还未停下，继续用极快的速度移动着，眼看就要隐入周围的丧尸群之中。
<br/>　　丧尸首领不仅有战力还有智力，这并不是公平的角斗场，周围的观众也不是观众，而是全部以它为首的丧尸们！
<br/>　　隐藏，破出，攻击，一旦配合周围的丧尸群，它想要收了江凌若的性命胜率便再次升高。
<br/>　　江凌若眼眸犀利一闪，镰刀上银色的流光刹那间绽放，忽然光芒四射在这漆黑的深夜里，四周一大·片的地带忽然耀眼刺目让人睁不开眼，而那些留下来围住两人的丧尸群没有视觉，却集体躁动了起来。
<br/>　　紧接着下一秒，江凌若的周身忽然光芒万丈般的爆发出一股能量。
<br/>　　领域出！

<br/>　　银色领域用远超光的速度蔓延，追上了丧尸首领的脚步将它收入，也直接笼罩了整个战斗范围，将在场的其他丧尸都隔绝在外，无论是视野、声音还是能量波动，也一瞬间的消失。
<br/>　　外部的丧尸没有思考能力，并不知道是怎么回事的它们，只感受到了江凌若和丧尸首领的气息双双消失，有限的智力让它们疯狂的骚·动不安。

<br/>　　打斗不断间连带着无数道电光火石，一阵阵刺耳的摩擦声在各个地方响起，领域内·壁更是不停的被冲撞挤压变形。
<br/>　　两道身影一黑一银，飞速的闪烁在前后左右，每一次的交锋都带着让人应接不暇的身影闪动。

<br/>　　那是江凌若一点点被削下的头发，丧尸首领的四肢指甲锋利如剑，每一次的攻击也都直逼江凌若致命部位。

<br/>　　刷——
<br/>　　双方再次分开，分别站在了领域的最两端。
<br/>　　江凌若手持巨大镰刀，暗紫色的长发不再，取而代之的是一头被削到贴近耳畔的短发，她的双手皮肤上好几道抓痕在不住流血，鲜血淋漓一点点落下，顺着镰刀柄一路延伸到刀刃上。

<br/>　　那就是丧尸首领的衣服！
