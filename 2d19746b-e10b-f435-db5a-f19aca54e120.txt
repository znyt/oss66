　　三、五天时间很快过去，老头子的春猎也如期展开，李承乾自然是混在队伍里面跟着一起去的，只不过相对于那头子，他的‘任务’却要繁重重多。
<br/>　　长达一天半的行军不用多说，到了地方之后安营扎寨也不用他来操心，只是把几个主事的老家伙叫到一起费了他不少的力气。
<br/>　　直到下午天色将黑的时候，长孙无忌、房玄龄、杜如晦、秦琼等人才如约到来，挤到李承乾那间并不大的帐篷里面共商大事，当然，这其中少不了老李的贴身太监——老方。
<br/>　　“小子，你这地上铺的是啥东西？”程老货按要求在帐篷外面把鞋子脱掉，走进去之后，发现在地上铺着一层厚厚的垫子，人踩在上面多少有些扎脚的感觉。
<br/>　　“这叫毛毡，就是那些我们收购来的羊毛制出来的。”李承乾一边笑着给一群大老们见礼，一边说道。
<br/>　　没办法，现在不是在正式场合，他这个十四岁的小年轻辈份最小，不先见礼说不过去。
<br/>　　“哦，不错！就是有点扎脚，没有你搞出来的那个什么地摊踩着舒服。”老程点点头，随便找了一个地方坐下，伸手在地上摸了摸：“这东西能卖钱？谁买啊？”
<br/>　　“买是一定会有人买，但是这个估计不会卖的很贵吧。”长孙无忌同样找了一个位置坐下，拈起一块放在地上的小块样品，拿在手里打量着。
<br/>　　“这个是真对普通牧民的。”李承乾同样拿过一块毛毡的样品，举在手里说道：“这东西有保暧的功效，可以制成鞋子、床垫、甚至可以制成房子，草原上能够保暖的东西不多，这个只要出现在草原上，就会牧民生活必备的用品。”
<br/>　　“那有啥用，牧民能有几个钱。”程妖精总是能够找到事情的重点，而且能一言点明主题。
<br/>　　老货手里同样有着一片作为样品的毛毡，只不过眼中满是不屑。
<br/>　　“程伯伯不要忘了，这东西只是普通羊毛制成的，一文钱五斤的收购价和白来的没什么区别，而制成这东西就算是按斤卖，也要卖到十文一斤，这已经很赚了。”
<br/>　　“这毛毡殿下打算按斤来卖？”一直没怎么说话的房玄龄突然问道。
<br/>　　李承乾扭头，看着老房笑着说道：“这怎么可能，按斤卖只是打个比方而以，实际上这东西是按匹卖的，一匹长四丈宽三尺，价格我打算定在五百到七百文。”
<br/>　　老家伙们对方对视着，李承乾说的这些东西他们在里面可都是有股份的，所以他们也要考虑这个价格他们能为家里赚多少钱。
<br/>　　身为一个男人在外面拼死拼活为了自己的理想是一回事，能让家人过的好的些，为子孙后代多留一点积蓄也是题中应有之意，总不能当了一辈子的官，最后家里穷的揭不开锅。
<br/>　　当然，历朝历代都有些官员一生清贫，死了之后家徒四壁。
<br/>　　但这样的总是少数，而且有些时候并不是他们不想给后代留些积蓄，实在是因为原则不想利用职位之便捞钱，做生意的话又不会，干什么赔什么，再加上没人带，所以才导致了家徒四壁这一情况发生。
<br/>　　可是大唐有了李承乾这个‘变态’之后就不一样了，这家伙简直就像是一个财神爷，总是有各种各样的办法能够弄到钱。
<br/>　　而且自己发财还不忘了朝中的那些清水衙门，总是会有意无意的扯着他们一起，让他们不至于把日子过的连普通老百姓都不如。
<br/>　　“高明啊，这价格似乎低了些吧，就算是七百文我们好像也没赚到什么钱。”良久之后，还是长孙无忌首先开口，必竟他和李承乾的关系近一些，说话方便。
<br/>　　“舅舅您的算法错了，这东西……”李承乾将小块的毛毡放在手里掂了掂：“这东西只是底端产品，弄出来只是为了换取更多羊毛，并不真的靠它来赚钱。”
<br/>　　“哎，我就说嘛，这小子一肚子坏水，他搞的事情怎么可能这么简单，来来来，小子，快点说说，还有什么别的门路！”程妖精咧着大嘴，大咧咧的说着，一点都没有因为李承乾行过冠礼就客气一星半点。
<br/>　　李承乾苦笑着摇摇头，对这老货也是没招，连老李都对这滚刀肉没什么好办法，他又能如何。
<br/>　　“知节莫要多言，且听殿下说完。”秦琼看出李承乾的尴尬，伸手在程老货的肩上拍了一下，把他的胡言乱语打断。
<br/>　　“到底靠什么赚钱我们一会儿再说，叔伯们先试试这草原上的牛奶如何？”李承乾先是对秦琼感激的笑笑，然后命人拿出一个玻璃罐子，打开封口之后，往十余个杯子里倒入一些白色的粉末。
<br/>　　“这又是什么东西？牛奶什么时候变成这个样子了？”程妖精好奇的拿过一个杯子，皱眉看着里面白色的东西，又放到鼻子下试着闻了闻：“小子，这该不会是砒霜吧？你想把俺们全都毒死？”
<br/>　　咳咳……，李承乾被这口不择言的老货气的差点被口水呛死，张了半天嘴一个字都没说出来，最后一生气，抓过一个杯子，一仰头把里面的白色粉末全部倒进了嘴里。
<br/>　　“这叫奶粉，是牛奶或者羊奶干置之后加工出来的，可以放置在阴凉的地方保存很长时间。”一大口干奶粉倒进嘴里，一时半刻还没有化开，随着李承乾吱吱唔唔的说话，不断从他嘴里喷出来，看上去说不出的滑稽。
<br/>　　“哎，老夫就是说说，小子你那么着急干什么。”程老货大眼珠子一翻，低声咕哝一句，也学着李承乾的样子，把杯子里的奶粉倒进了嘴里。
<br/>　　“程老匹夫……”杜如晦无奈的摇摇头。
<br/>　　屋子里这么多人，聪明的又不是只有程妖精一个，用得着把话说那么狠么，连砒霜都弄出来了，欺负大家没见过砒霜怎么着？在坐的谁不知道砒霜是什么颜色的？还用得着这老货来说。
<br/>　　“嗯嗯，这东西好吃，挺香的！”也不知道程妖精这老货身体到低是什么材料构成的。同样一口奶粉，李承乾那边被噎的到现在还在翻白眼，这老货却已经全都咽了进去，甚至还在向拿着奶粉瓶的杨雨馨招手，示意把瓶子给他拿过来。
