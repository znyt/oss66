<br/>				    郭信宁点了几个菜，又叫了些酒，几人围坐着，一时间他却不知说些什么。<br/>　　<br/>　　    初见时在林中，他觉得他们三人是弱者，他能帮则帮，一路上也以着照顾的方式与之相处，只是到后来，却发现对方的实力是深不可测。<br/>　　<br/>　　    如今，他纵是郭家出色的子弟，在凤九的面前也不敢托大，相反之，就算是以平辈之交也已经有些越距，就更别说像谢家兄妹那般随意的开口打趣之类了。<br/>　　<br/>　　    “这里的风景真好，小九，你看，这样在高处望去，那些灯像不是一颗颗星星？”谢诗思笑眯着一双眼睛问着，整个人趴在围栏边，兴奋的看着那远处的夜景。<br/>　　<br/>　　    “嗯，高处往下看，风景自是不一般。”凤九微点着头说着。<br/>　　<br/>　　    听着这话，郭信宁的目光落在他的身上，见他神情淡然，平凡的面容上带着一抹笑意，看着他那不起眼的容貌，他忽的有一个念头在脑海中浮起。<br/>　　<br/>　　    这也许并不是他的真容。<br/>　　<br/>　　    像他这样一名深不可测的强者，容貌又岂会这般普通？一时间，想着事情入了神，就这样直勾勾的盯着凤九看着，直到，看到他转眸挑眉朝他看来。<br/>　　<br/>　　    “我脸上有花吗？”凤九问着，一手端起面前的酒杯把玩着。<br/>　　<br/>　　    “我只是突然在想，你现在的这张脸，会不会不是你本来的容颜？”郭信宁神使鬼差的说了这句话，而后微怔，觉得有些冒犯了，便道：“抱歉。”<br/>　　<br/>　　    凤九听了也是一愣，继而轻笑着，她没有说话，只是轻晃着手中的酒杯，抿了一口酒。<br/>　　<br/>　　    旁边的谢玉堂听了不由狐疑的朝凤九的脸上看了看，盯了好半响，道：“这张脸上这么自然，怎么可能是假的？而且，也没有易容的痕迹啊！”<br/>　　<br/>　　    说着，看了郭信宁一眼，有些奇怪的问：“郭大哥，你怎么会那么觉得呢？”<br/>　　<br/>　　    听着谢玉堂的话，凤九嘴角抽了抽。这傻小子，她要是易容易得让他看得出，那她还用混吗？<br/>　　<br/>　　    “小九本来就是长这样的啊！”原本趴在围栏处的谢诗思也回过身来，道：“当时我们遇到他时他就长着这个样子的了，虽然不是很好看，不过看久了也觉得挺好的。”<br/>　　<br/>　　    就是当时他们初见他时，他给他们的感觉是一憨厚少年，就算到了谢家后，他给他们的感觉也就是一个不怎么说话，要是不找他，有可能一天都见不到人影的人而已。<br/>　　<br/>　　    但，自从他们掉进那林子后，她就觉得凤九跟变了个人似的，要不是他一直跟他们在一起，她都要怀疑是不是有人将他调包了，要不然怎么会突然变得那么厉害呢？<br/>　　<br/>　　    听着他们的话，凤九有些无语，她这样子不好看吗？这容貌比起她本来的容貌那自是没得比，但好歹也是清秀俊朗有余，倒也是，比起他们见惯的世家公子那些容貌，自然是稍微逊色，但怎么也不会说不好看吧？<br/>　　<br/>　　    看凤九眉头微拧的模样，谢玉堂用手肘撞了他妹妹一下，对着凤九讪讪的笑了。<br/>
