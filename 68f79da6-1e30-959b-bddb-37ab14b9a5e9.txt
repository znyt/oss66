　　月山家别墅的上空，一只由武装直升机组成的队伍呼啸而来，门前的空地上，布满了密密麻麻的身影。
<br/>　　这些身影，穿着CCG的统一制服，手中提着银白色的手提箱，其规模，甚至比进攻古董的时候都要强盛一些。
<br/>　　毕竟这次行动，除了月山家族之外，还有一名疑似SSS级别的喰种，CCG高层对这次行动也不敢有丝毫大意。
<br/>　　“啪嗒，啪嗒……”
<br/>　　一阵沉重的脚步声响起，即使是在如此杂乱的环境下，依旧让人听的十分清晰。
<br/>　　人群的最前方，有马贵将手持一柄长刀，缓缓走上了别墅前的阶梯。
<br/>　　“有马特等，劳烦你了。”对讲机中有些杂乱，和修政的声音从里面传来，有马贵将没有理会，平静的盯着面前的别墅大门。
<br/>　　身旁的搜查官们也都没有说话，有几名打扮和旁人不同的白色身影站在有马后面，这是他的下属，也是一直以来他的搭档们。
<br/>　　“咚！”
<br/>　　没有丝毫客气，有马直接踹塌了别墅的大门，在一阵尘土飞扬中走了进去。
<br/>　　别墅内的场景很豪华，摆满了价值不菲的家具，有一个阶梯连接到二楼，却意外的没有一个人影。
<br/>　　在这场行动之前，CCG早就派出了监管这里的斥候，月山家的喰种不可能跑掉，那这样说来，或许是还藏在什么地方。
<br/>　　有马脸色有些凝重，戒备着走上了二楼，脚下鲜红的地毯，让他有种怪异的感觉。
<br/>　　像是某种兽皮制作成的，踩上去软塌塌的，伴随着‘咯吱咯吱’的声音，像是想到了什么，有马蹲下身来，触摸了几下。
<br/>　　很细腻，不像任何家畜或者野兽，摸上去和人皮的质感相同，但是这个厚度，有点不太正常。
<br/>　　翻找了几下，没找到什么线索，有马站起身来，继续向二楼走去。
<br/>　　他闻到了一股极为浓郁的血腥味，味道的来源，就是楼上的某个地方，他心中涌上了一股不好的预感。
<br/>　　“咯吱，咯吱……”
<br/>　　安静的别墅内，灯火通明，有马带领着自己的一众下属，顺着血腥味来到了一个屋门紧闭的房间，一汪浓稠的血液，从门缝处流了出来。
<br/>　　似乎是因为时间的原因，这汪血液已经有干枯的迹象了，但因为浓度和数量实在太高，不仔细看的话是根本察觉不到的。
<br/>　　拧动了一下房门的把手，房间并没有上锁，心中带着戒备，有马将房门打开，看到了门后的场景，瞳目猛的一缩。
<br/>　　大概六七十平米的房间里，跪满了密密麻麻的无头尸体，这些尸体都已经干枯，似乎是体内的鲜血都被挤出来了一般。
<br/>　　看其背后已经干瘪下来的赫包，可以确认，这些尸体，原本都是等阶不低的喰种。
<br/>　　事实上这里的大多数喰种CCG都有记录，不管是任何等阶的喰种，身上的赫包都是有不同特性的，很容易让人分辨出来。
<br/>　　多多良，野吕，还有刚刚从库克利亚被营救出去的神代叉荣，这些都是隶属于青铜树的SS级喰种。
<br/>　　除此之外，还有众多身穿青铜树服饰的S级，A级喰种，包括他们这次行动的目标之一，月山观母也死在了这里。
<br/>　　难道是，月山家联合青铜树，对SSS级喰种‘修罗’发动了进攻，最后被屠杀掉了吗？
<br/>　　但是根据他们斥候传来的消息，并没有察觉到别墅内进出过什么人，更没感应到发生过什么战斗。
<br/>　　虽然为了避免暴露，斥候队所占据的地方稍远一些，但是这么多的高阶喰种发生战斗，也不可能让他们一点察觉都没有。
<br/>　　即使是他，也不可能将所有的战斗余波隐藏下去，仅有的解释，就是‘修罗’的力量远远超越了这些喰种，让他们没有任何反抗的余地。
<br/>　　或者说，除了‘修罗’之外，还有另外数量不明的高阶喰种帮忙，才可能发生这样的事情。
<br/>　　“有马特等，这里有发现。”就在有马皱着眉头查探尸体时，身后传来了自己副手的声音，随后，几人都凑上了前去。
<br/>　　床边的一个柜子里，副手发现了一串用绳子串联起来的干瘪头颅，这些头颅的下面，绑着一个昏迷了的A级喰种。
<br/>　　有马贵将微微一愣，走上前去，从柜子中拿起了一张便签，脸色微微阴沉了下来。
<br/>　　‘致CCG的各位同僚，我是三等搜查官刘哔，青铜树和月山家的麻烦我已经帮你们解决掉了，并且给你们留下了一个活口，便于你们审问月山家小少爷，月山习的动向，对了，请帮我向木嶋班长和我的好朋友旧二多福打声招呼，谢谢。’
<br/>　　纸条上的消息，证实了众人先前的猜测，这些死去的尸体，果然是青铜树和月山家的势力，但让有马贵将搞不清楚的是，同为喰种的‘修罗’，为何会和他们引起冲突。
<br/>　　青铜树还好理解，毕竟‘修罗’杀掉了他们的首领芳村艾特，但据他所知，修罗和月山家并没有什么仇恨。
<br/>　　甚至从库克利亚逃离的时候，还是月山家给他提供了方便，按理说，双方并不应该是敌对关系。
<br/>　　手中握着纸条，有马鬼将心中开始犹豫起来，他有些担心，这里面是否会有什么阴谋。
<br/>　　但根据纸条上的信息，月山家并没有被完全剿灭，至少还有一个情报之外的人，不知道躲藏到了哪里。
<br/>　　“有马特等，屋内是什么情况，可否安全？”腰间的对讲机中，和修政的声音再度传出，有马略微犹豫一下，摘下对讲机来，将这边的情况简单汇报了一遍。
<br/>　　随后，对讲机的另一边也安静了下来，等待了几分钟的时间，方才传来了指令。
<br/>　　“将喰种‘修罗’的危险程度调至SSS+，调出月山集团所有地区的清单，对别墅内的唯一活口进行审问，务必查出月山习的去向。”
<br/>　　停顿了一下，和修政的声音继续传来：“我怀疑，这是‘修罗’对我们设立的一个圈套，月山习藏身的地方，必然会有所埋伏，甚至‘修罗’也会在那里。”
<br/>　　“但是我们绝不能退缩，月山家族对于社会的影响力实在太过于巨大了，如果不清理干净，恐怕后果不堪设想！”
<br/>　　对讲机中，和修政的语气无比严肃，向参与这场行动的搜查官们，下达了指令。
