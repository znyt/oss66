        见童筱颖把自己的电话给挂断了，落以柔还是没忍住暗自窃喜。
<br/>
<br/>        师父说，不让她插足不该插足的事。
<br/>
<br/>        她这不算插足吧！
<br/>
<br/>        毕竟，她和傅夜沉之间清清白白的，而且还一点暧昧关系也没有。所以，她也只不过是跟童筱颖耍耍嘴皮子的功夫。
<br/>
<br/>        落以柔想通了，傅夜沉不爱她没关系，那么她就这么默默地一辈子待在他身边好了，在他出差的时候照顾他的生活起居，工作上成为他最得力的助手。
<br/>
<br/>        另一边，童筱颖冷静下来后觉得，她若是真的被落以柔气到了，那也太不值了。
<br/>
<br/>        她等了许久，也未等到傅夜沉的电话，可想而知，落以柔又耍了她。
<br/>
<br/>        但是，当她再次主动给傅夜沉打电话过去的时候，傅夜沉的手机仍旧打不通。
<br/>
<br/>        而接下来的这些天里，傅夜沉的手机号码一直处于“不在服务区内”的状态。
<br/>
<br/>        大哥到底又在做什么？连个电话也不主动打给她？
<br/>
<br/>        有个时候，童筱颖会产生一种恍惚，感觉她和大哥恍若不是一个世界的人。
<br/>
<br/>        直到一个星期后……
<br/>
<br/>        这天清晨，童筱颖在家吃过早餐后，臂弯上挂着手提包，离开家门后，在电梯门口，遇上了正从电梯里出来的傅昼景。
<br/>
<br/>        “爷爷让我接你，一起回一趟大庄园。”傅昼景眉头紧锁地说道，见童筱颖手中提着包，于是主动伸手过去帮她提着。
<br/>
<br/>        童筱颖还未缓过神来，臂弯上的手提包就被傅昼景拿走了。
<br/>
<br/>        以前他们在一起的时候，他每次都会主动替她提包，所以，这些小默契久而久之变成了一种习惯，以至于习以为常。
<br/>
<br/>        “怎么这么突然？”童筱颖并未在意自己和傅昼景之间的这种小细节，而是把关注的焦点落在了爷爷身上。
<br/>
<br/>        傅昼景忽然间冷笑了笑：“我如果说是爷爷让我接你一起回傅家大庄园，是为了参加大哥的葬礼，你信吗？”
<br/>
<br/>        下一秒，空气凝结。
<br/>
<br/>        童筱颖有些恼火地瞪着傅昼景，疾言厉色起来：“昼景，就算你对大哥再怎么不满，他也是你亲哥哥，你何必这么诅咒他！”
<br/>
<br/>        “我对大哥何止是‘不满’，而是恨！”傅昼景意味不明地笑了笑，“夺父之恨，夺妻之恨！”
<br/>
<br/>        童筱颖二话不说地从傅昼景的手中夺回自己的手提包，冷冷地回答：“我自己回傅家大庄园就行了。”
<br/>
<br/>        “筱颍，我两私奔吧！大哥他真的死了，在美国，为了救他身边的那个女秘书，车子被撞入了大海。”傅昼景踱步上前，一把抓住了童筱颖的手腕。
<br/>
<br/>        “傅昼景！你够了！”童筱颖用力扭动着手腕。
<br/>
<br/>        傅昼景却将她揽入怀中，紧紧地束缚住，呵斥道：“大哥刚到美国的第二天清早就出事了！”
<br/>
<br/>        时间点刚好对上了！
<br/>
<br/>        童筱颖记得那个时间点，也就是那个时候起，她就再也联系不上傅夜沉。
<br/>
<br/>        她神色慌张地从手提包里掏出手机，直接拨打落以柔的电话，但是落以柔的电话提示已关机，她不依不饶地又拨通了郑忠威的电话。
<br/>
<br/>        好在，郑忠威的手机号码是拨通了，但是无人接听。
<br/>
<br/>        “我说过，爷爷命我来接你，是带你回去参加大哥的葬礼的。不过，你不是以大哥的发妻身份去参加。而是以我的前妻的身份去参加。”傅昼景眉头紧锁道。
<br/>
<br/>        童筱颖双手握拳，愤恨地瞪着傅昼景。
<br/>
<br/>        “筱颍，爷爷已经命苏漫雪作为傅家的大少奶奶在替大哥守灵。”
<br/>
<br/>        “我不信！”童筱颖忍不住歇斯底里起来，双眼瞬间布满了血丝。
<br/>
<br/>        “你若不信，我带你回傅家大庄园。”傅昼景说完，直接抓着童筱颖的手腕，强行拉着她一起进了电梯，然后去了地下停车场。
<br/>
<br/>        童筱颖被傅昼景强行塞进了副驾驶座里，他一如既往地为她贴心地系上安全带。
<br/>
<br/>        从他的行为举止上来看，她看不出来他是在骗她的，心里已经不再是发慌，而是在一阵又一阵地发痛。
<br/>
<br/>        童筱颖的脑海里不停地在回忆着大哥傅夜沉这段时间的所作所为。
<br/>
<br/>        早在之前，她就感觉到大哥和连华生在计划着什么，因为连华生好几次行色匆匆的离开，都被她撞见了。
<br/>
<br/>        后来他们吵架，从他突然找裴金澈回来接管傅氏集团，然后毫无征兆地带着团队去法国起，她就感觉到他有事瞒着她。
<br/>
<br/>        明明感觉到他不对劲，他却能在她面前表现得十分若无其事。
<br/>
<br/>        童筱颖眼神涣散，目光没有焦点。她蜷缩在副驾驶座上，一手抱着双腿，一手握拳抵在了嘴唇上，而贝齿却在紧紧地咬着指骨不放。
