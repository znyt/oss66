　　为了腊梅的安全，叶少阳一咬牙，一道灵符打过去，在空中燃烧起来，落在道士身上，砸出了不少火星。
<br/>　　“开枪！”
<br/>　　砰砰砰！
<br/>　　一通枪响，这道士刚从围墙上跳下去，就被打成了筛子。他虽然会法术，但到底是个人，被一通射击，当场毙命。
<br/>　　叶少阳看他死了，倒没有急着去检查，而是钻进了他的房子，查找起来。结果在一个没有窗户的房间里，找到了很多法器、法药，叶少阳打着火把照过去，角落处突然有一个人影跳出来，双手来抓叶少阳。
<br/>　　叶少阳本能地后退避过，吓了一跳，把火把举到跟前一看，顿时倒吸一口冷气：是一具干尸，被一根铁链锁着脖子，铁链那一头钉在墙上，干尸发现生人，本能地想要扑杀，将铁链挣得咯吱咯吱响。
<br/>　　这时候门外响起脚步声，叶少阳用火把照过去，见是秋镇长，说道：“危险，别过来！”
<br/>　　秋镇长急忙站住，然后就发现了这具干尸，双腿顿时软了，靠在墙上，浑身筛糠般抖起来。
<br/>　　叶少阳道：“没什么，这是干尸。被拴在墙上，过不来的。”
<br/>　　叶少阳说完，拔出七星龙泉剑，从僵尸长大的嘴巴里插了进去，不需要用什么法术，七星龙泉剑自身的灵力，杀一个普通的僵尸不在话下。
<br/>　　“呜……”
<br/>　　僵尸浑身抽搐，口中发出含混不清的低吼，倒在地上，身上冒出一股尸气，很快就不动了。
<br/>　　叶少阳望着地上的尸体，叹了口气道：“没想到竟然杀人养尸，本来我还有些不忍心他的死，现在看来，他真的是死不足惜。”
<br/>　　“杀人养尸……”秋镇长见僵尸不动了，逐渐缓过来，问叶少阳：“什么意思？”
<br/>　　“就是把人杀了，然后用秘法激发尸变，圈养起来。当然如果有合适的尸体，也可以不用自己动手杀人，不过这个显然是杀人养尸的。”
<br/>　　秋镇长皱眉打量着这具尸体，道：“何以见得？”
<br/>　　叶少阳道：“这还不简单，你看这尸体身上的衣服，我就没见过自然死亡的人这么穿的。”
<br/>　　秋镇长一怔，打量过去，这尸体身上的衣服破破烂烂，碎成了条，但还是依稀可见，是一件普通的布衣。立刻就明白了，眼下虽然是乱世，但是内地这边一直还算太平，一般老百姓也有口饭吃，如果有人死了，就算是再穷的人，家人至少也给买件寿衣穿，不可能穿平时的衣服下葬。
<br/>　　“叶先生观察真仔细！”秋镇长由衷地赞道。
<br/>　　叶少阳往外面走，说道：“没心情看下去了，这房子里不知道还有什么魍魉，一把火烧了吧。”
<br/>　　秋镇长点头答应，跟着说道：“我回去好好调查这家主人的身份，这件事不能就这么算了。”
<br/>　　从宅子里出来，叶少阳检查了一下那个被打死的邪修法师，是个三十多岁的中年男子，看上去样子很普通，叶少阳在他身上搜查了一遍，也没找到什么有用的东西，干脆让秋镇长找人处理，然后两人一起返回。
<br/>　　“叶先生，不是我给你戴高帽子，你可真是……料事如神！算出那个家伙在这里，果然就在这！”回去路上，秋镇长一个劲地说着赞美之词，眼珠一转，说道：“叶先生，兄弟我说，你去江西，有什么贵干没有？”
<br/>　　叶少阳瞥了他一眼，没有作声。
<br/>　　秋镇长连忙摆手，解释道：“叶先生别误会，我不是打探你的隐私，兄弟的意思是……像叶先生这样的高人，我可以帮忙引荐给县长，以你本事，将来不愁没有大富大贵的机会。”
<br/>　　叶少阳本想张口拒绝，又想到什么，说道：“如果有需要帮忙的时候，我会来找你，不过我要先去江西办一件要事，别的只有等之后再说了。”
<br/>　　秋镇长拍着胸脯保证随时等候差遣。
<br/>　　回到秋镇长的家中，两人直奔后院，之前走的时候，叶少阳没有摘掉血渔网，因此腊梅还在下面趴着，不过情绪已经安静下来，阿兰在边上陪着她。
<br/>　　腊梅见到叶少阳，张口要说什么，叶少阳居高临下地看着她，说道：“你主人已经死了，想必你已经收到认主时候付出的魂晶了。我没什么再跟你说的，超度你去阴司吧。”
<br/>　　腊梅叹了口气，幽幽说道：“我业障蒙眼，做了蠢事，谢大法师不杀之恩。”
<br/>　　叶少阳将血渔网揭开，收了起来，腊梅没有走，来到阿兰身边，去跟她道别。阿兰也是一副依依不舍的样子。
<br/>　　叶少阳看着这一幕，不禁摇头，真是兜兜转转又回到原点，说起来，也只有四个字：造化弄人。
<br/>　　叶少阳画了一张引魂符，丢在空中，腊梅跟阿兰道别之后，附身上去，按照正常程序，引魂符应该飞走，然后燃烧，将鬼魂带到阴间去，然而，这张引魂符在空中飞了一会儿，却又飘落在了地上。
<br/>　　叶少阳看着飘落的引魂符，当场怔住了，上前将其捡起来，整个人是茫然的。
<br/>　　这引魂符，自己画了没有一千次也有八百次，绝不可能画错，可是……为什么没有飘到阴间去，反而坠落下来？
<br/>　　叶少阳捏着引魂符，怔怔地发了一会呆，抹去符文，将腊梅又抖了出来。
<br/>　　腊梅也是一脸懵逼，看着叶少阳，想了想，觉得这件事与自己无关，自己也没有资格去操心，随即说道：“大法师，我还是自己去阴司报道吧，大法师放心，我在人间心愿已了，绝不留恋。”
<br/>　　引魂符的作用，是帮助鬼魂超度，说白了鬼魂一旦进去，中途就不可能再离开灵符，只能被灵符带着前往阴司天子殿。说白了是法师担心鬼魂半路走脱，才用灵符超度，同时如果这鬼魂有什么冤情之类的，也可以在灵符上写下因果，给崔天子作为参考。
<br/>　　叶少阳点点头，道：“你自己去吧。”
<br/>　　腊梅躬身道了个万福，然后十分留恋地看了阿兰一眼，朝远处飘去，还没飞出围墙，身影就淡化消失，去了阴间。
<br/>(本章完)
