　　两位强大的中年龙族，伫立在半空之中，环视下方的二十余万龙族奴仆，已经大量带着奴仆前来的龙族子弟。
<br/>　　其中，那名白袍中年人二话没说，手掌一翻，掌中出现了一个四四方方的物体。
<br/>　　只见中年龙族大手一甩，那四四方方的不知名物体，便向远处飞去。
<br/>　　与此同时，只见此物迎风见涨，疯狂的膨胀，眨眼之间，便化为了一座城池大小的建筑。
<br/>　　建筑的四方围墙，高达百丈，如同横断天地的堡垒。
<br/>　　此物仿若通体石质，闪烁着灰白色的光芒。
<br/>　　“这是……？”
<br/>　　所有人望去，顿时愕然，因为这并不是一座城池，而是一个巨大的迷宫城。
<br/>　　只见这四四方方的围墙之中，还有无数同样的高墙，在其内纵横交错，每两堵高墙之间均是形成了一条宽约十丈左右的通道。
<br/>　　整个迷宫城之中，足足有数十万条这种宽广的通道，纵横交错，让人眼花缭乱。
<br/>　　最主要的是，除了四方的城墙之外，内部高墙，均是在不停的移动。
<br/>　　所以，这迷宫并不固定，各条通道随时随地都在变化。
<br/>　　这时，只见高空中的两位龙族中年人之中，那位白袍中年人，环视众人，朗声开口了：“这一次我龙族发放给奴仆的宝物，全在这座迷宫之中，共有十件极品宝物，五百件上品宝物，谁能得到全靠本事。”
<br/>　　闻听此言，所有奴仆均是精神大震，十件极品宝物，五百件上品宝物，这简直是让人血脉膨胀。
<br/>　　要知道，龙族发放的宝物，只要是上品宝物，都是外界难得一遇的至宝。
<br/>　　不过，很多奴仆的面色，均是凝重了下来，宝物虽多，但是，相对于奴仆的数量来说，实在是太少太少了。
<br/>　　五百多件宝物，分到二十余万的奴仆身上，可谓是四百余人之中，才能有一人得到宝物。
<br/>　　这时，白袍中年龙族强者再次开口了，道：“这迷宫之中，布置了数十个大阵，其一，进入迷宫之后，你们的修为都会被压制在真神境一品低级；其二，在里面如果受到重创，会被传送出迷宫，但若是被人一击灭杀，那就会陨落在其中；其三，迷宫只有一个出口，得到宝物之后，若想不被他人抢夺，需要自己找到出口出来。”
<br/>　　中年人的声音，沉闷而浑厚，响彻在众人的耳畔。
<br/>　　听闻此言，所有人都明白了，想要得到宝物，很可能要付出生命的代价。
<br/>　　这一点，龙族不会仁慈，因为龙族不需要普通的奴仆。
<br/>　　就算是做奴仆，也要天赋不凡，战力强大。
<br/>　　不过，很多修为不到一品的奴仆，却是立刻心中打起了退堂鼓，此种规则他们根本没有希望。
<br/>　　苏莫转头看了司空炎一眼，只见对方的嘴角，勾起了一丝轻笑。
<br/>　　苏莫知晓，此刻司空炎心中，肯定非常的欣喜。
<br/>　　因为，以司空炎的天赋，同阶之中，除了他之外，不可能有人是司空炎的对手。
<br/>　　“能不能得到宝物？能得的多少宝物？就看你们的实力了！”
<br/>　　白袍中年人的声音再次响起，继续道：“现在，想要进入迷宫的奴仆，可以直接飞进去！”
<br/>　　嗖嗖嗖！！
<br/>　　一瞬间，无数的奴仆，争先恐后的向迷宫飞去，随即纷纷的落进了迷宫之中。
<br/>　　这迷宫极大，比龙都城还要巨大，容纳上亿人都轻而易举，更别说区区二十余万奴仆了。
<br/>　　龙洌的两位女奴，冰儿和凝雪，身形如电，直接就飞入了迷宫之中。
<br/>　　龙烨的奴仆石崐，同样速度极快，闪电般的掠进了迷宫之中。
<br/>　　“分开行动！”苏莫看向司空炎，沉声说道，言罢，他身形一闪，便向迷宫飞去。
<br/>　　司空炎点了点头，同样向迷宫飞去，他与苏莫飞向了不同的方向
<br/>　　因为，以他和苏莫的实力，完全不用一起行动。
<br/>　　而龙城的奴仆巫铨，看了龙城一眼之中，心领神会，向着司空炎飞了过去。
<br/>　　苏莫飞到迷宫的上空，没有急着下去，在上方可以看清迷宫内的一切，也能感受到迷宫上空的空间波动，有阵法之力阻隔。
<br/>　　不过，虽然能看清迷宫中的一切，但是，却看不到宝物在何地。
<br/>　　他注意到了巫铨飞向司空炎，但他没有理会，以司空炎的天赋，他根本不用操心。
<br/>　　伫立在迷宫的上空，苏莫看到了迷宫的出口，这出口是一个虚幻之门，依附在最外围的高墙之上，在不停的移动。
<br/>　　他心中有一个想法，自己直接降落在出口处，然后所有得到宝物之人，都要经历他的抢夺。
<br/>　　不过，想到这是龙族，自己身份特殊，不宜将事情闹大，苏莫便压制住了这个想法。
<br/>　　少倾，苏莫选择了一个无人之地，身形下沉，落入了迷宫之中，双脚踏在了坚实的石板地面之上。
<br/>　　一进入迷宫，他便感受到了压制之力，无穷无尽的压制之力，作用在他的身上，立刻就强行将他的修为，压制在了一品低级。
<br/>　　不仅如此，他的神识和感知力也被压制，探出体外不足五丈。
<br/>　　不过，让他欣喜的是，血脉之力和元神之力，完全没有被压制，这就让他彻底的放心了。
<br/>　　不过，他也不会大意，毕竟，修为高的人，就算被压制了修为，还是占据了很大的优势。
<br/>　　毕竟，修为高的人，神魂强大，精神力浑厚，这会使得元神和血脉之力更加的强大。
<br/>　　苏莫转头看向上空，在迷宫之中，已经无法看到外界，上空是朦胧的阵法之光，阻隔了一切。
<br/>　　敲了敲身边的墙壁，苏莫心中明了，这墙壁有很多阵法加固，以他的实力都不可能击碎。
<br/>　　而且，墙壁的上方也有阵法之力，无法翻越过去，只能顺着迷宫的通道行走。
<br/>　　转头环视一圈，这条通道宽广，坚实的石板地面空无一物，没有半个人影，苏莫便大步向前方走去。
<br/>　　……
<br/>　　迷宫城之外，几乎是呼吸之间，二十余万奴仆，便有九成进入了迷宫之中。
<br/>　　还有约一成的数量的奴仆，在其主人的吩咐之下，放弃了进入迷宫。
<br/>　　因为这些人修为不到真神，就算进入迷宫，也不是别人的对方。
<br/>　　“哈哈，龙腾，你的这个狗奴，今日怕是难逃一劫了！”龙城望着远处的迷宫，强大的神念笼罩在迷宫上空，观看着迷宫内的一切。
<br/>　　因为，他的奴仆巫铨，紧跟着司空炎进入了迷宫之中，两人降落在了迷宫中的同一处通道之中。
<br/>　　“他——真是可怜！”龙腾无奈的摇了摇头，这个巫铨去找司空炎的麻烦，当真是可怜又可悲。
<br/>　　若是迷宫中不压制修为，巫铨战胜司空炎应该很轻松，但是同阶一战，只有被碾压的命运。
<br/>　　“不可怜，这狗奴以下犯上，罪不可赦！”龙城闻听龙腾之言，微笑着摇了摇头。
<br/>　　他以为龙腾所说可怜之人，乃是司空炎。
