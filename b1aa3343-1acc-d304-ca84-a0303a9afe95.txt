　　就在老教皇柯莱特正为自己的阴谋洋洋自得之时，他浑然不知道，此时的叶尘，已经悄然踏入欧洲大陆。
<br/>　　意国首都，罗马机场，
<br/>　　一身休闲装的叶尘，缓步从机场走出，直奔圣彼得大教堂而去。
<br/>　　圣彼得大教堂，位于世界上最小的国家梵蒂冈之中，在罗马城的西北角。
<br/>　　叶尘的身形在人群中，一连闪烁几次过后，便已经来到目的地。
<br/>　　作为西方教廷的圣地，圣彼得大教堂建造的美轮美奂，雄伟壮阔，足足可以容纳数万人。
<br/>　　叶尘身体一闪，便已经绕过检票入口，直接踏入其中。
<br/>　　教堂之内，风景如画，游客如云。
<br/>　　叶尘神念一动，很快便已经锁定，在圣彼得大教堂深处，有着几道颇为不弱的气息。
<br/>　　正准备直接过去，就在这个时候，一旁忽然响起一道清脆动听的女子声音，而说的居然是华夏语，
<br/>　　“嗨，小帅哥，你也是华夏人吧？”
<br/>　　叶尘微微扭过头，
<br/>　　这是一名容貌极美的亚裔女子，上身紫色T恤，下身穿着紧身牛仔裤，将一对修长、笔直的双腿，还有那完美的身材，展现的淋漓尽致。
<br/>　　而且，让叶尘颇感意外的是，这女人竟然是一名内劲武者！
<br/>　　看在同为华夏人的份上，叶尘倒也没有直接走人，
<br/>　　“有事？”
<br/>　　那火辣美女见叶尘竟然如此淡定，顿时秀眉微微一蹙，
<br/>　　以她的容貌身材和气质，一般男人看到她，早就被迷的神魂颠倒了，更何况对方，还只是一个二十岁不到的少年，正是血气方刚的年纪。
<br/>　　她原本只是随口搭讪，本以为对方肯定受宠若惊，却没想到对方的反应，竟然如此淡漠。
<br/>　　“那个，小帅哥，你能帮我拍个照吗？”
<br/>　　火辣美女有些不甘心，继续开口道，而且一边说着，一边将自己的手机，递到叶尘的跟前，一脸的期待之色。
<br/>　　叶尘顿时眉头一皱，冷冷道：
<br/>　　“不能！”
<br/>　　说完这话的同时，叶尘已经从她跟前直接越过。
<br/>　　那火辣美女先是一愣，继而气得差点吐血。
<br/>　　她一向自视甚高，而且又身份不凡，从来只有她拒绝别人，还从来没有被人如此干脆直接的拒绝过！
<br/>　　“你、你给我站住！你竟然敢拒绝我？”
<br/>　　韩颖玉有些气急败坏，快步追上叶尘的脚步，一把拦在了她的跟前。
<br/>　　叶尘被这女人三番两次骚扰，顿时有些厌烦，不过他自持身份，也懒得和这女人一般见识。
<br/>　　就在他正打算，直接动用缩地成寸的神通，摆脱这个无聊女人的时候。
<br/>　　忽然又有一道男子的声音，在一旁陡然响起，
<br/>　　“小玉，出什么事？是不是这小子骚扰你？看我帮你教训教训他！”
<br/>　　声音落下的同时，一名身材魁梧高大的青年男子，从一旁冲了过来，一脸的凶神恶煞，二话不说，抬起手掌，便要朝叶尘抓来！
<br/>　　叶尘脸色顿时微微一寒，不由得升起一股杀意。
<br/>　　眼前的这名魁梧青年，竟然还是一名武道高手，差不多半步化劲的境界，以他的年龄，在华夏武道界，已经算得上是难得的武道天才。
<br/>　　可惜，在叶尘的面前就有些不够看了，甚至连让他动手的资格都没有，直接一道神念就能轻易抹杀。
<br/>　　不过，那魁梧青年要对叶尘出手，却被韩颖玉一把拦了下来，一脸不耐道：
<br/>　　“王硕，你少多管闲事！这事和你没关系！还有，虽然我们韩、王两家是世交，但是我和你不熟，请你以后不要叫我小玉！”
<br/>　　那名被称作‘王硕’的魁梧青年，脸色顿时微微一变，眼眸中先是闪过一抹怒色，不过旋即又笑道：
<br/>　　“小玉，看你这话说的！咱们两个可是有婚约在身的，我这才从非洲战场赶过来，就是要带你回去成婚的，你注定是我王硕的女人！”
<br/>　　韩颖玉听到这话，顿时越发反感，正要开口反驳，此时叶尘却已经又从一旁越过。
<br/>　　那王硕身体微微一晃，直接便拦住了叶尘的去路，嘴角露出一抹冷笑道：
<br/>　　“小子，本公子的未婚妻，你也敢招惹？胆子不小啊！”
<br/>　　叶尘缓缓抬起头，淡淡瞥了王硕一眼，只轻轻吐出一个字，
<br/>　　“滚！”
<br/>　　轰隆！
<br/>　　王硕顿时感觉整个人的脑袋，犹如被一道春雷轰中，仿佛要炸裂开来一般！
<br/>　　蹬蹬蹬！
<br/>　　身体更是猛然一震，不自觉的就倒退了七八步，险些一屁股坐倒在地。
<br/>　　“什么！你、你到底是什么人？”
<br/>　　王硕脸色惨白，不可思议的望着叶尘，满脸骇然之色。
<br/>　　连他自己也说不上来方才那种感觉，对方明明什么都没有做，可是他却发自心底的感到胆寒，犹如被恶魔盯上了一般！
<br/>　　若非他从小就在战场上历练，意志远远超过常人，此刻就算不被活活吓死，也得身受重伤！
<br/>　　“嗯？”
<br/>　　就连叶尘也微微有些意外。
<br/>　　虽然方才他只动用了一成不到的神念之力，可即便是化劲宗师级别的武者，也绝对扛不住，非得吓得大小便失禁不可。
<br/>　　没想到眼前这个叫王硕的青年男子，竟然可以挡得住他一道神念攻击，也算得上是天赋异禀了。
<br/>　　他又哪里知道，这王硕乃是华夏八大家族中，王家年轻一代中有名的武道天才，论武道天赋，绝对不在当初的叶无伤之下。
<br/>　　当然，
<br/>　　叶尘微微惊愕过后，便不觉得有什么了。
<br/>　　只不过，他一击之后，也懒得再继续出手，
<br/>　　“我不管你是什么人，不要招惹我，否则，死！”
<br/>　　撂下一句话，叶尘直接缓步朝教廷深处走了过去。
<br/>　　足足过了好一会，王硕方才从之前的惊骇中恢复过来，不过脸色依旧无比难看。
<br/>　　他自负是武道天才，在华夏武道界的年轻一代之中，能和他比肩的，也不过寥寥数人而已。
<br/>　　没想到，今日居然被一个，比他还要年轻许多的少年一言斥退，这简直就是奇耻大辱。
<br/>　　只不过，他一想起之前那道诡异的感觉，便再也不敢去找叶尘的麻烦了。
<br/>　　至于一旁，目睹了全过程的韩颖玉，先是微微一呆，继而美眸发亮，
<br/>　　她知道王硕武功极高，正苦于无法摆脱此人的纠缠，如今看到叶尘竟然轻易将王硕逼退，心里顿时有了主意，于是当即快步朝叶尘跟了上去。
<br/>　　王硕见状，微微愣了片刻，最终一咬牙，也跟了上去。
