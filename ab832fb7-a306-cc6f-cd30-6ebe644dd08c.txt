　　权衡再三之下，刘哔还是放弃了前去查看的想法，因为这件事细细琢磨，他发现并没有表面看上去那样简单。
<br/>　　主要是任发今天的表现实在是太过淡定，在咖啡厅时也只是和九叔商谈了迁坟，对于此事却只字未提。
<br/>　　从这样的角度看来，这件事情就十分反常了，如果真的是厉鬼或者僵尸所为，恐怕任发也不会是这般反应。
<br/>　　回到义庄的时候，太阳已经完全落了下来，九叔和文才两人正在大厅吃着晚饭，看见刘哔回来，招呼了一声，问他吃过了没有。
<br/>　　看到桌上几盘清淡的蔬菜，一股淡淡的恶心感又从刘哔的喉咙间涌了上来，经过上一次的事件，他对吃饭这件事情已经有了阴影。
<br/>　　“师父，师兄，你们先吃吧，我在外面吃过了。”摆了摆手，刘哔向两人打了一声招呼，随后径直上楼，走回了自己的房间。
<br/>　　面对这样的情况，他有些苦恼，一次两次的他都可以糊弄过去，但是时间长了，难免会露出什么端倪。
<br/>　　但是现在任家的事情更加重要，等这次任务完成，在想办法解决自己的问题吧，实在不行的话，也只能搬出去住了。
<br/>　　打开自己的房门，脱鞋上炕，刘哔心中依旧思索着任家闹鬼的事情。
<br/>　　他心中知道，在这个世界中鬼怪僵尸是真实存在的，但这并不代表他能轻易接受。
<br/>　　虽然在刘哔原本的世界里，各种丧尸变异兽也比这些东西好不到哪里去，见多了光怪陆离的事情，他也产生了一些抗性。
<br/>　　可是，丧尸之类的，还可以用科学来勉强解释，但自从系统出现之后，他心中脆弱的世界观就彻底崩塌了。
<br/>　　这和勇气无关，任何人类面对超乎自己理解范围的东西，都会有一种天生的恐惧感，况且这个东西还有一定的智能。
<br/>　　如果不是依靠末世生存十年磨砺出来的坚韧意志，换成寻常的普通人，恐怕心神早就崩溃了。
<br/>　　能淡然接受这种事情的，除了疯子，也只有小说里那些作者YY出来的主角了，事情真的发生，谁的心能有那么大。
<br/>　　从事情发生到现在，刘哔依旧有一种如梦似幻的感觉，恐惧更是如同一块散不开的乌云一般，一直笼罩在他的心头。
<br/>　　……
<br/>　　深夜，万籁俱静，小镇上大部分的居民都已沉沉睡去，偶尔有几家住户亮起灯盏，随后也很快的熄灭。
<br/>　　皎洁的月光洒下，落在下面各色的建筑上，微风刮过树叶，带起一阵‘沙沙’的声音，细听下去，还夹杂着几声不知名昆虫所发出的模糊叫声。
<br/>　　空无一人的街头上，突然出现了一道黑影，手持油灯，一点微弱的光芒自他手中散发出来。
<br/>　　老王觉得今天有些倒霉，今晚他去镇上李寡妇的家里，一起探讨关于豆腐脑是咸的好吃还是甜的好吃这件事情，半夜的时候，却被人给扔下了床。
<br/>　　扔他下床的是李寡妇的儿子大头，本来之前都探听好了，大头晚上不会回来，但天算不如人算，最终计划还是出现了纰漏。
<br/>　　上了年纪的老王，自然不是年轻力壮大头的对手，在一顿惨无人道的殴打过后，更是被迫签下了一张巨额欠条，并威胁他，如果不还钱的话，就把他的事情给捅咕出去。
<br/>　　想到大头先前那副嚣张跋扈的样子，老王就气不打一处来，但他也不是傻子，仔细一想就知道肯定是被人给坑了。
<br/>　　否则时间又怎么会卡的那么及时，在他刚进屋没多久李大头就冲了进来，这明显就是针对他而设下的圈套。
<br/>　　但是想明白过来又能如何，事情已经犯下，欠条也已经按上了手印，把柄落在了别人手上，他根本没有一点反抗的余地。
<br/>　　况且大头也是镇上有名的混混，据说和警察局的保安队长还有些关系，如果不还钱的话，他也要吃不了兜着走。
<br/>　　抵不过心中的气愤，老王狠狠踢起了路边的一颗石子，却不料身体重心不稳，径直跌倒在了地上。
<br/>　　“哐当！”
<br/>　　一时慌乱之下，老王手心一松，玻璃质的油灯在地面上翻滚两圈之后，破碎了一地。
<br/>　　本来就十分微弱的灯火，也在这一摔之下，彻底熄灭了。
<br/>　　屋漏偏逢连夜雨，心中暗骂一声，老王从地上爬了起来，油灯肯定是没法用了，只能摸黑走回家了。
<br/>　　拍了拍身上的尘土，刚刚抬起头来，他却陡然发现，不远处有两团黑乎乎的影子，蹲伏在道旁的建筑门前。
<br/>　　这惊悚的一幕，让他头皮都要炸裂了，不由自主的向后退了几步，喝问道：“什么人？”
<br/>　　但是黑影并没给他丝毫回应，老王心中有些慌乱，心脏在胸腔快速跳动着，冥冥中感觉似乎有什么事情即将要发生一般。
<br/>　　“妈的，都来欺负老子，老子倒要看看究竟是什么东西！”一股怨气从心底涌了上来，老王脸色狰狞，恶胆旁生，径直走了上去。
<br/>　　当看清两团黑影的真实面容之后，老王心中稍稍松了口气，原来是两头一人大小的石狮子。
<br/>　　还当是什么东西，原来只是两头狮子，嗤笑一声，老王摇了摇头，心中依旧有些莫名的恐惧感，就像是有什么东西在暗中注视着他一般。
<br/>　　这两头狮子他有些印象，镇上在门前放石狮的人家不少，但是这么大的也只有任家有资格摆出来，在这个地方，他总是有些心神不宁。
<br/>　　就在不久前，他刚听自己弟弟说了，任家最近发生了一些诡异的事情，听说还死了一些人，他可不敢在这里停留太长时间。
<br/>　　月黑风高，一阵阴冷的晚风吹来，似乎是感觉寒冷，他禁不住打了一个寒战，嘴上嘀咕一声，摸索着向自己家中的方向走去。
<br/>　　夜更加漆黑了，两头石狮在微弱月光的照射下，紧紧注视着老王的身影，眼珠随着他离去的方向，缓缓转动着。
<br/>　　可惜已经离去的老王，并没有发觉身后的异样，摇摆的身影，在黑暗中显得有些孤独，如同随时都会熄灭的蜡烛一般。
