　　“天机印？”苏重拿着手中泛黄书册装作满脸不解，一副不明白这是什么东西的样子。心里却着实震惊，天机印在诛仙世界只有惊鸿一瞥，却明确显示它和诛仙剑阵有关。
<br/>　　“咱们青云还有这种阵法？”苏重佯装不懂询问道。
<br/>　　“天机印是诛仙剑阵的一部分。”田不易凝重道。
<br/>　　“诛仙剑阵？！”苏重大吃一惊。虽然早就知道它和诛仙剑阵有关，却没想到天机印本来就是诛仙剑阵的一部分。
<br/>　　“你只能在藏书楼翻阅，看完后交给我。不能让书册离开藏书楼。”田不易郑重叮嘱。
<br/>　　“是师傅，我知道了！”苏重肃然道。
<br/>　　和诛仙剑阵有关那就非同小可了。这是和太极玄清道一样品级的东西，属于青云门的根本。
<br/>　　田不易能把它交给自己观看，苏重还是非常感激的。
<br/>　　苏重直接打开书册开始翻阅，他看的的不快，但也不慢。一张张慢慢翻阅，从头到尾仔仔细细观看，保证没有丝毫遗漏。然后阖起书册闭眼回想，之后再打开书册翻阅。
<br/>　　这样来回重复三遍，苏重睁开眼把书册递给田不易。
<br/>　　“怎么？有问题？”田不易疑惑接过书册：“有问题可以问我，天机印奥秘无穷，短时间很难理解。”
<br/>　　田不易也是做了这些年首座，不断参悟才有了点儿心得。
<br/>　　“天机印确实深奥。”苏重只是大体浏览就感觉到其中艰深：“书中内容我都记下了，秘籍还是师傅您来保管的好。”
<br/>　　“全记下啦？”田不易诧异。
<br/>　　“是的。”苏重淡然承认。
<br/>　　“过目不忘？”田不易已经震惊的麻木了。刚才看苏重翻书闭眼已经隐隐猜测，只是事实还是太惊人。
<br/>　　“好吧，如果有记忆模糊的地方，再来找我校对。”田不易已经开始适应自家徒弟的妖孽了。
<br/>　　“谢师傅！”苏重把田不易送走，自己却再次扎进藏书阁。
<br/>　　“本来只是想弄点儿高级阵法，没想到直接就搞来了部分诛仙剑阵。”苏重不禁摇头，以他知识储备还不足以解析天机印。刚才翻阅他就明白这一点。
<br/>　　“还需要其他资料参考。”他精通奇门阵法，但诛仙世界阵法和奇门阵法是两个体系，他得重新学。
<br/>　　首先一点儿，诛仙世界阵法针对的是天地灵气，利用的也是天地灵气。
<br/>　　奇门阵法更多是利用自然环境，牵引草木精气日月精华，和这个世界的阵法格格不入。
<br/>　　“奇门阵法依靠环境和谐排布，这里的阵法却是借用符文布置。对环境要求大大降低！”
<br/>　　就像藏书楼，要是苏重布置奇门阵法，那就得需要一个巨大的外围环境来放置阵法。
<br/>　　但用本世界阵法，所有阵法相关符文，全都绘制在藏书楼上，占地面积小，威力强劲！
<br/>　　他之前就在梳理诛仙世界阵法，现在已进行大半。放下天机印不理会，苏重继续攀登阵法技能书。
<br/>　　半个月后，苏重抬起头，双眼还有些迷茫，脑海中各种阵法知识汇聚，逐渐变得清晰起来。
<br/>　　“以前还没感觉，现在看去，藏书楼还真不是一般的危险。”站在藏书楼门口打量，苏重感慨。
<br/>　　这大概就是无知者无畏。以前感受不深，现在懂得阵法，苏重放眼看去，各种符文若隐若现，灵机牵引运转见莫大威能阵阵传来。
<br/>　　“如果谁要强闯，死都不知道怎么死的！”
<br/>　　进入藏书楼唯一方法就是用钥匙开门，平平无奇的钥匙是阵法关键核心。
<br/>　　这种用一个小小物件左右整套阵法的方式，让苏重打开眼界。
<br/>　　回到后山竹屋，苏重没有立刻参悟天机印。他没忘记答应田不易的事，说好的要监督众位师兄修炼，苏重不会食言。
<br/>　　“正好趁机试验一下阵法威力。”苏重嘿嘿一笑。
<br/>　　之后他就在后山划出一片平坦区域开始布置阵法。
<br/>　　诛仙世界的阵法核心在于符文禁止，用法力绘制符文，勾连激发后就会有奇效。
<br/>　　苏重所有法力都是庚金剑气符，锋锐犀利质量极高。但对于绘制符文这种精细操作就有些困难。
<br/>　　他并不打算找帮手：“正好用来磨炼剑气操作。”
<br/>　　一连忙活了十多天，苏重才把一个粗糙阵法布置好。
<br/>　　“庚金剑气还是太过锋锐刚硬。用来攻击事半功倍，用来进行布阵这种精细活吃力不讨好。”苏重擦了擦额头冷汗。这几天可把他折腾的够呛。
<br/>　　每一枚阵法符文，他都要绘制二十遍以上，不然根本就画不出来。一个不注意就剑气就把符文崩坏，不然也不会花费这么长时间。
<br/>　　“好在终于完成，而且剑气操控更加从心，也算一举两得。而且阵法也可以检验一下啦。”苏重小心翼翼放出丝丝剑气没入一根黑竹。
<br/>　　一阵微风突兀刮过，苏重选中的这片空旷区域，莫名的变得湿润了些。苏重闭眼感知。
<br/>　　“草木精气果然变的浓郁。”这只是一个特性，另一个特性就好玩了。这片区域内，天地灵气少的可怜，简直就是一片无灵气区域！
<br/>　　这就是苏重为几个师兄弟准备的修炼场所。
<br/>　　这天，几人吃完早饭，不等众人离开，苏重突然开口：“师傅，我这段时间研究阵法知识，小有收获。在后山布置了一个阵法，想请诸位师兄帮忙测试一番。”
<br/>　　田不易眉头一动，看苏重笑眯眯身色，就知道这阵法必有猫腻。
<br/>　　不等他说话，几个师兄弟就好奇问起来。
<br/>　　“是什么阵法？”这是宋大仁。
<br/>　　“好玩吗？”这是田灵儿。
<br/>　　“去了你们就知道了。”苏重意味深长道。
<br/>　　几个师兄弟毫无所觉，乐呵呵东问西问。因为帮苏重测试阵法，就不用面对师娘的考校。这就和郊游似的，怎么能不高兴。
<br/>　　“我们要不要带着点心？”这是作死能手必输六师兄。
<br/>　　“好好，二蛋师弟，你快去做些甜点，剩下的碗筷让小凡帮你洗。”
<br/>　　苏重笑呵呵：“没问题，交给我。”还甜点？希望到时候你们吃的进去。
<br/>　　田不易脸发黑，本来还想提点几句，现在他也不说了。
<br/>　　几个徒弟太不长心，自家小徒弟是简单角色吗？说是测试阵法还真就信了，太单纯！。
<br/>　　田不易反正是不信的。这当然是苏重变着法督促师兄弟修炼。
<br/>　　当初苏重可是答应过他的。田不易也好奇，苏重到底用什么法子敦促这一帮懒货。
