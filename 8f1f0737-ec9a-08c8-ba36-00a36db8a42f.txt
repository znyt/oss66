厉嫣然身边的同学也有订婚的，按理说订婚不是结婚，订婚后女方还是住在娘家，男方也住在自己的住所里。

<br/>        但是现在时代很开放了，大家都是年轻人，订婚就相当于拿到了一张房卡，大部分人都开始同居了。
<br/>
<br/>        厉嫣然不知道陆寒霆是怎么打算的，现在她试探着问，其实她对今天晚上相当的期待了。
<br/>
<br/>        订婚之前，陆寒霆跟她说，他并不喜欢她，但是这又如何，她现在可以得到他的人，他已经是她的了！
<br/>
<br/>        陆寒霆目不斜视的看着前方在开车，戴着名贵腕表的大手搭在方向盘上，加速，转弯，变道，行云流水，薄唇抿了一下，他嗓音低沉磁性道，“我现在送你回家。”
<br/>
<br/>        送她回家？
<br/>
<br/>        厉嫣然所有的期待都像是被一盆冷水给扑灭了，他竟然没有任何跟她亲密的打算，这对于一个女孩子而言很难堪。
<br/>
<br/>        厉嫣然对自己很有自信的，她生的漂亮夺目，身材姣好，又是水灵灵的年纪，喜欢她的男人多如牛毛，但是现在她放下身段来暗示他，他竟然不为所动。
<br/>
<br/>        “寒霆哥哥，我们现在已经订婚了，以后是要结婚的，我们是不是应该准备一个婚房？”
<br/>
<br/>        陆寒霆名下的房产很多，准备婚房很容易，“好。”
<br/>
<br/>        厉嫣然双眼一亮，她趁热打铁道，“其实房子再多，也要准备一个婚房的，这个婚房里只有我们两个人，意义不一样的，什么时候有空寒霆哥哥带我一起去看婚房啊，我想选一个我喜欢的，而且里面的装修也要听我的。”
<br/>
<br/>        陆寒霆没什么表情，一张俊脸冷淡疏离，既然已经跟厉嫣然订婚了，那在物质上他都会满足她的，“好。”
<br/>
<br/>        劳斯莱斯幻影豪车很快就停在了厉家的草坪上，厉嫣然打开了安全带，“寒霆哥哥，那我回去了。”
<br/>
<br/>        陆寒霆看了她一眼，“恩。”
<br/>
<br/>        厉嫣然有点失望，今晚的订婚各回各家就算了，临别之前他也没有一点表示吗？
<br/>
<br/>        厉嫣然决定主动一点，她飞快的扑过去，用两只小手搂抱住了陆寒霆的脖子，“寒霆哥哥，亲亲。”
<br/>
<br/>        她要亲亲。
<br/>
<br/>        一个20多岁，漂亮夺目的女孩儿搂抱着他，撒娇要亲亲，换成任何一个男人都不会拒绝的，但是陆寒霆下意识就躲避了一下，今天厉嫣然化妆了，身上还有香水味，总之是各种人工添加的香味，让他厌烦。
<br/>
<br/>        陆寒霆突然就想起了夏夕绾身上那股清甜的少女体香，让他喜欢和沉迷的味道。

<br/>        他怎么又想起了夏夕绾？
<br/>
<br/>        陆寒霆有点烦躁，他伸手将厉嫣然给推开，“时间不早了，快回去吧。”
