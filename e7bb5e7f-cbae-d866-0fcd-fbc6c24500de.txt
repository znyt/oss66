
<br/>　　<br/>　　    东郊大野地里的人开始越来越多，这里就好像忽然开启了一个野外party，如果不是欠缺了灯红酒绿的话。<br/>　　<br/>　　    而来的这些人全都围绕着个中心点，那个中心是一个姑娘，她穿着身白色连衣裙，长发被二尺红头绳扎在了脑后，手中拎着一盏古朴的铜灯，她就像是舞会上当之无愧的女王一样吸引了所有人的目光，那些人是贪婪的，充满了渴望的。<br/>　　<br/>　　    这些人中只有秋成子上来就自报家门了，其余人都没有吭声说出自己的来历，因为只有秋成子是名正言顺的以张天师后世弟子的身份要求茅小草把那盏引魂灯归还回来的，至于其他人想要，则只能是明抢才行了。<br/>　　<br/>　　    茅小草用手掌托起那盏引魂灯朝着四周转了一圈，说道：“你们想要？”<br/>　　<br/>　　    浓重的呼吸声顿时此起彼伏的传来，对于这些人来说，一件法器的重要性堪比一座金矿了，对金钱他们可能是很感兴趣，但对这盏灯则是非常感兴趣了。<br/>　　<br/>　　    秋成子红着眼镜说道：“那是我们正一教的引魂灯，是师祖张道陵天师的”<br/>　　<br/>　　    茅小草鄙夷，嗤笑着说道：“你这么说，问问他们会答应么？别天真了，他们会从我的手里抢走，同样也会从你的手里抢走，杀人越货听过么？”<br/>　　<br/>　　    秋成子的脸上顿时呆滞了，他从很多人的眼中都看到了贪婪和赤裸裸的欲望，还有一抹凶狠的味道，很显然今天所有的人都起了歹念。<br/>　　<br/>　　    茅小草又忽然说道：“我就权且当作这引魂灯是你们正一教的东西好了，我就算交还给你，你能守的住么？呵呵，别说你能不能留住了，恐怕自己的小命都不好说了吧，除非你们整个正一教的人都过来护着，否则剩下的还用我说么？不过，我可以答应你一件事情·····”<br/>　　<br/>　　    秋成子皱眉问道：“什么？”<br/>　　<br/>　　    “我可以答应你，如果过了今天这引魂灯还在我手里的话，我答应你亲自去一趟正一教双手把它奉还回去”<br/>　　<br/>　　    秋成子顿时眼神一拧凝：“当真？你说的是真的，会把引魂灯还给我们正一教”<br/>　　<br/>　　    “我答应你，过了今天引魂灯要是还在我手里的话，我会亲自奉还给你们正一教的”茅小草郑重其事的又重复了一遍后，说道：“前提是，我们得能安然无恙的过了今晚啊，是不是？”<br/>　　<br/>　　    可能谁都没有意识到，已经深陷重围的茅小草三言两语之间就把秋成子用一张空头支票拉近了自己的阵营当中，所有人都把注意力集中在了她说的那句我会把引魂灯还给正一教上，特别是秋成子眼中顿时冒出了一抹感激和激动的神色。<br/>　　<br/>　　    “谢谢你了！”秋成子诚挚的拱手说道。<br/>　　<br/>　　    茅小草嫣然一笑：“都是同道中人，不用客气”<br/>　　<br/>　　    “唰”秋成子从背后抽出那把桃木剑，严阵以待的挡在了茅小草的身前，他转身之际并没有看见刚才对他推心置腹的姑娘嘴角略微的翘了起来。<br/>　　<br/>　　    这么轻易的就被忽悠瘸了？<br/>　　<br/>　　    东干在一旁看见这一幕，恨的牙直痒痒的说道：“秋成子，你傻了是不是？她是拿你当枪使呢，你被她给诓了不知道么？”<br/>　　<br/>　　    茅小草淡淡的说道：“我们道家中人出口就成章，一字千金，难道我会为了这区区身外物而坏了我的道心不成？”<br/>　　<br/>　　    秋成子点头说道：“这位姑娘说的不错，她既然亲口认了就绝对不会出尔反尔，来吧，我倒要看看你们谁敢明目张胆的抢我们正一教的镇教引魂灯”<br/>　　<br/>　　    围观的人顿时都皱起了眉头，正一教是正统道教，祖师为张道陵，天下间也就茅山，龙虎山和天师教等道家大派能与其相提并论，让人去无端得罪正一教这样的大门大户，真的不是什么明智的选择。<br/>　　<br/>　　    茅小草扫了一圈，见人都没有什么动静，就朝后退了两步，秋成子听见她的脚步声后，也往后退了过去，两人慢慢的往外面转移着。<br/>　　<br/>　　    东干不甘心的看所有人都没动，朝着文华和秋生打了个眼色，说道：“天这么黑，谁下的黑手都不一定看的清楚，我们还犹豫什么？抢就抢了，谁知道是谁干的啊？”<br/>　　<br/>　　    干坏事的时候，就差那一头领头羊，只要有一个领头的先动了，其他人就全都随波逐流了，陈胜吴广为啥登高一呼就起义成功了，因为他俩领头了。<br/>　　<br/>　　    东干朝着秋生和文华打了个眼色，两人率先就动手了，秋成子这人脑袋虽然有点木讷容易被忽悠，但动手的时候却绝不含糊，手中桃木剑挽了个剑花之后，就点向了两人，但与此同时旁边至少七八个人见他们已经拔枪走火了，也全都冲了进来，团战瞬间爆发。<br/>　　<br/>　　    这个时候，不久之前刚从火车站出来的王惊蛰，也看见了东郊野地上方那突然冒出来的尸气，只是他赶过来的时候却费了一点时间，也没想到赶过来之后会正好碰上他要找的小草姑娘。<br/>　　<br/>　　    本来，王惊蛰是打算占卜一卦，推算出茅小草的方位然后再去找她的，但此时长安城东郊的异样让他产生了一点好奇，八卦和凑热闹的心思他也有，王惊蛰就想看看那一带到底发生了什么事，会有那么重的尸气冒出来。<br/>　　<br/>　　    等王惊蛰打了一辆车到了东郊野地的时候，还没等下车就看见了一团乱战的人影。<br/>　　<br/>　　    出租车司机都吓了一跳，瞅着打扮古怪的王惊蛰就问道：“唉，朋友，那是火拼呢么，社会人在谈判？”<br/>　　<br/>　　    “不知道·····”<br/>　　<br/>　　    “我看那边打的都要冒火星子了，不是火拼是什么啊？”司机抻着脖子，由于离得比较远，只隐约看见有人拿着长剑在挥舞，就咂舌的说道：“打的真狠，搞的好像铜锣湾浩南跟东兴乌鸦似的呢”<br/>　　<br/>　　    王惊蛰在车里就看见了人群中娇小的人影，当即就急眼了，推开车门就下去了。<br/>　　<br/>　　    “哥们，你小心点哈，刀剑无眼啊”<br/>　　<br/>　　    “去他么的东兴乌鸦吧，我还是鸿星尔克呢·····”王惊蛰看见茅小草，顿时就跟尾巴着火了似的冲了过去。
