　　大罗天星域的另外三大势力，五行天，齐林山，沧海都，都曾经追杀过楚风眠。
<br/>　　五行天，只是楚风眠下手的第一个，接下来就该轮到齐林山了。
<br/>　　楚风眠在将五行天之中所有的古帝殿主的身上，都给种下了禁制之后，便是跟青梦启程，离开了五行天，向着齐林山的方向飞去。
<br/>　　齐林山，位于齐林大陆之中，是齐林大陆的霸主，整个齐林大陆，就只有齐林山一方势力。
<br/>　　楚风眠跟青梦的遁光，在无尽星空之中穿行，半个时辰过后，从楚风眠的眼中看到了一方大陆，漂浮在无尽星空之中。
<br/>　　这大陆，就是齐林大陆。
<br/>　　楚风眠的目光看了过去，整个齐林大陆，都被齐林山所经营了起来，化为了一个坚固无比的老巢。
<br/>　　齐林大陆的各个角落之中，都可以看见一枚枚镇压，整个齐林大陆，就是一座阵图，是守护齐林山的护山大阵。
<br/>　　现在这齐林山的护山大阵，已经是全然催动了起来。
<br/>　　齐林大陆之上，一名名齐林山弟子，都已经是严阵以待。
<br/>　　看来他们显然是听到了一些风生，知道了五行天的遭遇，现在已经是等待着楚风眠的到来。
<br/>　　“看来这齐林山也不打算臣服了。”
<br/>　　楚风眠看到这一幕，眼神之中露出几分杀意道。
<br/>　　这三大势力，只有臣服跟死这两条路，不愿意臣服，就只有死。
<br/>　　“不臣服，只有死。”
<br/>　　青梦也是在一旁冰冷的开口道。
<br/>　　这三大势力，是最早联手对付圣龙宗的，如果不臣服，那也就没有存在下去的必要了。
<br/>　　“不错。”
<br/>　　楚风眠点了点头道。
<br/>　　“走，我们过去。”
<br/>　　楚风眠跟青梦，丝毫不顾这齐林山已经是开启的护山大阵，一步踏入到了齐林山的上空之中。
<br/>　　“敌袭！”
<br/>　　无数齐林山弟子，都可以看到两道身影，降临在齐林山之上，心中都是无比紧张了起来。
<br/>　　他们可都是得到了消息，就在仅仅半个时辰之前，同为大罗天星域的四大势力之一，五行天就彻底的降伏了楚风眠。
<br/>　　这个消息传递出来，让齐林山众人心中都无比慌张。
<br/>　　现在看到楚风眠到来，不少齐林山弟子心中已经都是无比恐惧了。
<br/>　　“不知圣龙宗二位到来，所为何事？”
<br/>　　在这护山大阵之中，几道身影从其中突然飞了出来，这几位都是九劫古帝，也是齐林山之中的高层了。
<br/>　　本来齐林山，五行天之中，还有着半步天人境的强者的。
<br/>　　可他们想要趁着楚风眠度过逆天大劫，来捡便宜，导致齐林山一共足足五位半步天人境强者，现在已经是全部死在楚风眠的手里了。
<br/>　　这些九劫古帝，已经是齐林山的最强者了。
<br/>　　“这是齐林山的副宗主，林书古帝。”
<br/>　　青梦在一旁，指着刚刚开口的哪一位齐林山古帝，开口道。
<br/>　　这林书古帝一开口，倒是一副十分友善的态度。
<br/>　　不过楚风眠可知道，这林书古帝不过是想要拖延时间罢了。
<br/>　　“怎么，你们齐林山自己做的事，现在居然都不记得了吗？”
<br/>　　楚风眠看向林书古帝，嘴角一动，冰冷的开口道。
<br/>　　“我没功夫跟你们浪费时间，现在我给你们一个选择，一，所有人臣服于我，古帝之上的武者妖神，敞开心扉，被我中下禁制。”
<br/>　　“第二，便是死！”
<br/>　　“是生是死，自己选择吧。”
<br/>　　“楚风眠，青梦，我们齐林山一向是跟你们圣龙宗井水不犯河水，你这么做太过了吧！”
<br/>　　站在林书古帝背后，令一名齐林山古帝开口了，这人却是楚风眠曾经见到过的林烈公子。
<br/>　　之前就是在圣地之中，想要迎娶洛汐的那一名齐林山天才，现在已经是踏入了八劫古帝境界，也成为了齐林山的高层之一。
<br/>　　他看向楚风眠，眼神无比仇恨，之前楚风眠让他跪下的屈辱，他可是记忆犹新。
<br/>　　现在看到楚风眠的一刻，便是忍耐不住了。
<br/>　　“住嘴！”
<br/>　　听到那林烈公子开口，林书古帝的脸色大变，可他的话音未落，楚风眠的声音便是响起了。
<br/>　　“哦？林烈公子？上一次我没有杀了你，饶了你一条性命，今日你还敢出现在我面前？正好，给我死吧！”
<br/>　　楚风眠声音响起的一刻，他突然出手了，凝聚灵力在掌心之中，轰然之间一掌对着那林烈公子，猛然轰击了过去。
<br/>　　这林烈公子，居然敢打洛汐的注意，早楚风眠的心中此人早就是一个死人了。
<br/>　　这林烈公子，上一次在圣地之中的楚风眠就想要杀了他，不过是考虑到那个时候楚风眠的实力不足，还不想要跟齐林山撕破脸，才没有杀这林烈公子。
<br/>　　现在既然又见到这林烈公子，楚风眠当然没有放过此人的机会，刹那之间的出手了。
<br/>　　“手下留情！”
<br/>　　林书古帝急忙的喊道。
<br/>　　这林烈公子，可是齐林山之中最为耀眼的天才之一了，虽然无法跟楚风眠媲美，可小小年纪就踏入了八劫古帝，未来踏入九劫古帝几乎是板上钉钉的事情。
<br/>　　“住手！”
<br/>　　轰！
<br/>　　就在这林烈公子身边的几位齐林山古帝，也都是同时出手了，纷纷打出灵力来，来帮助林烈公子，抵挡着楚风眠的一掌。
<br/>　　“小子，这里是齐林山，你杀不了我！”
<br/>　　林烈公子见到这一幕，冷笑一声。
<br/>　　这林烈公子显然是早有准备，从他的身上，这几名齐林山古帝，已经是跟他串通一气，联手来对付楚风眠。
<br/>　　“杀不了你？”
<br/>　　楚风眠冷喝一声，血脉之力催动起来，就在他的掌心之中蕴含的龙力，突然攀升了数十倍，雷霆之势轰击而下。
<br/>　　砰！
<br/>　　纵然是数位齐林山古帝联手的防御，都在楚风眠的一掌之下被撕裂开来。
<br/>　　狂暴的灵力，轰击在了林烈公子的身躯之上。
<br/>　　“不！”
<br/>　　那林烈公子在楚风眠的一掌之下，身躯顿时被撕裂开来，一身精血，想要逃走，可根本无法逃出楚风眠掌控的范围之内，就在楚风眠的一掌之下，林烈公子，被当场轰杀。
