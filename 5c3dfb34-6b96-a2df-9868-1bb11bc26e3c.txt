　　又等了一会儿，见下面没人再跳出来指手划脚，李承乾这才咳了一声说道：“继续说轮战的事情，朕决定调程咬金回京，接下来半年，将由薛仁贵带兵出征，敌人依旧是大食，不过这次朕不要求打仗，只要僵持，目的就是练兵，你等可明白？”
<br/>　　“陛下，明明我们可以战而胜之……”下面有人有些想不通，不知道为什么明明有灭掉大食的机会，李承乾却突然改变了想法。
<br/>　　“战而胜之当然容易，但是朕不知道你们发现了没有，人一但没了目标就会内斗，当年强汉如何，打垮了匈奴之后发生了什么？”李承乾看着下面众朝臣正色问道。
<br/>　　“这，应该不至于吧！”张行成有些不大确定，他认为汉朝只是一个例。
<br/>　　李承乾摆摆手：“至不至于先不说，留着大食并不是说大食可以给我们足够的压力，以我大唐现在的实力，别说他一个大食，就是再来两个也没有办法给我们足够的压力。朕之所以要留着他们只是提醒军方，还有一个敌人，不要轻言放松，要时刻警惕。”
<br/>　　由点及面的道理李承乾并没有对张行成说明，他之所以会认为没有外敌必定内乱其实不是有很多原因的，毕竟多了一千三百年的记忆，国内国外这样的例子其实不知道有多少，只是有很多他不方便说罢了。
<br/>　　倒是军方一些将领多少领会了李承乾意思，若有所思的点着头。
<br/>　　给军方留下一个靶子总是好的，否则当军人没有人敌人的时候，便会懈怠，到时候大唐的军力便会下降。
<br/>　　更何况现在的装备每时每刻都在更新，大唐的军队总要不断去适应新的装备，如果没有敌人，只靠空想那根本就是扯蛋。
<br/>　　就好像当初贞观炮刚刚出现的时候，唐军还是依靠旧的战法在打仗，结果好不容易把敌人从城头上炸下去了，因为下面步兵跑的太慢，结果错失良机。
<br/>　　直到后来经过多次试验，掌握了步炮协同之后，攻城战的时候大唐才一点点掌握了先机，炮击过后步兵已经到了城下，等到敌人反应过来再上城头的时候，唐军已经占领城墙。
<br/>　　这便是有敌人的好处，至少可以随时检验新的战法和新的装备。
<br/>　　而如果没有敌人，那就只能靠假想敌，但假想终究只是假想，与实战还是有区别的，真的到了战场上，战局千变万化，假想敌……呵呵！
<br/>　　便是这样，李承乾所提出的轮战在朝堂上得到了通过，薛仁贵兴奋的进行着出征的准备。
<br/>　　根据前方的战报，这一次薛仁贵准备了大量的短枪，也就是左轮手枪装备部队，贞观炮什么的他并没有带，毕竟老程那里就有。
<br/>　　另外薛仁贵携带最多的就是火箭弹与新研制的燃烧弹，这两种攻击距离半远不近的装备在大食国的战争中发挥了很大的作用。
<br/>　　至于铁菠萝，这东西多多益善，有多少带多少，近距离作战没什么比这东西更好使的了。
<br/>　　不过在老薛临时出发的时候，李承乾特地把他叫到了大唐科学院：“仁贵，这次你去大食还有另一个任务。”
<br/>　　“请陛下吩咐！”薛仁贵是标准的军人，没有那么多废话，与李承乾关系虽然熟络，但外表却看不出任何异状。
<br/>　　“这些人你要带上，在天竺道给朕建立个基地。”李承乾将一份名单交给薛仁贵。
<br/>　　看着厚度几乎接近半寸的名单，薛仁贵嘴角抽了抽，但最后还是重重一点头：“请陛下放心，臣一定完成任务。”
<br/>　　李承乾深吸一口气，看着他笑道：“这个你先不要保证的太早，能不能完成任务还是未知。”
<br/>　　薛仁贵疑惑的从手中那份名单上抬起头：“陛下何出此言？”
<br/>　　“这次朕安排你的任务是在天竺建立一座发电厂，电厂建成之后最大的作用就是给那个东西供电。”李承乾一边解释，一边抬手指了指摆在不远处的一个大概有房子大小的设备。
<br/>　　“那……，是什么？”科学院的疯子时不时总会给人惊喜，但房子大小的设备除了火车与拖拉机，薛仁贵还真没见过其它的。
<br/>　　“这个叫无线电收发机，它的作用与信鸽类似，不过效率却比信鸽要快的多，传递消息甚至连一个呼吸的时间都不用。”李承乾看着那房间大小的设备，双目泛光。
<br/>　　无线电终于算是搞出来了，不过因为技术的原因，这东西只能做到这么大，而且其耗电量还不是一般的大。
<br/>　　但不管如何，这东西到底还是搞出来了，有了这东西，信息的传递已经无限近接后世，边境的消息再也不用等上好几天才能送到自己手中。
<br/>　　只不过东西虽然造出来了，理论上其传输距离也是无限远，但现实情况却并不一定会如此，所以李承乾希望薛仁贵能把这东西带到天竺道进行试验。
<br/>　　而薛仁贵这个时候也是一头雾水，在他看来信鸽传递信息已经足够快，可是李承乾竟然说从天竺道发一个信息到大唐只需要一个呼吸，这，这不是扯蛋么？什么东西能跑的这么快？难道这个房子大小的东西里面装的是一个神仙？
<br/>　　想着薛仁贵便不自觉的将自己心中的疑惑问了出来，甚至还打算等李承乾走了之后，他就钻进这东西里面看看，那个神仙长的是什么样子。
<br/>　　李承乾有些好笑的看着薛仁贵，拍了拍他的肩膀说道：“仁贵啊，朕可以给你十万个保证，这里面绝对没有神仙，信息之所以会传递的这么快，那是因为这里面装的是一种叫作科学的东西。”
<br/>　　“科学？”薛仁贵眨眨眼睛，科学他知道，但科学只是一个词而已，并不是并某种可以看得见摸得着的东西，怎么可能装在这个东西里面？
<br/>　　对于老薛这种脑子里只有打仗的家伙，李承乾发现自己真的没有办法解释什么叫无线电，什么叫波长，什么叫电离层，什么叫反射波，索性最后只能强行命令他任何时间，任何地点，任何与科学院无关的人都不能把这个东西拆了，否则砍头！
