　　第545章
<br/>　　那一日万卷山上，鬼哭狼嚎，百鸟惊飞，便是咚咚咚的海鼓之声，也被压下去半筹。
<br/>　　钟牧山几人失魂落魄，面色苍白，等他们好不容易缓过劲儿来时，程大雷三人已经迈步上山，身影消失不见。
<br/>　　三人拾阶而上，越来越接近山顶，亭台楼阁已历历在目，回头望好大一座琅琊城，却已如芥子一般。
<br/>　　刘发财冲程大雷比出大拇指，用口型道：“程当家，高呀！”
<br/>　　“哈哈，一般啦，一般啦。”程大雷面有得色，也可以说是嘚瑟。
<br/>　　楚云生一张脸仍有些发白，他忐忑不安道：“我们这样好么？”
<br/>　　“有什么不好。”程大雷不屑一股道：“他们终日沉迷在靡靡之音中，长此以往，如何匡扶社稷，如何挽救天下苍生。所以才要当头棒喝，让他们清醒过来。”
<br/>　　“啊，欧公子竟有如此良苦用心，实在令在下佩服。”
<br/>　　“哈，一般，一般。”程大雷更加嘚瑟。
<br/>　　“三位请留步。”
<br/>　　三人继续上山，前方石阶上坐着一人。他一身白袍，席地而坐，面前摆着一方棋盘。微微皱眉。
<br/>　　“什么来路，报个万吧？”
<br/>　　程大雷知道这帮读书人心眼小，不会让自己顺顺利利上山。面前有拦路虎，程大雷也没有别的法子，无非是兵来将挡水来土掩，实在不行，自己还可以揍他们嘛。
<br/>　　“在下李舜时，抱歉拦了三位的路，只因有一方残局未解，三位可否帮在下解惑？”
<br/>　　他垂下头去，注视着面前棋盘，心神似乎已沉在黑白对弈之间，浑不知外物。
<br/>　　装，转什么大尾巴狼。
<br/>　　“这个……什么路数？”程大雷问刘发财。
<br/>　　“李舜时，七贤十二秀中排行第四。号称纵横十九道无敌，曾在琅琊城摆下擂台三年，前后对弈四十九局，无一败绩。”刘发财回忆着脑海中关于李舜时的信息，最后问：“那个……您会下棋么？”
<br/>　　“你说我用棋盘将他脑袋砸晕了，咱们算不算过关？”
<br/>　　“……”刘发财。
<br/>　　程大雷对棋盘上的排兵布阵，不是不擅长，而是根本不懂。也幸亏不懂，就算会上三招两式，很明显也不是这书院对弈第一的对手。如此，也省得浪费时间了。
<br/>　　“不行，还是把他砸晕吧。”程大雷已开始看周围有没有趁手的石头。
<br/>　　正在这时候，程大雷忽然发现楚云生迈步向前，蹲在棋盘前，执起黑子。
<br/>　　咦，倒是忽略了他。毕竟是个读书人，虽然看上去才不惊人，貌不出众，但总该会些琴棋书画，否则怎么叫读书人。只是不知道他棋艺如何，能不能换来自己的『通关文牒』。
<br/>　　程大雷和刘发财上前，盯着棋盘，然后二人又抬起头对视一眼。从彼此眼中看出同一句话：
<br/>　　原来你也不懂啊。
<br/>　　程大雷对黑白对弈一窍不通，仅有的了解，大概也就是知道谁的棋子多谁占优势。至于这点了解，程大雷也不好意思说出来。
<br/>　　他也就是盯着楚云生和李舜时的表情，从中推测一二。
<br/>　　只见李舜时刚才还歪坐在台阶上，此时已坐直了身子，手捻着棋子微微出神。
<br/>　　看来，这楚云生也不是草包一个，大概也是有些本事的么。
<br/>　　程大雷渐渐对楚云生有了些信心。
<br/>　　而看着看着，程大雷也看出些门路。二人半山对弈，棋风却是大不相同，李舜时汪洋恣意，信马由缰，黑白构成的世界间任他翱翔。当然，这需要极高超的棋艺支撑，驾驶一匹脱缰狂奔的野马，却让它不离自己的掌控。
<br/>　　而楚云生的棋风却截然相反，他稳扎稳打，步步为营，方寸间的丁点得失都锱铢必较，杀得头破血流。
<br/>　　程大雷打了个哈欠，却见刘发财已经快睡着。二人本来不通此道，自然谈不上有什么兴趣。看了一会儿，二人都快看不下去了，据说一盘棋磨叽起来要花很长时间。当初李舜时摆擂台三年，才下了四十九局，可见一般。
<br/>　　“我觉得，刚才用石头将他砸晕，也不失为一个好主意。”刘发财。
<br/>　　“你瞧瞧，如果他们这一盘棋下不完，咱们难道要在半山过夜。”
<br/>　　“会不会有人给咱们送饭？”
<br/>　　“你应该关心饭中会不会下毒。”
<br/>　　其实李舜时也没有和楚云生大战三百回合，打持久战的准备。只要楚云生稍微拿出点令人尊重的实力，李舜时就会放他们上山。毕竟书山虽有路，可你想读书破万卷，也得拿出点真东西来。
<br/>　　至于楚云生在对弈中击败他，却是从未有过的事情。
<br/>　　但随着棋局慢慢推进，事情变得有些不一样了。楚云生宛若老僧安坐，稳扎稳打，任你东南西北风，上蹿下跳，我自巍然不动。方寸间拼得头破血流，优势在一点点累积，已渐渐要动摇李舜时的根本。李舜时不得不认真起来，手捻着棋子，思考得时间越来越长。
<br/>　　程大雷百无聊赖的打着哈欠，刘发财也是一副无精打采。而此刻山上书院弟子间，已经乱作一团。
<br/>　　“现在局面如何了？四师兄还没获胜么？”
<br/>　　“已经对到三十六手，不分胜负，师兄略占优势。”
<br/>　　“还好，还好……”
<br/>　　“现在两人已经下到七十二手，师兄优势尽失。”
<br/>　　“什么！”
<br/>　　程大雷真的等不急了，眼有焦躁，刘发财识趣的递过来一块石头。程大雷赞许的看了他一眼，觉得用暴力解决目前纠缠的局面。
<br/>　　就这个时候，见李舜时忽地站了起来。两眼发直，风轻云淡，超然物外的气质顿时，他身子晃了三晃，忽然哇呀吐出一口鲜血。
<br/>　　有人将这消息报到山上去，山上诸弟子，一片震惊。
<br/>　　程大雷满意的拍了拍楚云生的肩膀，给了他一个赞许的眼神。
<br/>　　瞧不出来么，这也不是个一路躺赢的货。
<br/>　　“你还有没有其他本事，给我透个底儿，待会估计还有拦路虎。”
<br/>　　“在下毕生最得意的是数算，倒是不敢妄自菲薄，至于琴棋书画，末学小技，不敢在人前放肆。”
<br/>　　“啧，有些狂气。”程大雷迈步撩开袍子：“走着，上山。”
<br/>(本章完)
