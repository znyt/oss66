　　洪荒天地，惊天大战，血雨纷飞，残躯裂甲，铺红天涯！
<br/>　　三族大战惊天动地，天地之间劫煞之气充斥着整个洪荒天地，天地间的修士都被这劫煞之气迷红了双眼，纷纷加入到大战之中，也只有天道保护或者有大阵守护之人，才没有被影响。
<br/>　　天地之间，爆炸声响不断，大战随处可见，地面上，血流成河，纷纷汇入到幽冥血海之中。
<br/>　　“哈哈哈，好！好多血啊！强者的味道！我喜欢！”
<br/>　　血海深处，随着注入血海的血液的增加，冥河道人越来越高兴，到最后甚至大笑起来。因为血液的增加，意味着他的修为也会随之增加，他的一身修为，全在这幽冥血海之中。
<br/>　　越是强者的血液，对他的修为越是重要，如今天地大战，血流成河，对于其他人来说，这是地狱，但对于他来说，这简直是天堂啊，他还希望有更多的血液呢。
<br/>　　龙凤麒麟三族大战，刚开始的时候，还是凤凰族与麒麟族两族共抗龙族，但随着大战时间的延长，这些修士都被劫煞之气影响，分不清谁是敌人，谁是盟友？只要是修士，他们不管什么，见到就杀。
<br/>　　杀！杀！杀！
<br/>　　杀红了天，杀红了地，杀红了整个洪荒！
<br/>　　厮杀声震耳欲聋，兵器的交接，道法的破坏，修士之间的自爆，敌人的狂轰滥炸，天地间无数生灵惨遭破灭，大地断裂，虚空生洞，无尽杀戮，不可能停下。
<br/>　　越是到最后，大战越是惊人，就连龙祖、麒麟王、凤祖等人，最后都被劫煞之气迷红了双眼，分不清谁为敌人，谁是盟友？三人互相攻击，不管谁谁谁！遇到就杀。
<br/>　　“杀！龙巡天下！”
<br/>　　龙祖脚踏天地，一步一震，踏碎苍穹，踏向麒麟王与凤祖，每踏出一步，气势就会上升一倍，挥动手中霸拳，轰向两人。
<br/>　　“战！麒麟镇世！”
<br/>　　麒麟王红着双眼，幻化出本体，仰天大吼，四脚踏天，气势大涨，口吐杀机，杀向两人。
<br/>　　“死！凤舞九天！”
<br/>　　凤祖亦幻化出本体，一只高傲的凤凰凌驾九天之上，挥舞着他那巨大的双翅，真火漫天，焚天裂地，破灭苍穹。
<br/>　　“昂！”
<br/>　　龙祖也幻化出天龙本体，一条巨大的九爪金龙横立虚空，龙威震慑天下，口吐龙息，迎向两人，龙爪撕裂空间，一爪拍向两人。
<br/>　　“麒麟杖！吼！”
<br/>　　麒麟王感受到巨大威胁，手持麒麟杖，发出巨大五彩光芒，相互交织，砸向龙祖。
<br/>　　“啾！”
<br/>　　凤祖朝天一声鸣叫，巨大的声响惊动天地，响彻九霄，迎着两人，眼眸里尽是炙热，全身火光耀天，金红色遮住半边天空。张口一吐，便是一道巨大神焰洪流。
<br/>　　“凤凰啸天击！”
<br/>　　三人战得不可开交，周围众人尽皆倒霉，谁上去谁死，大地火山爆发，生灵尽灭，方圆数千万里，寸草不生。
<br/>　　其余的三族成员一样没有闲着，金木水火土五大麒麟摆出五行大阵，灭杀两族成员，所向无敌。
<br/>　　玄龙化作一条巨大银龙，穿梭虚空，灭杀凤凰族人，不过，他被一只巨大的青鸾拖住，与之交战在一起。
<br/>　　黄龙化作一条巨大黄龙，手持长枪，杀向麒麟一族，长枪所至，麒麟倒地，一枪灭一个，枪枪刺破麒麟脑袋。
<br/>　　不过，他亦是没有多久便被金木水火土五只麒麟用五行大阵困住，双方谁也奈何不了谁，尽在僵持之中。
<br/>　　至于其余三族成员就没有那么好的事了，三族不分敌我，遇上就杀，杀不了，就自爆，脱着敌人同归于尽。
<br/>　　一时之间，大地之上，残骸遍地，血洒天涯，残骑裂甲，漫天纷飞。
<br/>　　洪荒之上，一处没人能够看到的地方，几道人影隐于虚空，默默地注视这洪荒之中三族大战。
<br/>　　“三族底蕴，真是可怕。”一个头顶乾坤鼎的灰袍老道看着大战中的三族，眼中略带震惊之色，缓缓说道。
<br/>　　这是乾坤老祖。
<br/>　　“是啊，三族底蕴真是太强了。”另一个穿着青色道袍的老道赞同道。他是穹天老祖。
<br/>　　另一个身穿黑白双色道袍，胸前一个巨大阴阳图的老道默默无语，只是眼中震惊之色难以掩饰。他是阴阳老祖。
<br/>　　最后一个老道面上尽是淡漠之情，看着三族大战，不为所动。他是鸿钧老祖。
<br/>　　在另一片虚空之中，还有几个大能，都隐藏于虚空，看着大战之中的三族，眼中尽是震惊之色。他们都为龙族、麒麟族、凤凰族所展现出来的底蕴震惊了。
<br/>　　他们是强，没错，但他们却是孤家寡人，要让他们对上单独一人还好，但要是让他们对上一个种族，他们就算再强，也得跪啊。
<br/>　　几人相视一眼，尽皆无语。
<br/>　　“哈哈哈！”
<br/>　　西方灵山，魔祖罗睺看着源源不断向自己用来的劫煞之气、杀戮凶戾之气，哈哈大笑，面对在他算计之下无尽惨死的生灵，面上一点愧疚之色都没有。
<br/>　　他手持四剑，布下阵图，引导四剑吸收那些蜂拥而来的无尽煞气，使得四剑变得更加可怕。
<br/>　　“哈哈哈！鸿钧，你等着，本尊定会杀了你，然后成圣！”
<br/>　　……
<br/>　　“金龙枪！出！”
<br/>　　龙祖大吼一声，手中顿时出现一柄巨大的金色龙纹长枪，枪尖尽是冰冷之意，漫天杀意惊人，天地仿佛变得寒冷刺骨一般。实际上天地并没有变冷，而是那长枪释放出来的杀意所导致的。
<br/>　　龙祖手持金龙枪，金系法则，冷厉，杀伐，金戈，肃杀之意，再加上力量法则的霸道、强势、一力降十会，轰向麒麟王与凤祖。
<br/>　　枪尖所过之处，虚空生洞，滚滚的虚空混洞之力蔓延开来，带着无上伟力，破灭苍穹，杀向两人。
<br/>　　“吼！麒麟杖！挡住！”
<br/>　　麒麟王挥动麒麟杖，放出无边神力，五行道法加持，法则叠加，用来抵御龙祖的金龙枪。
<br/>　　“啾！”
<br/>　　凤祖一声鸣叫，手持火红长剑，挥动巨力，火系法则加持在剑身，堪比太阳真火的巨大火焰形成天柱，轰向龙祖。
<br/>　　一时间，三人大战不息，就算受伤，就算流血，仍然不停，不到最后一刻，大战就永不终止。
<br/>　　血泪交织，谱写天地哀歌！
