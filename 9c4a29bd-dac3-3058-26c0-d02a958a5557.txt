　　龙隐部队的几个人在驻K国的中方大使馆汇合了。
<br/>　　李然从头到尾把事情梳理了一遍，对K国这次阻拦自己这些人离开的事情进行了分析。
<br/>　　“没想到这件事情的背后推手会是洛菲克。”
<br/>　　在得到赵天宇提供的消息后，李然倒是显得有些意外，不过很快他就明白了其中的关键，看了眼坐在房间里的人，说道：“洛菲克说到底也只是个代言人，他们这么做的目的也不是我们，而是……”
<br/>　　“是我！”
<br/>　　一个清冷的声音从门口传来，云熙从房间外走了进来，她的脸色还不算太好，带着一丝病态白。
<br/>　　“小熙，你怎么来了？”罗昊赶紧站起身，拿了一张椅子放在距离云熙最近的位置。
<br/>　　“睡了那么久，也该出来活动活动了。”
<br/>　　云熙露出一个微笑，坐在椅子上，说道：“洛菲克之前在代码小镇外要对付老师，就是为了老师手里那些过去从各大武器公司拿走的设计源代码，现在老师死了，他们自然是盯上了我，因为我是GHOST的学生！”
<br/>　　“可是他们怎么会知道你是GHOST的学生？”慕容飞对云熙问道。
<br/>　　“指挥中心里面，可不止是只有我们的人。”李然代替云熙回答道：“以‘暗月’的能力要在K国里塞几个他们的人，实在太容易了，又或者是潜伏在K国的特工把消息传递回去的也不一定；当然，除此以外，还有一个可能，那就是‘暗月’的爪牙深入了洛克希德公司甚至是美国政府中，消息直接是从智天使那边泄露出去的。‘暗月’实在是个可怕的组织。”
<br/>　　“既然他们想要得到GHOST的遗产，那我就让GHOST重现在他们面前！”
<br/>　　云熙那双漂亮的眼睛中闪烁着冰冷和锐利，对赵天宇说道：“我需要信息电子作战部队的指挥权，所有接收过老师指导的人全都参与，一切听我的指挥，你也一样。”云熙看向坐在一旁的林晓阳。
<br/>　　“这个没有问题，我可以给你授权。”赵天宇说道。
<br/>　　子午教过的学生有不少，但是真正得到子午真传的也就只有云熙和林晓阳两个人，而在所有子午的学生中，云熙是毫无疑问的大师姐。就连一些军衔比云熙高的，私下里看到云熙都会尊称一声“大师姐”！
<br/>　　“你身体才刚好，这样会不会太勉强？”
<br/>　　罗昊有些担心云熙的身体状况，
<br/>　　“不用担心。”云熙脸上的冰冷融化开来，说道：“这次我只是调度指挥，而且还有蜂鸟帮忙，不会有事的。别忘了，我可是‘炼狱’的优秀学员呢。”
<br/>　　罗昊没有在劝解云熙，因为他清楚，只要是云熙决定的事情，别人是很难改变的，即便是自己也一样。
<br/>　　国内，某信息化作战部队作战室。
<br/>　　如果不是竖在大门口的“军事重地，禁止入内”的牌子，还有站在门口，手持95-1自动步枪警戒的士兵，直接走进房间的话，很容易把这里当成是一个豪华的网吧，而不是一个军事单位。
<br/>　　房间有两三百米大，里面摆放着五十台顶级配置的电脑，五十台电脑，每五台电脑围成一个圈作为一个小组。
<br/>　　这里就是信息化作战部队的战场，这里看似宁静，风平浪静，也不会有枪林弹雨的场景；但是一旦开战，这里所掀起的腥风血雨并不比真枪实弹的战场相差多少，虽不见血但同样残酷。
<br/>　　此时坐在房间里的五十个人是全都信息化作战部队中的尖子，尽管他们不是子午一对一带出来的，可他们都算是子午的学生，一道命令，把他们从全国各地召集到这里。
<br/>　　“人都到了？”
<br/>　　云熙的影像出现在各台电脑中。
<br/>　　“大师姐！”
<br/>　　房间里每个人都对云熙保持着尊敬的态度，即便他们中间军衔最高的已经是上校。
<br/>　　“老师的事情，你们也都知道了。”云熙说道：“老师选择了他认为最好的归宿。从此这个世界不会再有SHADOW，但是GHOST，却依旧会存在。不仅是我，你们每一个人都是GHOST。有人想要从我这里得到GHOST留下的遗产，既然如此，我们就满足他们的想法。我拟定了几个攻击目标，这次行动由我全权指挥，我们不去夺取什么东西，只是为了告诉所有人，GHOST还存在！”
<br/>　　两个小时后，在云熙的调度指挥，林晓阳的协助下，五十个人分成十个小组，同时对国外十个知名公司发动袭击，其中包括推特、谷歌、脸书等搜索引擎和社交平台的服务器。
<br/>　　云熙没有使用“永恒之蓝”这样变态的病毒，子午从代码小镇带回来两组EG黑客病毒实验室中的病毒源代码，这两组病毒源代码的破坏性超过了“永恒之蓝”，如果它们被投入使用，短时间内无人能解。
<br/>　　但是云熙没有那么去做，因为这会让自己这边成为众矢之的，就算他们是GHOST教出来的学生，但是他们也不可能跟全世界的黑客交手。
<br/>　　如果云熙这么去做了，只要“暗月”这边稍微放出一点消息，很快就能把矛头全都指向中国！
<br/>　　所以，云熙只是攻陷了这些公司的服务器，让他们公司的服务器停止工作一段时间，并且在他们公司网站主页上留下了一个“GHOST”的动态图像然后就撤离。
<br/>　　将全国最好的信息对抗战人员集中到一起，只是为了做这些一般黑客都能做到的事情，看上去有些大材小用，杀鸡用牛刀，但是房间里国家级人才却没有人偷工、放水，这不是为他们而战，而是为了他们的老师，为了“GHOST”之名而战！
<br/>　　某国外知名网站主页，一个幽灵动画从主页上飘过，紧接出现G、H、O、S四个跟幽灵一样飘忽不定的字母，然后一把大镰刀突然在字母“S”后面划下一道刀痕，镰刀一转跟那道刀痕组成了最后一个字母“T”！
<br/>　　GHOST，这个字母就这么出现在这个公司网站主页上，而在“GHOST”出现的那一刻，公司服务器沦陷，管理权限丢失。屏幕上出现两个小时的倒计时，告诉电脑前的人，需要等待两个小时他们才能拿回管理权限，让网站重新工作。
