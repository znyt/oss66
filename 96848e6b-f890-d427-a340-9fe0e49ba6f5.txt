　　第301章
<br/>　　船分波浪，水拍船帮，大河之上，斗大一副莫字旗迎风招展。
<br/>　　甲板之上，莫鸣米披甲摁剑，身后左右立着牧长庚与严笛二人。
<br/>　　“将军，再有十里地，我们就可以抵达蛤蟆寨。”
<br/>　　“好。”
<br/>　　莫鸣米手摁剑柄，抬头看着天空，只见晴空万里，烈日高照，只有西北风赫赫吹着。
<br/>　　“将军，天气如此炎热，正适合火攻，又有西北风助阵，到时候风助火势，火借风威，蛤蟆寨必是一片火海。”牧长庚。
<br/>　　“正是。”莫鸣米点点头：“此战要成，牧校尉当为首功。”
<br/>　　“为将军做事，某将万死不辞。只不过，末将还有一事相求。”牧长庚道。
<br/>　　“什么事？”
<br/>　　“末将想第一个登岛，破寨之后，我要亲手杀了程大雷和徐神机。”牧长庚几乎是咬着牙说出这番话。连日来在蛤蟆寨受的折磨，现在都想了起来，牧长庚心里早就窝着火，此刻蓄势待发。
<br/>　　“程大雷可以交给你，徐神机要留着。”莫鸣米道：“此人若能为我所用，必是一大助力。”
<br/>　　牧长庚想了想，能好好折磨程大雷，也可以发泄心中的火气。对于徐神机，倒也不用太在意。
<br/>　　“大当家，大当家……”
<br/>　　徐神机跑到程大雷面前，累得气喘吁吁。还未开口，程大雷就摆摆手。
<br/>　　“好啦，别说了，事情我已经知道了。”程大雷撇撇嘴：“莫鸣米打过来了。”
<br/>　　“大当家，你怎么知道的？”
<br/>　　“江湖传言嘛。”程大雷道：“这牧长庚刚走不久，莫鸣米就打了过来。恐怕，这牧长庚就是莫鸣米的人。军师，你确定真没有与他说什么吧？”
<br/>　　“当然没有，我防他像防贼似的，什么口风都没有给他露。”
<br/>　　程大雷看着徐神机，这老头的话总觉得信不过。这件事也不得不防，毕竟牧长庚在山寨待了这么久，山寨的地形至少已被他摸熟，如果真被他们打上岛，却也是一件麻烦。
<br/>　　“准备防御吧。”
<br/>　　程大雷挥挥手，兵来将挡水来土掩，且也看看这些日子的训练成果，四百精兵能否击溃五千乌合之众，实在不行，自己也可以再召唤一批人出现，倒出不了什么闪失。
<br/>　　蛤蟆寨在极短时间进入战斗状态，程大雷手持五色旗，在箭楼上亲自指挥。
<br/>　　而这个时候，莫鸣米的大军已杀到蛤蟆岛前。
<br/>　　“将军，你请看，岛上的人已经慌了。”严笛道。
<br/>　　只只见岛上人头攒动，露出紧张模样。莫鸣米点点头：“这颗钉子，终于可以拔掉了。”
<br/>　　“这是我们的地盘，还容得了他作威作福，早该拔掉他们了。”牧长庚咬着牙道。
<br/>　　“也不能这么说，这程大雷还是有些本事的，我以前一直不出手，也是忌惮这一点。不过今日有牧校尉冒死打探来的情报，我又派了重兵，今日的事情该当万无一失了。”
<br/>　　“将军，哪里就是蛤蟆寨的船坞。”牧长庚手指着前方。
<br/>　　莫鸣米已经看到了，只要毁了船坞，就能毁掉蛤蟆寨的海上行动力量。之后便是大军登岛，杀人放火。
<br/>　　他抬头看向天空，蓝天白云，烈日高悬，肆无忌惮的撒下流火。
<br/>　　今天是个杀人放火的好天气呐。
<br/>　　莫鸣米举起令旗，手伸在半空中，他身后各船上，穿着整齐号甲的士兵举起手中的弓箭，箭头上都包着火油，此刻熊熊燃着。
<br/>　　战意连绵，杀机漫天。
<br/>　　莫鸣米深吸一口气，令旗重重挥下。
<br/>　　“放箭！”
<br/>　　万箭齐发，燃烧的火箭升在半空中，像一只只火鸦。
<br/>　　咔嚓！
<br/>　　忽地一道惊雷响过，莫鸣米诧异的抬起头来，只见刚才还艳阳高照，此刻却突然彤云密布。紫色的闪电流过，然后是豆大的雨点啪啪打在脸上。
<br/>　　空中的火鸦瞬间被浇灭，箭支在空中被吹得东倒西歪。
<br/>　　“这是怎么会事，这是怎么会事！”莫鸣米大叫。
<br/>　　好一场大雨，将众人衣甲浇得湿透，船上都是火药硝石等引火之物，此刻也都被大雨淋透。
<br/>　　若只有雨也就罢了，雨还带着风，西北风呼呼刮着，在大河上掀起波浪。莫鸣米的船只被吹得东倒西歪，大船倒是还扛得住，有些小船直接被掀翻了。
<br/>　　众人忙着降帆，稳住船身，几下摇晃，莫鸣米也差点被晃倒。
<br/>　　他踩着湿漉漉的甲板，一把抓住牧长庚的衣领。
<br/>　　“谁特么说要火攻的，这样的天气能放火么！”莫鸣米声嘶力竭的吼着。
<br/>　　“这个，这个……”莫鸣米张口结舌，无法回答。
<br/>　　箭楼上，徐神机也是目瞪口呆，对这场突然来临的大雨毫无准备。
<br/>　　“怎么会事，怎么会事？”程大雷大吼道：“你真的什么都没和他说？”
<br/>　　“这个……肯定没有啦，我嘴巴特别严。”
<br/>　　“你如果没用神通，事情不该这么邪乎啊。”程大雷皱眉思索。
<br/>　　徐神机无语，本来准备挥旗的手杵在空中。
<br/>　　“走啦，走啦。”程大雷拖着发僵的徐神机：“待会雨大了，咱们就不好下去了。”
<br/>　　二人颤颤巍巍，扶着绳梯下楼。牧长庚看到了这一幕，心中忽然一动，恍然想起了什么。
<br/>　　“将军，将军……”牧长庚道：“那徐神机可是有呼风唤雨之能啊！”
<br/>　　“呼风唤雨，你当我是蠢货么！”
<br/>　　“将军，将军，我的话句句是真，不敢有半句欺骗。这话是徐神机亲口与我说的……”牧长庚道：“将军，你不觉得这场雨来得有些怪么？”
<br/>　　莫鸣米一怔，这场雨何止是怪，简直是妖气肆虐，事出反常必有妖，难道这片河上，真发生着什么自己无法理解的事情。
<br/>　　目光看去，只见船队在波浪中起伏，大小船只东倒西歪，根本不成阵型。有些船只被吹翻，士兵在水里拼命挣扎。
<br/>　　人定不可胜天，一场突如其来的暴雨，让这支还未与敌人照面的队伍就面临惨败。五千大军丢盔弃甲，最后究竟有多少人能活下来，还不好说。
<br/>　　莫鸣米深深叹口气，把揪着牧长庚衣领的手放开。
<br/>　　“若真有此奇人，非战之罪呐。”
<br/>(本章完)
