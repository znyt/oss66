　　虚空小世界之中，地藏神王走后，全场现如今了寂静之中。
<br/>　　苏莫连战两场，斩杀三人，不费吹灰之力，展露的实力远远的超越了四品的范畴。
<br/>　　这让无数的人震惊，这份实力，丝毫没有辱骂轮回之身的身份。
<br/>　　“涂赢，你和苏莫有一战之力吗？”天尘宇面色凝重，向身边的涂赢问道。
<br/>　　“没有！”涂赢摇了摇头，他和苏莫交手不是一次了，两次都是略差一筹，而且当初在神路，他的修为是高于苏莫的，但现如今，他和苏莫同阶。
<br/>　　同阶的修为之人，更加不可能战胜苏莫了。
<br/>　　当然，他就算是有实力，也不会出手，帮助帝一魂对于他来说，并没有什么意义。
<br/>　　天尘宇闻言，沉默不语，他也知道涂赢不可能战胜苏莫，若是涂赢有四品的修为，或许还有可能。
<br/>　　“你们可还有人敢来一战？”苏莫声音淡淡的问道，他心中底气十足，五品修为之下，没有人是他的对手。
<br/>　　天尘宇和紫霄、雷火神王等人，面面相觑，一时之间都不知道该如何是好，虽然他们都还有修为适合的弟子，但是，这些人上场无异于送死。
<br/>　　“韩生，你有没有把握？”紫霄看向身边的韩生，传音问道。
<br/>　　“这……完全没有！”韩生急忙摇了摇头，他可不敢上场，当初在万道剑冢，他就见识了苏莫的可怕，现如今的苏莫早已不是当初可比，他就更加不是对手了。
<br/>　　紫霄面色难看，好不容易有擒获苏莫的机会，他们却是没有能力。
<br/>　　“苏莫，既然你自诩同阶无敌，我们同阶一战如何？”紫霄阴沉着脸问道，现在的情况，修为适合的人根本无人能胜，他只能硬着头皮提出这种建议了。
<br/>　　此言一出，所有人都是神情一怔，大感心中无语。
<br/>　　苏莫也不禁有些愕然，紫霄居然想要亲自出手，不过，他可不会与对方一战，这根本没有意义。
<br/>　　“紫霄神王，你的脸皮可真是够厚的！”龙腾开口了，冷笑一声，面带不屑的继续道：“你可是中级神王，也好意思一战？”
<br/>　　“我压制修为，也不算违反规则吧？”紫霄厚着脸皮道，虽然他也知道这不太可能，但是他还是想要争取一番。
<br/>　　“你那点心思，谁人不知？”龙腾冷笑一声道，他不可能同意，紫霄若是有什么阴谋诡计，到时候就麻烦了。
<br/>　　言罢，龙腾环视紫霄和天尘宇等人，继续道：“既然你们几大势力无人敢战，那么，今日之事，就此结束了。”
<br/>　　紫霄脸色难看无比，天尘宇等人的脸色，同样非常的难看，虽然他们恨不得现在立刻擒拿苏莫，但是，此时却也只能忍着。
<br/>　　他们均是沉默不语，无言以对，实力不如龙族，只能屈辱的结受这个结果。
<br/>　　苏莫见天尘宇等人沉默不语，便转身向龙腾走去，既然结束了，他也不会再求战。
<br/>　　今日，已经杀了几人，也算是暂解心头之恨了。
<br/>　　这时，一道清朗的声音，突然响起：“不知苏莫小友，是在何处学得的混沌道统？”
<br/>　　开口之人，是一位中年男子，此人穿着一声黄袍，面容儒雅，气质独特。
<br/>　　“在下界，偶然所得！”苏莫回到了龙腾的身边，转头看向此人，根据气息，他知道此人是混沌神朝之人，因为，此人同样修炼混沌源力。
<br/>　　“原来如此！”中年男子微微颌首，面上并无变化，混沌道统在混沌神朝，修炼之人不在少数，早已流传而出，而在下界四方星域，也遗留有传承。
<br/>　　只是，这些传承都并不正统，而据他的观察，苏莫修炼的混沌道统，居然颇为纯正。
<br/>　　随即，略一沉吟，中年男子又道：“我观小友的混沌道统颇为纯正，若是你想修炼混沌大道，还需去我混沌神朝！”
<br/>　　众人闻听此言，均是眼眸一凝，而天尘宇和紫霄等人，则是忍不住眉头一皱。
<br/>　　开口之人，名为未天神王，乃是混沌神朝的强者，此人此番言语，是要招揽苏莫吗？
<br/>　　苏莫眼眸之中，闪过一道异色，对于混沌神朝，他自然也心生向往，只不过，现如今他伸出龙族，可不能露出叛离龙族的姿态。
<br/>　　“前辈过奖了，若是有机会，晚辈会到混沌神朝历练一番！”苏莫面色郑重的说道。
<br/>　　“呵呵，混沌神朝自然欢迎！”未天神王笑着说道。
<br/>　　“未天，你的手未免伸的太长了吧？”
<br/>　　这时，一直未开口的龙帝，声音淡漠的开口了，仿佛是不满未天神王的言语。
<br/>　　“龙帝勿怪，在下可不是要招揽苏莫！”未天神王向龙帝抱拳，郑重的说道。
<br/>　　“嗯！”
<br/>　　龙帝微微颌首，并未纠结此事，环视四方道：“既然此间事了，诸位便离开此地吧，我龙族今日摆下了酒宴，宴请诸位！”
<br/>　　“多谢龙帝款待！”
<br/>　　“龙帝费心了！”
<br/>　　“多谢！”
<br/>　　各大势力的强者，纷纷开口，均是向龙帝道谢。
<br/>　　混沌神朝、亘古魔国、霸天神朝等大势力，也带来了一些顶尖的万古天骄，本来是想要一试龙腾的实力。
<br/>　　但是，现在出现了苏莫这个怪胎，各大势力也打消了让弟子挑战龙腾的打算。
<br/>　　毕竟，有这个苏莫在，他们无法展示出耀眼的光彩。
<br/>　　至于挑战苏莫？
<br/>　　的确有一些人心动，但是，苏莫展现的实力太过恐怖，同阶之中，每一个人有信心。
<br/>　　至于修为高于苏莫的人，就算是胜了，也有些胜之不武。
<br/>　　毕竟，他们不是天王宫和地王宫的人，和苏莫无仇，不会仗着修为去击败苏莫。
<br/>　　随后，一众人便再次跟随龙帝，离开了这处虚空小世界，参加龙族举行的酒宴。
<br/>　　苏莫并未参加酒宴，离开虚空小世界之后，便径直回到了龙腾的苍穹苍穹祖龙府。
<br/>　　龙族的酒宴，整整举行了三日，可谓是热闹非凡，各方强者谈论天下，苏莫之名也不时的在众人口中响起。
<br/>　　值得一提的是，天尘宇和紫霄等人，在酒宴开始之后，只是各自向龙帝和龙腾敬了一倍神酒，便匆匆告辞离去。
<br/>　　对此，无论是龙帝还是龙腾，都并未阻拦，派人护送他们离开。
<br/>　　不过，几乎是所有人都知道，真正的争斗这才开始，无论是天王宫、地王宫、还是终极神王殿，都不会放过苏莫。
<br/>　　三位轮回之身的争斗，会让日后的神域腥风血雨，龙族也将被这股腥风所笼罩。
