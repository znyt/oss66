        “我们先来说说你的事情。”夏玄炘微微扬起下巴，端起刚刚落以柔给她泡的那杯茶抿了一口，一副胸有成竹的样子。
<br/>
<br/>        傅夜沉摊了摊手，做了个请便的手势。
<br/>
<br/>        夏玄炘意味深长地反问：“光管着这么大一个集团就已经让你够忙的了，你为什么还要帮我九叔做事？难道是觊觎我九叔的权力吗？我不觉得你是一个贪图权利的男人。”
<br/>
<br/>        “你九叔是我干爹，我帮干爹做事，难道不是理所当然吗？”傅夜沉云淡风轻地回答。
<br/>
<br/>        而他的回答，确实看不出什么破绽出来。
<br/>
<br/>        但是夏玄炘却不吃他这一套，一语点破他的最终意图：“不，你没这么好心。你是为了你父亲。你觉得你父亲并非为国捐躯，而是遭人暗算。你还怀疑，你那个同父异母的弟弟傅昼景的母亲龙半夏，也并非为你父亲殉情而死那么简单。这么多年过去了，你一直想查明这件事情的真相。”
<br/>
<br/>        “你比我想象中的要聪明点。”傅夜沉薄唇勾了勾。
<br/>
<br/>        夏玄炘放下手里的茶杯，接着说道：“连华生一直在查一本‘蛊札’的下落，而那本‘蛊札’关系到你父亲的死因。”
<br/>
<br/>        “是不是我干爹让你暗中调查的这些？”傅夜沉剑眉微蹙，警惕了起来。
<br/>
<br/>        “我九叔，只是让我帮你查那个‘冒险者游戏’的幕后集团而已，并非让我去查你的私事。”夏玄炘坦然回答道。
<br/>
<br/>        傅夜沉心底莫名其妙地松了口气，但很显然，夏玄炘这个女人不简单：“你不愧是情报组的成员，连我和华生的私事，你都在调查。”
<br/>
<br/>        “我只是好奇罢了，就秘密探探喽。更何况，知道一些关于你这位傅大总裁的私事，并非不是什么好事，说不定我还能从中捞一笔，对不对？夜沉哥你到底是从部队里出来的男人，有血有肉、重情重义，而且不会滥杀无辜。”夏玄炘对傅夜沉倒是有十足的把握。
<br/>
<br/>        傅夜沉淡然地笑了笑：“你只了解了我一半而已。”
<br/>
<br/>        “这一半对我来说，足矣。现在，我们聊聊你太太童筱颖的事情，如何？”夏玄炘红唇微扬。
<br/>
<br/>        傅夜沉却在不经意间皱了下眉头。
<br/>
<br/>        夏玄炘洋洋得意地接着说道：“我刚刚所说的，还漏了一个，而漏掉的那个女人名字叫‘童筱颖’！也就是你太太。想必，夜沉哥你一定也查过你太太的底细了吧！不然，你又岂敢如此轻易地就跟她领证结婚了呢？”
<br/>
<br/>        “我更想知道，你查到了些什么？”傅夜沉黑眸深邃了几分。
<br/>
<br/>        夏玄炘把自己所知道的，全部向傅夜沉摊牌：“我查了她所有底细，二十一年前被人贩子卖到云城西镇一对童氏夫妇手中，童氏夫妇的女主人叫龙茯苓，龙茯苓把童筱颖转交给了梁家女主人龙紫苏，龙紫苏是龙半夏和龙茯苓的亲姐姐，她们三个都是龙沉香的女儿，龙紫苏又将童筱颖交由龙沉香抚养长大，为了隐瞒童筱颖的真实身份，龙沉香将童筱颖的真实年龄故意改大了两岁。你可知道，龙沉香跟我爷爷夏海东是什么关系？”
<br/>
<br/>        “初恋情人。”傅夜沉十分淡定地回答。
<br/>
<br/>        夏玄炘会心一笑：“你果然什么都知道。”
<br/>
<br/>        “龙紫苏、龙茯苓和龙半夏，是夏海东和龙沉香的私生女。夏海东和龙沉香其实早有婚约，但是你奶奶当年是大首长的女儿，不接受一夫多妻。所以，才委屈了龙沉香。”傅夜沉也不打算卖关子。
<br/>
<br/>        这就是为什么，当时童筱颖的养兄诺司峻因为种植罂粟犯事坐了牢，他傅夜沉却可以轻而易举地捞他出来的原因。
<br/>
<br/>        诺司峻和傅昼景一样，都是夏海东不能对外公开的亲外孙。
<br/>
<br/>        夏海东虽然早已退休，但是还有人脉关系留在那里，所以，放诺司峻出狱并不是难事。
<br/>
<br/>        而为了安全起见，傅夜沉才让诺司峻带着他们梁家一家子从此隐姓埋名远走他乡，这其实也是夏海东的意思。
<br/>
<br/>        说到这里，傅夜沉微微闭了闭眼睛：“这些事情，都跟筱颖没关系。”
<br/>
<br/>        “谁说没关系！”夏玄炘却不以为然，“龙沉香当年是云城苗族的第一美女，同时也是承传了祖上医术的巫医，她会巫蛊之术，也会赶尸术，后面为了抚养你太太才成了山里村上的赤脚医生。你太太并非是被人贩子机缘巧合才贩卖给童家的，而是那个人贩子故意贩卖给童家，再间接地转交给龙沉香抚养。目的就是隐藏你太太的真实身份。”
