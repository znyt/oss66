　　九域对归墟天的记载，都是在归墟大陆附近的情况。
<br/>　　而至于归墟大陆之外，甚至是许多隐秘之地，就连云老都不曾知晓，那些地方，都是危险无比的禁地。
<br/>　　一般踏入归墟天的人，都是会被带到距离归墟大陆不远的星球上，甚至是直接带到归墟大陆。
<br/>　　但是这一次，楚风眠一直是向着归墟天的边缘飞去，甚至是已经超越了楚风眠认知的范围了。
<br/>　　如今这周围的一切，都让楚风眠感觉到无比陌生。
<br/>　　归墟天的这一股灵力，是无法打破的，楚风眠索性也就放弃了，安静的等待着，看着归墟天要将他带到哪里去。
<br/>　　穿越了无数星空，一座座秘境，最终一座大陆，落入到了楚风眠的目光之中。
<br/>　　这一座大陆，满地赤红，比起归墟大陆，还要庞大几分。
<br/>　　“在这归墟天内？居然还有第二座大陆？”
<br/>　　楚风眠眼神之中露出了几分震惊之色。
<br/>　　归墟天之中，最为著名的便是归墟大陆，还没有听说过，在归墟天之内，还有着第二座大陆。
<br/>　　轰隆！
<br/>　　就在楚风眠想象之际，那一股包裹着楚风眠的灵力也开始涌动起来，带着楚风眠身躯，直接落到了那一座大陆之上。
<br/>　　这一股灵力，直接撞击在大陆之上，让楚风眠都感觉到了一种震荡，天旋地转。
<br/>　　等楚风眠睁开双眼的时候，眼前已经是一片荒无，在这土地之内，方圆十里，都是荒无人烟。
<br/>　　“先看看这一座大陆上的情况！”
<br/>　　这一片大陆，楚风眠未曾来过，而且在跟云老的交谈之中，就连云老也没有提过这样一方大陆的存在。
<br/>　　极有可能，这一座大陆应该是在九域剧变之后，才出世的，不然的话，无数进入归墟天的强者，不可能没有发现这样一座大陆。
<br/>　　一座新出世的大陆！
<br/>　　归墟天之中，每一座大陆，每一颗星辰之中，都藏有着大量的宝藏。
<br/>　　这样一座新出世大陆，到底会有着多少的宝藏，楚风眠都有些不敢想象。
<br/>　　楚风眠扫视了周围一眼，便是站了起来，化为了一道遁光，直接向着在这一片大陆之中飞行这，观察着这赤红大陆的情况。
<br/>　　这一座赤红大陆，比起归墟大陆，看起来要沉寂的多，方圆百里，楚风眠都没有见到人影的存在。
<br/>　　妖兽，各种荒兽，也没有见到，如同是一片死域一般。
<br/>　　至于宝藏，楚风眠更是难得一见。
<br/>　　“这怎么可能？”
<br/>　　楚风眠眉头一皱。
<br/>　　这样的情况如果是在外面的话，楚风眠倒是不觉得有什么，可是这里是归墟天。
<br/>　　整个九域，甚至是整个大罗天星域都最为富足的一方区域，遍地都是宝藏的地方，怎么会如此荒凉？
<br/>　　“血腥味？”
<br/>　　就在一边飞行一边思考的时候，突然之间一抹血腥味被楚风眠感觉到。
<br/>　　这血腥味，是楚风眠来到这赤红大陆第一次感觉到有生灵的存在。
<br/>　　楚风眠身躯连连闪动，来到了百里之外的一处山谷之中，在这山谷内，已经是没有任何生灵的存在。
<br/>　　只是在地上有着一谈血迹，这血腥味正是从这一滩血迹之中散发出来的。
<br/>　　“最多不出五个时辰，这里应该是发生过战斗！”
<br/>　　楚风眠观察了一下，这血液还没有彻底的消失，那至少是在五个时辰之内。
<br/>　　“就让我看看，这赤红大陆到底是发生了什么！”
<br/>　　楚风眠手指一点，点向那地上的一摊血祭，运转起灵力来，一股灵力凝聚在了楚风眠的面前，顿时化为了一枚水镜。
<br/>　　在这水镜之中，图像开始显现，正是五个时辰之前的景象。
<br/>　　只见在这山谷之中，趴着一名足足有着千丈之长的巨兽，鹿角虎眼，一身赤红色的鳞片，在这鳞片之上，还浑身散发着炽热的火焰。
<br/>　　“火麟！这血液的主人，居然是一头火麟？”
<br/>　　楚风眠看到这一尊千丈巨兽，有些吃惊的说道。
<br/>　　火麟，据说是继承了麒麟血脉的一种神兽，虽然没有麒麟那般的强大，但是一身血脉，不亚于如今龙族的天龙，是无数神兽族群之中的佼佼者了。
<br/>　　而眼前这一头火麟，一身实力更是已经达到了七劫古帝的级别，据对是归墟天内霸主的级别。
<br/>　　可不一会，就在这山谷之上，十几道身影，突然出现。
<br/>　　这十几道身影，都是武者，几乎每一人，都是有着古帝的实力，其中为首的二人，更是八劫古帝的巅峰，已经是达到了接近九劫古帝的层次。
<br/>　　这一群人到来，看到这火麟，便是大笑连连，一起出手，将这火麟所斩杀。
<br/>　　同时在这山谷之内生长的圣药，也都被这一群人扫荡一空，随即便是这一群人离开了，向着赤红大陆的深处飞去。
<br/>　　“居然有人捷足先登了！”
<br/>　　看到这景象，楚风眠才明白过来，为何这赤红大陆看起来如此的荒凉了。
<br/>　　原来是他这一方区域，早已经是被人扫荡过一次去。
<br/>　　“这一群人，可不好对付，看来他们绝不是九域中人。”
<br/>　　楚风眠目光闪烁。
<br/>　　这一群人，有着十几名古帝，其中的四位，都是八劫古帝，这样的势力让九域的哪一方势力，都难以拿得出来。
<br/>　　而且这些人，可都是无比年轻，甚至看起来很像是一方势力所培养出的天才。
<br/>　　“不对，这是？”
<br/>　　楚风眠刚刚想要关闭水镜，突然之间他发现，在杀了这火麟之后，其中为首的一名年轻人，居然是拿出了一枚古老的瓦片。
<br/>　　“人皇罐？这是人皇罐！”
<br/>　　外人看不出这古老瓦片的玄妙，楚风眠却是一眼看的出来，这古老的瓦片到底是什么东西。
<br/>　　人皇罐的一部分，这年轻人，居然也得到了人皇罐的一部分。
<br/>　　只见他拿出这古老瓦罐，注入灵力，这古老瓦罐便是显现出了一个方向。
<br/>　　随即这一群年轻人，便是向着那个方向当即化为了一道遁光飞了过去。
<br/>　　“这一群人，难道是在找其他人皇罐的碎片？”
<br/>　　楚风眠眉头一皱。
<br/>　　他记得之前大罗天星域之中，就有着九域有着极道仙兵碎片现世的消息。
