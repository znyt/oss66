　　第78章
<br/>　　“…………”
<br/>　　在一篇长篇大论后，杜茂无动于衷，只是将刚才程大雷看白痴的眼神还给他。
<br/>　　杜茂一开口就知道是老江湖，并不是林少羽那样单纯善良好忽悠。自然，程大雷也没有幻想能把杜茂忽悠进山寨，他只是试试自己的『雄辩』技能的效果。
<br/>　　目前看来，对付杜茂这种老江湖没多大用。不过，也说不定等日后技能提升了，就可以凭三言两语，忽悠得杜茂二话不说，纳头便拜。
<br/>　　“哼，想要我加入蛤蟆寨也行，除非你有本事堂堂正正打败我！”杜茂撂下这句话。
<br/>　　程大雷并没有打虎的本事，他冲杜茂道：“不如，我在山寨中找个人给你打，你输了，就留在山寨？”
<br/>　　杜茂又露出看白痴的眼神。
<br/>　　程大雷耸耸肩，哼着小曲离开地牢。没有说服杜茂，程大雷并没有失望，本来他也不报期望。反正岁月长，以后未必没有机会。
<br/>　　拔掉杜茂这只大虫后，等于让蛤蟆寨的势力扩展到落叶城。程大雷用信鸽同苏樱联系，知道杜茂被拔掉后，他所豢养的那帮恶汉立刻做鸟兽散。
<br/>　　因此，也没人再去湖底捞捣乱，酒楼重新开业。生意可以说相当不错，每天客满为患。
<br/>　　而通过这件事，程大雷还狠狠收割了一波恐惧值。杜茂本身就是落叶城一恶，懒大虫的名头是一拳一拳砸出来的，如今，程大雷收拾掉杜茂，岂不是证明，程大雷比杜茂更恐怖。
<br/>　　尤其是一些受杜茂保护的大户，在知道杜茂倒下后，终日寝食难安，生怕被山贼惦记，如此也给程大雷贡献了许多恐惧值。只不过，有些小门小户，做小本生意的商贩平素被杜茂欺凌，这些人甚至有些感激程大雷。
<br/>　　这让程大雷的恐惧值收入损失许多。
<br/>　　程大雷自然不满足如此，他撒出人手去，在落叶城宣扬这件事。宣扬的场所，自然是酒楼，茶馆，街头巷尾的人群聚集处。
<br/>　　在这样的场合，总有人站出来，开场词一般无二。
<br/>　　“喂，杜老大的事情你们听说了么，那天我可是亲眼所见……”
<br/>　　接下来便是程大雷如何如何神武，开场双方争议唇枪舌剑，最后程大雷又是如何击败杜茂。
<br/>　　传言自然和事实真相就有很大出入，流言以讹传讹的散开，后来说得便完全不像样子。不过无论怎么传，最后总要用一句话结尾：程大雷一拳就把杜茂打倒了……
<br/>　　事实自然不是如此，以秦蛮的本事，也没有打败杜茂的把握，更何况程大雷。不过莫说没人知道真相，便是有人知道，也没多少人在意了。
<br/>　　暮秋时节，天气渐冷，这一天程大雷吩咐下人去，进城采购一批棉衣棉被，准备过冬。
<br/>　　也就是这天，飞虎寨的高飞豹和于虬然来到蛤蟆寨。
<br/>　　这次人家答应了帮忙，虽然没帮上多少，蛤蟆寨还是要承这份情。于是这两人被请入山寨议事厅，只是程大雷琢磨不明白，这二人上山有什么用意。
<br/>　　“程当家，许久未见，近来可好？”高飞豹。
<br/>　　“蛮好的。”程大雷这句话说的是实话，最近蛤蟆寨蒸蒸日上，一切的确蛮好的。只是程大雷想不明白，高飞豹这个粗人，今天怎么拽起文来。
<br/>　　高飞豹白了程大雷一眼，心中道：谁不知道你如今那杜茂没有办法，如今心里恐怕苦闷得狠，现在你嘴硬还有什么用。
<br/>　　“好就好，大家在一座山吃饭，你们好，我也就放心了。”
<br/>　　程大雷更加糊涂，这娃今天来是干嘛来的。他也只能打着哈哈：“是呐，大家就是要互相帮忙，才能越来越好嘛。”
<br/>　　高飞豹和于虬然对视一眼，彼此心中都道：终于要扯到正题了，你拿杜茂没有办法，终究还是要求到我大哥身上。
<br/>　　以于虬然的意思，要再难为难为程大雷，让他自己说出来请求的话。
<br/>　　但高飞豹已经率先说了出来：“程当家，事情我们都已经知道了。”
<br/>　　知道了……知道什么了？程大雷想了想，定然是自己抓住杜茂的事情，最近也就这件事情闹得大些，于程大雷以及整个蛤蟆寨都脸上有光。
<br/>　　不过，此刻程大雷却谦虚起来：“哈，这也算不上什么了不得的事情。”
<br/>　　唉……高飞豹心中叹口气，话我都给你垫到这个份上了，你怎么就不知道往下台阶呢。既然如此，也别怪我把话说破了。
<br/>　　“呵，这么说，程当家已经有法子了？”高飞豹脸色冷下来。
<br/>　　什么法子……程大雷莫名其妙，试探着道：“应该……有法子吧。”
<br/>　　“什么法子！”高飞豹立刻倾身问。
<br/>　　“……”程大雷。
<br/>　　程大雷真恨不得给高飞豹一耳光，我怎么知道什么法子，我连你说得什么事情都不知道。
<br/>　　“高二当家，于军师，我求你们一件事……”程大雷现在想说，求求你们把话说明白好不好。
<br/>　　高飞豹眼睛一亮，立刻道：“我还以为程当家会一直逞强呢，现在总算是说出来了。”
<br/>　　“想要从程当家口中听到一个求字还是真不容易啊。”于虬然抿口茶，淡然道。
<br/>　　程大雷心灵崩溃，一副生无可恋的表情。
<br/>　　“既然程当家开口了，那我就勉为其难答应你。我大哥决定了，这次陪你去落叶城走一趟，亲自见一见杜茂，请他放过这件事。”高飞豹趾高气扬道。
<br/>　　程大雷眨了眨眼睛，忽然福至心灵，终于明白了些什么。紧跟着睁大眼睛，匪夷所思的看着高飞豹和于虬然二人。
<br/>　　“那件事已经解决了，杜茂已经被我打败了。”
<br/>　　“什么！”高飞豹和于虬然同时道。
<br/>　　“是啊，杜茂现在就被关在蛤蟆寨地牢。”程大雷不可思议道：“这件事……你们竟然不知道！”
<br/>　　的确，杜茂的事情在落叶城传得沸沸扬扬，飞虎寨距离落叶城也不过五十里，怎么说，这件事他们也该知道的。
<br/>　　可飞虎寨真的不知道，大家呆在山寨中，除了偶尔出门打劫，也不经常出来转。而山寨和山寨之间，也不是说没事串串门。
<br/>　　“程当家，你没有开玩笑吧？”
<br/>　　程大雷真的再次被这个时代的信息闭塞震惊，他无奈站起来，道：“请二位与我去地牢看看吧。”
<br/>　　
<br/>(本章完)
