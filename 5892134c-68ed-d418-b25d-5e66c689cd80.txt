        晚上见……
<br/>
<br/>        他真怕自己晚上见到她会忍不住将她给上了……
<br/>
<br/>        傅夜沉带着一身欲火，开车从私宅回到公司后，在自己办公室里二楼的浴室里冲了个凉澡给自己降温。
<br/>
<br/>        为什么他会觉得筱颖满脸通红的样子十分可爱？可爱到他起初想要直接将她扑倒在餐桌上狠狠地要了她。
<br/>
<br/>        不过，终究，他还是忍了下来。
<br/>
<br/>        傅夜沉换上另一套西装，下了楼坐在办公桌后，掏出手机给助理郑忠威打了通电话。
<br/>
<br/>        郑忠威得知傅夜沉的决定后一脸诧异，但他并未问原因，只是替落以柔感到惋惜地说道：“大少爷，小落真的是一个难得的人才，她这段时间的工作能力，我们大家有目共睹，您这个时候将她调去云城分公司，对总部来说，是一种损失。”
<br/>
<br/>        “你可以另招一个男秘书进来，再帮我好好培养就是。”傅夜沉毫不留情地说道。
<br/>
<br/>        郑忠威不再好言相劝，而是将傅夜沉的这个决定，转告知了正在秘书室里整理文件的落以柔。
<br/>
<br/>        得知傅夜沉要将自己调走，落以柔难以置信地从办公桌前站了起来，质问：“为什么？”
<br/>
<br/>        郑忠威微微耸肩，一脸无奈地摊了摊手：“这是大少爷的意思，我也没办法。”
<br/>
<br/>        “我去找他！”落以柔放下手里的文件，疾步走出了秘书室。
<br/>
<br/>        总裁办的门忽然被敲响，傅夜沉冷淡睨了一眼：“进来。”
<br/>
<br/>        落以柔大步流星地走了进去，直接抵达傅夜沉的办公桌前，瘪着红唇，一脸委屈地问道：“大哥，为什么要将我调去云城分公司？”
<br/>
<br/>        “云城分公司的总经理刚上任，你去辅佐他。”傅夜沉波澜不惊地回答，。
<br/>
<br/>        但落以柔心里总感觉，这绝对不是主要原因。
<br/>
<br/>        难道，是那个童筱颖给傅夜沉吹了什么耳边风？
<br/>
<br/>        “大哥，我不想去！我要留在这里！”落以柔咬牙切齿地抗议。
<br/>
<br/>        傅夜沉放下手中的文件，怀抱双臂，倚着靠背，抬眸看着落以柔，眸光的温度低到冻人：“给你两种选择，一、接受安排去云城分公司胜任总经理助理一职，二、辞职走人。”
<br/>
<br/>        “大哥，不要把我调走，好不好？”落以柔鼻子一酸，压低了嗓音，苦涩地哀求。
<br/>
<br/>        傅夜沉却淡淡地回答道：“同样的话，我不想说第二遍。”
<br/>
<br/>        落以柔深知傅夜沉的脾气，向来是说一不二。
<br/>
<br/>        她自顾自地聋拉着脑袋，不再跟傅夜沉争辩，悻悻地转身离开了傅夜沉的总裁办。
<br/>
<br/>        落以柔以为，她在傅夜沉的心里有比较特殊的分量，却没想到终究是敌不过童筱颖那个女人。
<br/>
<br/>        而事实上，她在傅夜沉的心里，只是一个万能的秘书，工作上最令他省心的辅助，仅此而已。
<br/>
<br/>        落以柔能进入他傅夜沉的公司，比她同龄的应届毕业生领着高几倍的薪水，全是托了童筱颖的福。
<br/>
<br/>        就像苏漫雪，也是托了童筱颖的福，才让他傅夜沉捧她进了娱乐圈，成为当红女明星一样的道理。
<br/>
<br/>        如果不是落以柔的一声“大哥”，以及她的身世背景，与童筱颖有着同命相怜的一面，她落以柔只怕都进不了傅氏集团。
<br/>
<br/>        傅夜沉不否认，落以柔确实很能干。
<br/>
<br/>        可以替他忙碌的打点一切，她替他接洽生意，连络安排所有事宜与各大老板的饭局；替他规划工作及私人行程；陪他四处开会，甚至出国出差。
<br/>
<br/>        私事上，当他忙起来或是无暇顾及他自己的时候，她也会尽责尽职地打理他的饮食起居。
<br/>
<br/>        那高几倍的薪水，她落以柔并不是白拿的。
<br/>
<br/>        只是，中午那份便当，里面的荤菜和汤被掉包的事情，是他傅夜沉自己吃出来的。
<br/>
<br/>        几道菜摆在他面前，只要是他吃过的手艺，他都能吃得出来哪道菜是谁做的。
<br/>
<br/>        今天中午，筱颖打开保温饭盒后，一定也知道自己做的菜被掉包的这件事情，只是她没有当着他的面揭穿落以柔。
<br/>
<br/>        既然，他心知肚明，又岂能让他的筱颖受一丁点的委屈？
<br/>
<br/>        所以，中午那道被掉包的荤菜和那份汤，他后来一口未动。
<br/>
<br/>        落以柔若不对筱颖动歪念头，他或许会继续将她留在自己身边辅佐自己工作。
<br/>
<br/>        但是，她没有做到这点。
<br/>
<br/>        一开始，他就向她坦白过，筱颖是他的妻子，她对筱颖不敬，就是对他的不敬。
<br/>
<br/>        若不是念在她落以柔为公司立过功，不然他今天就直接将她给辞退了。
<br/>
<br/>        傅夜沉所做的，只是把所有对筱颖不利的事情，全部扼杀在摇篮里。
<br/>
<br/>        筱颖是被他宠在心尖上的女人，只有他才可以欺负她，其他任何人都不可以，就连阿景也不行。
